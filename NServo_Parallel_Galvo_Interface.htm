Nservo supports up to 4 chip select (/CS) signals. Each /CS is tied to a specific axis. Axes X, Y, Z and U control /CS1, /CS2, /CS3 and /CS4 respectively.

J111B has capacitors for EMI noise suppression. These capacitors must be removed for the interface to work correctly. Removing these capacitors is done by Aerotech under ES-15941.

To enable parallel galvo interface support on Nservo, execute the following command for each axis that requires parallel galvo interface support:

DRIVECMD <Axis> 67 3 1 0 0 0 0 0

Parallel galvo interface support remains enabled after executing this command for each axis until Nservo is reset.

The axes that will have parallel galvo interface support should be set to have no feedback. In the Parameter Editor, set PositionFeedbackType and VelocityFeedbackType parameters to “None (Null Axis)."

While parallel galvo interface support is enabled, Nservo will take over control of the general purpose I/O ports 3, 4 and 5. These ports will not be available to the user. The user should not execute commands to change the direction or the status of these ports.
