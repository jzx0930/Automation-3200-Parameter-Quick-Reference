Diagnostics Tab of the Status Utility

The Diagnostics tab of the Status Utility shows the physical hardware level of the inputs for all axes of the controller and the analog and digital inputs to each drive. The states are independent of all axis parameters that specify the active state of these signals, such as the EndofTravelLimitSetup axis parameter. Refer to Getting Started for information about initial system testing.

Figure: Diagnostics Tab in the Status Utility

STATUS

Position Feedback is the actual encoder position in counts. G92 command, POSOFFSET command, Fixture offsets, and HomePositionSet parameter do not have an effect on this value.

Position Calibration All is the position command contribution that is made by 1D or 2D calibration.

Position Master/Slave is the position command contribution that is made to a slave axis in gearing, camming, or gantry systems.

Auxiliary Position Feedback is the auxiliary encoder position reading that is in counts.

HARDWARE

These are bits in the Drive Status status item.

FAULT

These are bits in the FaultMask parameter.
