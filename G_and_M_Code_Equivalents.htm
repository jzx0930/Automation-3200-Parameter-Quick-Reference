G-Code and M-Code Equivalents

Previous versions of the A3200 used G-Codes and M-Codes for motion programming. The current version of the software uses AeroBasic equivalents instead of many of these G-Codes and M-Codes. G-Codes and M-Codes that were present in prior versions of the software are now part of the CNC Option. Some G-Codes and M-Codes do not have AeroBasic equivalents, and they are only available with the CNC Option.

The following tables list the G-Codes and M-Codes used in prior versions of the software and their AeroBasic equivalent if one is available. The table also indicates which commands are now only part of the CNC Option.

Note: If you have purchased the CNC Option, refer to G-Code and M-Code Programming Overview for more information.
