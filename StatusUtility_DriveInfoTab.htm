Drive Information Tab of the Status Utility

The Drive Info tab of the Status Utility shows hardware information about the connected drives.

Figure: Drive Info Tab in the Status Utility

Shows the maximum current rating of the amplifier.

Shows the main FPGA version of the drive.

Identifies the type of drive that is connected to this axis.

0

PSO Counter 2

PSO Bit Array Index

PSO Laser FIFO index (used by PSOOUTPUT BIT MAP and PSOOUTPUT PULSE BIT MASK commands)

0

Resolver Channel 2/1

Shows the hexadecimal representation of the absolute resolver angle for the optional resolver inputs. The right four characters represent resolver channel 1. The left four characters apply only to Ndrive HLe and HPe and represent resolver channel 2. A value of FFFF represents 360°.

1

MX Absolute Position

Shows the absolute encoder angle for the feedback types that follow:

FireWire Feedback Types

  • -MXH Feedback
  • -MXR Feedback

For the Ndrive FCL and FLS and the Nmark CLS and GCL, a value of 262,143 represents 360°.

For the Ndrive HLe and HPe, and the Npaq, a value of 65,535 represents 360°.

HyperWire Feedback Types

  • -MX1
  • -MX2
  • -MX3
  • -MX4

For the GL4 and the XL4s, a value of 262,143 represents 360°.

For the XC2, the XC4, the XC4e, and the XR3, a value of 65,535 represents 360°.

2

PSO Status

Shows the PSO hardware status register. Refer to PSO Hardware Status for bit descriptions and hardware applications.

3

Drive Timer (usec)

On single-axis drives and on the XR3 multi-axis drive, the Status Utility shows the Servo Loop Timer, which is the same as the FireWire and HyperWire Interrupt Timers.

On Axis 0 of the Npaq, the Status Utility shows the Servo Loop Timer, which is the same as the FireWire Interrupt Timer. On Axis 1, the Status Utility shows the Current Loop Timer.

On Axis 0 of the Ndrive FCL and FLS, the Nmark CLS and GCL, the Status Utility shows the FireWire Interrupt Timer. On Axis 1, the Status Utility shows the Servo Loop Timer.

On Axis 0 of the GL4 and the XL4s, the Status Utility shows the HyperWire Interrupt Timer. On Axis 1 of the GL4, the Status Utility shows the Servo Loop Timer.

5

0

Data Acquisition Samples

Shows the number of samples that you collected using the DATAACQ command.

1

Servo Update Rate

Shows the servo rate of the axis in kilohertz.

2

Settling time (msec)

Shows the settling time for the most recently completed move on this axis. The settling time is the time from the end of the motion command (that is, velocity command reaches zero) to the point at which the axis meets its in-position criteria. Refer to InPositionDistance and InPositionTime.

3

Internal Error Code

This value represents an internal error code from the drive, usually as a result of catastrophic FireWire communications failure. If you see a nonzero value here, contact Aerotech Global Technical Support.

6

0

Firmware Version

Shows the current firmware version that is loaded on the drive.
