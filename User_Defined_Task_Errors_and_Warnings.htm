User-Defined Task Errors and Warnings

You can make your own task errors and warnings for the A3200. User-defined error messages are displayed in the Motion Composer Suite, and they are available through the .NET and C libraries.

Note: If the message that you make contains non-ASCII characters, the UserDefinedMessages.txt file must be encoded as UTF-8 without BOM so that the characters will be shown correctly.

To create user-defined messages, use the following steps.

  1. Create a text file named UserDefinedMessages.txt and put the file in the following location. You must make this file and it is not available by default.

    Windows 7

    C:\Users\Public\Documents\Aerotech\A3200\UserDefinedMessages.txt

  2. Add error and warning messages to the file by using the format that follows. You can add any number of messages, but they must start with a unique number and must be in the format shown.

    Example

    0=Air is not on 
1=Door is not closed
  3. Generate a user-defined task error using the USERTASKERROR command or a user-defined task warning using the USERTASKWARNING command. The <ErrorLocation> or <WarningLocation> that you pass to USERTASKERROR and USERTASKWARNING must correspond to the number listed in the UserDefinedMessages.txt file.

The example that follows shows you how to cause both a user-defined error and warning and assumes the contents of the UserDefinedMessages.txt matches the example listed in Step 2.

Example

' Cause a task warning. The warning message is "Air is not on". 
USERTASKWARNING 0  
' Cause a task error. The warning message is "Door is not closed". 
USERTASKERROR 1
