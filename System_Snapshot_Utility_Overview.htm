System Snapshot Overview

System Snapshot is a debugging tool that captures the state of your system and saves it to an HTML file. The information that is saved can help to diagnose problems or to identify a setup that is not valid. The tool collects more information if the A3200 SMC is running when you run System Snapshot.

Using the System Snapshot Interface

System Snapshot outputs a standard HTML file that you can open in any Web browser. When you run System Snapshot, you can specify where you want to save the HTML file. The tool saves all of the diagnostics and embeds some additional files into that HTML file. After you run System Snapshot, you can open the HTML file and see all of the files that the tool captured.

To run System Snapshot, click Start → Programs → Aerotech → Automation 3200 and then double-click System Snapshot. A dialog comes into view that asks you where you want to save the file.

If you have problems with your system and you contact Aerotech Global Technical Support, you can use System Snapshot to supply Service with a capture of the state of your system.

Automating System Snapshot

You can start System Snapshot from the command line. To make sure that the application is done, wait for this process to exit.

Syntax:

 
systemsnapshot [-o <OutputFile>]
Example: 
systemsnapshot -o "c:\snapshots\snapshot0.html"

Arguments

-o <OutputFile>

This optional argument lets you specify the name of the output file. If you do not specify a file name, the application asks the user to enter a file name.
