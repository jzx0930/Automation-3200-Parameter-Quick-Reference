Task Scheduling Overview

The controller has up to 32 tasks for running AeroBasic programs. Use the EnabledTasks Parameter to configure the number of enabled tasks.

Optional Purchase Required: You can only enable more than five tasks if you have purchased the Professional option. For more information, refer to License Options for more information.

By default, the controller executes one line of an AeroBasic program every millisecond. You can use the ExecuteNumLines Parameter to change the maximum number of AeroBasic program lines that execute every millisecond. When you use the velocity profiling (VELOCITY ON), corner rounding, or cutter radius compensation features, the controller executes up to the number of lines that are in MaxLookaheadMoves Parameter. If velocity profiling (VELOCITY ON), corner rounding, or cutter radius compensation are not in operation and the task is in automatic wait mode (WAIT MODE AUTO Command), the controller executes up to the value of the ExecuteNumLines parameter or 8 lines, whichever is larger, when a coordinated (LINEAR, CW, CCW, BEZIER) or RAPID move completes.

You can use the CRITICAL START Command to cause the task to hold its time slice to execute AeroBasic program lines indefinitely.

If the program contains a command that requires more than one millisecond to execute (such as RAPID, LINEAR, CW, CCW, DWELL, ENABLE, DISABLE, SETPOSCMD, WAIT, or Callback Commands), the task stops trying to execute lines and releases execution to the next task.

One task polling cycle includes all of the enabled tasks and occurs every millisecond. The polling cycle is executed as follows:

FOR EACH TASK
    Check feedhold, MFO, MSO, and PROBE conditions, and react as required.
    Examine ON command conditions, and react as required.
    Execute immediate commands if any are pending.
    Execute a canned function if one is pending.
    IF Task is in the ProgramRunning state THEN
        Examine ONGOSUB command conditions and react as required.
        Execute one or more lines of the current AeroBasic program.
    END IF
END FOR
