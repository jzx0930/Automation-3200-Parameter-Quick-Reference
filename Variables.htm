Variables

Variables are variants whose values can be changed by initialization and the assignment command.

The following table summarizes the variables that are available, based on type and scope. Each column represents a type available on the A3200. The first row is an example syntax of a constant of that type. The remaining rows give syntax examples for variables of that type.

In cases where variables are not available, use type conversions to accomplish the same task. For example, although axis names have no variables, strings do, and you can convert a string into an axis name.
