Connection Settings

To Open the Connection Settings Dialog

In the Network Explorer of either Configuration Manager or Motion Composer, right-click the Controller node and select Connection Settings.

The Drive Communication Tab

Use the settings in this tab to configure the Drive Communication card that you want to use with the A3200.

Figure: Connection Settings Dialog — Drive Communication Tab

 

Use the Drive Communication Interface settings to configure the FireWire or HyperWire card that you want to use to control the A3200 hardware devices. A Drive Communication card that is configured for the A3200 shows in the list. Refer to Configuring and Verifying the FireWire or HyperWire™ Card for more information.

The Fieldbus Tab

Use the settings in this tab to configure the Fieldbus settings you want to use with the A3200.

Figure: Connection Settings Dialog — Fieldbus Tab

Use the Fieldbus settings to configure the Ethernet card that you want to use with the A3200 PC Modbus and the configuration file. Refer to PC Modbus Overview for more information on PC Modbus and Fieldbus Options Overview for more information on A3200 Hilscher Fieldbus.

The Connection Mode Tab

Use the settings in this tab to configure the Connection Mode of the A3200

Figure: Connection Settings Dialog — Connection Mode Tab

The Connection Mode is usually set to local, which is the default setting. With a local connection, the A3200 operates from the same machine where the A3200 SMC is installed. You can also operate the A3200 under a Client/Server architecture. Under the Client/Server architecture, two computers connected by Ethernet are used to control the A3200. Refer to Remote Server and Client Installation and Configuration for more information on Client/Server.

The Advanced Tab

Use the settings in this tab to configure the advanced options of the A3200.

Figure: Connection Settings Dialog — Advanced Tab

The Start A3200 selection determines how and when the A3200 system is started. When you select Automatically, the A3200 and Program Automation are initialized the first time you log into the computer. When you select On-Demand, which is the default setting, the A3200 is not initialized until it is required. For example, the A3200 is initialized when you open one of the A3200 applications and connect.

Use SMC Arguments to set arguments to the SMC that change its functionality. For example, to operate the A3200 without a FireWire card, set the -G option. Refer to SMC Arguments List for a complete list of SMC Argument options.

Use Communication Service Arguments to set the arguments to the Communication Service. These arguments are typically used for debugging purposes, and you should only change them when instructed to do so by Aerotech.
