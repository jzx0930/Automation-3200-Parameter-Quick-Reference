Changed Parameter Reference

Changed parameters can have an effect on your programs. Sometimes they require you to make changes to the AeroBasic code. The functionality of a parameter can change in all of the ways that follow:

  • Functionality Removed - The parameter and its functionality were fully removed. No equivalent functionality was added.
  • Renamed to - Only the parameter name changed. The functionality stayed the same.
  • Replaced by - The functionality of the parameter was replaced by an equivalent functionality that you can use to do the same or an equivalent operation. Refer to the examples that follow.
  • Examples of Replaced Parameter Functionality

    Parameters can be replaced by:

    • A different parameter with an equivalent functionality
    • An AeroBasic command
    • A combination of different parameters with an equivalent functionality and AeroBasic commands
    • A combination of AeroBasic commands

Some parameters are different in different software versions. The sections that follow give you the changes in parameter names and functionality. They also show parameters that are no longer available for each software version.
