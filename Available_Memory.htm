Available Memory

The SMC is a real-time component of the A3200 that uses nonpaged kernel memory, which is a subset of the total PC memory. The Controller tab of the Status Utility shows the quantity of nonpaged kernel memory that the A3200 currently uses.

Figure: The Status Utility

To see the quantity of nonpaged kernel memory that the full system uses, open Windows Task Manager. This includes the A3200 and other PC applications.

Figure: Task Manager

To see quantity of nonpaged kernel memory that is currently available (free) on your machine, open the Console utility and use the MEMINFO command. Do not run this command while you are running motion or the axes might jerk.

Figure: Console

These activities can use large quantities of memory:

  • Digital Scope plotting buffers. Refer to DataCollectionItems Parameter and DataCollectionPoints Parameter for more information.
  • Running large programs.
  • Declaring large numbers of global or task variables. Refer to GlobalDoubles Parameter for more information.

User programs, cam tables, global variables, and so on are all limited by available memory. Adding more axes does not use additional memory. Decreasing the number of enabled tasks does not have an effect on available memory.
