Configuring Parameters for Position Feedback Averaging on Multi-Axis Drives

By default, the position feedback of the axis is the specified channel of the PositionFeedbackChannel Parameter. If you want to use more feedback channels in the position feedback, you can specify them for the PositionAveragingChannel Parameter. The controller will average the feedback signals from the specified channels of the PositionFeedbackChannel and the PositionAveragingChannel parameters. Then the axis uses the average of these signals as the position feedback. The controller averages the feedback signals after you configure the applicable parameters, enable Feedback Averaging, and home the axis.

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

TO AVERAGE FEEDBACK SIGNALS FROM MULTIPLE FEEDBACK CHANNELS AS THE POSITION FEEDBACK

  1. Connect each encoder to the drive so that all the encoders count in the same direction.
  2. Open Configuration Manager.
  3. Make sure the PositionFeedbackType Parameter is set to one of the settings that follow:
    • On an Npaq, this parameter can be set to Encoder Multiplier or Local Encoder Counter based on the feedback type you are using.

      • Note: If you set the PositionFeedbackType Parameter to Encoder Multiplier on the Npaq, you must use the same EncoderMultiplicationFactor for the two axes used by the PositionFeedbackChannel and PositionAveragingChannel parameters.
    • On all other drives, this parameter must be set to Local Encoder Counter.
  4. For the PositionFeedbackChannel Parameter, specify the channel that you want to use for the position feedback device. The axis uses this channel as its position feedback until you enable Feedback Averaging and home the axis.

    5. NOTE: You can use the PositionFeedbackChannel Parameter to let any axis of the drive use feedback from any feedback channel. If you select Default, the axis gets feedback from the channel that corresponds to its axis index within the unit. For example, Axis 0 uses Channel 0, Axis 1 uses Channel 1, and so on.
  5. For the PositionAveragingChannel Parameter, do the steps that follow:
    1. Make a decision about the additional channels from which you want the controller to average feedback signals as part of the position feedback. The controller will average these signals with the feedback signals from the channel specified for the PositionFeedbackChannel Parameter.
    2. Specify the channels that you selected for the PositionAveragingChannel Parameter.
  6. For the FeedbackSetup Parameter, do the steps that follow:
    1. Set the Feedback Averaging option to Enabled.
    2. Set the Position Feedback Direction option to Normal.
  7. Save the parameter file. Then reset the controller for these changes to have an effect.
  8. Open Motion Composer.
  9. Enable the axis on which you configured the parameter values.
  10. Home the axis. After the home cycle completes, the controller averages the feedback signals from the channels that you specified for the PositionFeedbackChannel and PositionAveragingChannel parameters. The axis uses the average of these signals as the position feedback.
