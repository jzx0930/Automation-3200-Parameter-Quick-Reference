Global Variable Names

Key Concepts

Program Automation

#define statements

Editing the project file

Objective

Define descriptive global variables names.

Defining Descriptive Global Variable Names

Create a program file (or arbitrary name) with the commands that follow.

Example

#define SlowSpeed $global[7]
#define FastSpeed $global[8]
' ...etc...

Use Program Automation through Configuration Manager or Motion Composer to add this file to all tasks. Select the Auto Include execution type.

Reset the controller and use the names SlowSpeed and FastSpeed in your programs. It is important to know that these names are case sensitive and the $global names are not case sensitive.

The Watch Window in Motion Composer still shows the $global names.
