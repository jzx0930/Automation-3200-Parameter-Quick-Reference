Configuring Parameters for Gantry Mode

Hardware: You cannot use Gantry mode with the GL4, the Ndrive FCL, FLS, QL, or QLe, the Nmark CLS, GCL, or SSaM, or the XL4s.

Use the GantrySetup parameter to set the gantry modes. For more information about the different gantry modes, refer to GantrySetup parameter.

Note: Gantries that use two drives, such as an Ndrive CP and an Ndrive CP, must use the same drive type for the master and slave axes. But, gantries cannot use two multi-axis drives. For example, if a gantry master axis is set on one Npaq and a gantry slave axis is set on a different Npaq, an error occurs. For more information, refer to GantryConfiguration in the Troubleshooting section of the Help file.

Gantry systems are configured so that the limits are linked. If you configure your own gantry, make sure that you link the limits of the master and slave axes or an error can occur.

to configure axes as a gantry

For Position Gantry Mode (GantrySetup = 1)
  1. Configure all of the axis parameters for the master axis.

  2. NOTE: The gantry master communications channel must be lower than the gantry slave communications channel. If the master communications channel is higher, you receive an error message.

  3. Configure the HomeType parameter on the master axis.

  4. NOTE: A position gantry must use a HomeType that includes the Marker.

  5. Use the following procedure to configure the parameters for the slave axis.
    1. In the Network Explorer of Configuration Manager, go to Computer → Open Files. Expand the parameter file that you want to configure.
    2. Drag the Axis folder onto the file name of the parameter file. The Copy Parameters dialog comes into view.
      • The name of the parameter file that you are using is specified as the Source file. The Destination file is specified as Same as source.
      • In the Parameter Categories section of the Copy Parameters dialog, the Axis folder is selected. This means that only the Axis parameters are copied.
    3. In the Parameter Mapping section of the Copy Parameters dialog, use the Axis tab to specify how the axes are mapped from the Source (which is the master axis) to the Destination (which is the slave axis). The Source and the Destination cannot be the same axis.
    4. Verify your selections and click OK.
  6. Set the GantryMasterAxis parameter for the slave axis.
  7. Enable the correct gantry mode by setting the GantrySetup parameter for the slave axis.
  8. Configure the HomeOffset parameter on the master axis.
  9. Save the parameter file and reset the A3200 controller.
  10. Right-click the parameter file and select Use As Active Parameters to make this file the active parameter file.

After you configure the gantry, use the following procedure to calculate the value of the HomeOffset parameter for the slave axis.

To calculate the value of the HomeOffset parameter

  1. Connect to the SMC.
  2. In the active parameter file, select the HomeOffset parameter for the slave axis. In the Editor section of Configuration Manager, the Calculator button comes into view.
  3. Click the Calculator button. The Gantry Home Offset Calculator comes into view.
  4. Figure: Gantry Home Offset Calculator

  5. In the Gantry Home Offset Calculator, set the Max Current to a safe value (usually 1 - 2 amps). This is the temporary maximum allowed current used during the calculation procedure.
  6. In the Gantry Home Offset Calculator, click Calculate.

  7. WARNING: This calculation procedure enables and homes the master axis and slave axis. Before you continue with this procedure, make sure that it is safe to enable and home the master axis and slave axis.
  8. The Gantry Offset Calculation dialog comes into view. Click OK.
  9. The Gantry Offset Calculation warning dialog comes into view and specifies that the gantry is about to move. Make sure that it is safe for the gantry to move and click OK.
  10. Verify that the calculated HomeOffset parameter for the slave axis is correct and click OK.
  11. Save the updated parameter file and reset the A3200 controller.

to verify the new parameter configuration.

  1. Enable the master axis. (The slave axis also enables at this time.)
  2. Home the master axis. (The slave axis also homes at this time.)
  3. Move the gantry to the approximate center of travel.
  4. Open Digital Scope and configure the plot as follows.
    • Collect 1000 points.
    • Select the Current Feedback for the master axis and slave axis.
  5. Collect one set of data in Digital Scope and monitor the motor currents. Make sure that the motor current for both the master axis and slave axis is less than 0.5 amps.
For Current Gantry Mode (GantrySetup = 2, 3, or 4)
  1. Configure all axis parameters for the master axis.
  2. NOTE: The gantry master communications channel must be lower than the gantry slave communications channel. If the master communications channel is higher, you receive an error message.
  3. Configure the HomeType parameter on the master axis.
  4. Use the following procedure to configure the parameters for one or more slave axes. 
    1. In the Network Explorer of Configuration Manager, go to Computer → Open Files. Expand the parameter file you want to configure.
    2.  Drag the Axis folder onto the file name of the parameter file. The Copy Parameters dialog comes into view.
      • The name of the parameter file that you are using is specified as the file Source. The file Destination is specified as Same as source.
      • In the Parameter Categories section of the Copy Parameters dialog, the Axis folder is selected. This means that only the Axis parameters are copied.
  5. Set the GantryMasterAxis parameter for the slave axes.
  6. Enable the correct gantry mode by setting the GantrySetup parameter for the slave axes.
  7. If the GantrySetup parameter is set to Current Gantry Mode (Synchronized Commutation) or Current Gantry Mode (Offset Commutation), set the PositionFeedbackType parameter to 0 for the slave axes.
  8. Save the parameter file and reset the A3200 controller.
  9. Right-click the parameter file and select Use As Active Parameters to make this file the active parameter file.

Make the axis parameters the same on the master axis and slave axis, with the following exceptions.

  • The names of the axes must be different.
  • The GantryMasterAxis and GantrySetup parameters. Refer to the To configure axes as a gantry section of this topic.
  • The HomeOffset parameter if GantrySetup = 1. Refer to the To configure axes as a gantry section of this topic.
  • The ESTOP input should be connected to only the master axis.
