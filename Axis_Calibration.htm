Axis Calibration

Axis calibration tables apply position corrections to an axis based on the position of a master axis. The SMC software continuously reads the master axis position and uses this to find a corresponding correction value from the calibration table. The correction value is then added to the commanded axis position.

You can use calibration tables for:

 

Accuracy Correction

if the master and corrected axis are the same
 

Orthogonality Correction

if the master and corrected axis are different
 

Straightness Correction

to compensate for the variation in straightness of table travel
 

Temperature Compensation

to correct for thermal expansion
  Thermal Compensation to correct for changes to the thermal state of the stage
