Conditional Plotting

Use conditional plotting to configure plots that you collect in Digital Scope to show data differently based on input from the controller. You can set data that is collected from an axis to show as a different color, or to not show at all, based on the state of the controller's digital outputs and inputs.

Conditional plotting is set by axes. If you enable conditional plotting for an axis, the conditions that you select for that axis apply to all values that are collected from that axis. You can configure conditional plotting on multiple axes, and you can have multiple sets of conditions on the same axis.

Use the following procedure to configure conditional plotting.

  1. In Digital Scope, on the Configuration menu, click Conditional Plotting. The Conditional Plot Formatting dialog comes into view.

    Figure: Conditional Plot Formatting Dialog

  2. In the Conditional Plot Formatting dialog, click the Signals tab.
  3. In the Signal Conditional Plotting section, select Enable Signal Conditional Plotting.
  4. In the Axis menu, select the axis on which you want to use conditional plotting.
  5. In the Signal Type menu, select the type of signal from the controller on which you want to trigger conditional plotting.
  6. In the Signal Name menu, select the specific signal that you want to use to trigger conditional plotting. For Digital Input and Digital Output signal types, the signal must be a bit number.
  7. In the Apply Action When Signal Is section, select On or Off to do one of the following.
    • Select On if you want conditional plotting to occur when the signal that you select is on.
    • Select Off if you want conditional plotting to occur when the signal that you select is off.
  8. In the Plot Action section, select the action that you want to occur when the signal that you select is On or Off.
  9. Click the Add button to add your specified conditions to conditional plotting. Then, click OK.
  10. Click the Remove button to remove a set of conditions from conditional plotting.
