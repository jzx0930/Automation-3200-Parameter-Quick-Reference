Autotuning an Axis

Autotuning an axis involves exciting the axis using a predefined input, measuring the resulting data, and calculating a set of servo gains that match a given criteria. The goal of the autotune algorithm is to translate a set of tuning specifications, described by the open-loop crossover frequencyThe open-loop crossover frequency is the frequency at which the open-loop gain is equal to one (0 dB). The simplest model for motor has a single open-loop crossover frequency. The open-loop crossover frequency is a measure of system performance. A higher crossover frequency implies better system performance through improved position tracking and decreased response time. Too high of a crossover frequency can however degrade system performance by decreasing damping in the system and causing instabilities in the servo-loop. and phase marginThe phase margin of a system is defined as the difference, in degrees, of the phase of the open-loop system at the crossover frequency and -180 degrees. The phase margin is a measure of system stability and provides a measure of robustness against variations in system dynamics. Increasing the phase margin increases system stability while decreasing the phase margin decreases stability and can eventually lead to an unstable system. If the phase margin is less than or equal to zero then the system is unstable. A larger phase margin results in a less oscillatory and better-damped system. If an open-loop system has multiple crossover frequencies, the overall phase margin is taken as the worse case or lowest phase margin. Typically, a phase margin between 30 and 60 degrees results in a reasonable trade-off between system performance and stability., into a set of servo gains. The autotune functionality is supplied by the Tuning toolbar in the Scope tab of Digital Scope.

To Run an Autotune

NOTE: This procedure applies only to closed-loop autotune. In the Scope tab of Digital Scope, on the Configuration menu, make sure that Autotune Type is set to Closed-Loop.
  1. In Digital Scope, select the axis that you want from the axis list in the main menu or from the Axis box on the Axis toolbar.
    2. Figure: Axis Toolbar

  2. Enable the axis by clicking the Enable button on the Axis toolbar.
  3. In the Tuning toolbar, set the autotune amplitude to an appropriate value. A recommended starting value is approximately 1 inch (25.4 mm) for an axis that produces linear motion and 180° for an axis that produces rotational motion.
  4. You can change the units of the amplitude value by selecting a new value on the Units menu.
    5. Figure: Units Menu

  5. Set the excitation frequency, labeled Freq (Hz), to 1.5.
  6. Set the crossover frequency, labeled Fc (Hz), to 30.
  7. Set the phase margin, labeled PM (Deg), to 60.
  8. Start the autotune cycle by clicking the Tune button on the Tuning toolbar.
    9. Figure: Tuning Toolbar — Tune Button

  9. Now the system should oscillate back and forth at three different frequencies. After the autotune algorithm completes collecting data and calculating new servo gains, the plot is updated and the Autotune Results dialog shows the calculated servo gains.
    10. Figure: Autotune Results Box

  10. Click Accept to accept the gains that are selected or Cancel to discard the newly calculated gains. The autotune result screen shows an estimate of the quality of the autotune measurement from the collected data. If the quality estimate is less than 50%, we recommend that you do not accept the gains. In this case, the quality estimate might be improved by increasing the autotune amplitude and/or excitation frequency. Refer to steps 3 and 4.
  11. If the system is stable after an autotune cycle, you can increase the crossover frequency in small increments (5 to 10 Hz) and repeat the autotune cycle until you get an acceptable system performance. If the system is unstable, the crossover frequency might be too high.
  12. If you accept the autotune gains, the gains are active on the controller but they are not saved to the file system. To save the gains to the file system, click the Save button on the Servo Gains toolbar.
    13. Figure: Servo Gains Toolbar
