Data Collection in Digital Scope

To collect data in Digital Scope, you must select a set of signals, set the number of points to collect, and trigger data collection. Digital Scope always collects a standard set of signals and the signals that you select.

Collecting Signals and Configuring a Plot

To configure data collection and collect signals in a 1D (one-dimensional) or 2D plot in Digital Scope, click the Scope tab and use one of the following procedures.

To Configure a 1D Plot
  1. Click Configuration.
  2. On the Configuration menu, select Configure Data Collection.
  3. In the Configure Data Collection dialog, click the Dimensions arrow and select 1D.
  4. To collect Axis, System, or Task signals, use one of the following procedures. The signals that you specify to be collected are shown in the Summary tab. Axis, system, and task signals that you add are also shown in the Configured Signals list of each signal tab.
  5. To Collect Axis Signals
    1. Click the Axis tab.
    2. In the Axes list, select the axis that you want to use.
    3. In the Signals list, select the signal that you want to collect.
    4. Click the Add button .
  6. To Collect System Signals
    1. Click the System tab.
    2. In the Signals list, select the signal that you want to collect.
    3. Click the Add button .
  7. To Collect Task Signals
    1. Click the Task tab.
    2. In the Tasks list, select the task that you want to use.
    3. In the Signals list, select the signal that you want to collect.
    4. Click the Add button .
  8. To remove Axis, System, or Task signals, use one of the following procedures.
  9. To Remove Axis Signals
    1. Click the Axis tab.
    2. In the Configured Signals list, select the row that contains the signal that you want to remove.
    3. Click the Remove button .
  10. To Remove System Signals
    1. Click the System tab.
    2. In the Configured Signals list, select the row that contains the signal that you want to remove.
    3. Click the Remove button .
  11. To Remove Task Signals
    1. Click the Task tab.
    2. In the Configured Signals list, select the row that contains the signal that you want to remove.
    3. Click the Remove button .
  1. After you select all of the signals that you want to collect, click OK and continue to Collect Points.
To Configure a 2D Plot
  1. Click Configuration.
  2. On the Configuration menu, select Configure Data Collection.
  3. In the Configure Data Collection dialog, click the Dimensions arrow and select 2D.
  4. In the X Axes list, select an axis for the X (horizontal) plot axis.
  5. In the X Signals list, select a signal for the X (horizontal) plot axis.
  6. In the Y Axes list, select an axis for the Y (vertical) plot axis.
  7. In the Y Signals list, select a signal for the Y (vertical) plot axis.
  8. Click the Add button . The signals that you added are shown in the Configured Signals list.
  9. To remove a signal that you have added, use the following procedure.
    1. In the Configured Signals list, select the row that contains the signal that you want to remove.
    2. Click the Remove button .
  1. After you select all of the signals that you want to collect, click OK and continue to Collect Points.

Collecting Points

Use the following procedure to configure the number of points that you want to collect.

  1. Click Configuration.
  2. On the Configuration menu, select Points.
  3. In the Points box, enter the number of points that you want to collect and continue to Collect Data.

Collecting Data

You can collect either a single set of data or multiple consecutive sets of data.

  • To collect a single set of data, on the Action menu select Collect Data.
  • To collect multiple consecutive sets of data, on the Action menu select Collect Snapshots.

NOTE: During data collection, if you click the Stop button Digital Scope stops collecting data, and retrieves and plots that data that it has collected.

NOTE: If you click the Stop After Completion button while you are collecting data snapshots, Digital Scope completes collection of the current data set, retrieves and plots the data, and then stops collecting data.

Saving and Loading Data Collection Configurations

You can save the signals that you currently have selected to an external file. You can also load configurations that you previously saved.

  • To save your current configuration, click the Export Configuration button . The 1D and 2D signals that you currently have selected are saved as a .dcca file.
  • To load a configuration that you previously saved, click the Import Configuration button and select the .dcca file that you want to load.
