J111 on the Nservo

J111 on the Nservo supplies six bytes of additional I/O (48 extra bits). You can configure each byte as either an input or an output by using the PORT command. To read and write data, use the $DO command and the $DI and $PORT functions.

To map J111 (48 bits) to the $DI or $DO

The J111 connector uses digital bits 16 through 63. You can access these bits using the following syntax.

for $DI

$DI[16].X through $DI[31].X, $DI[00].Y through $DI[31].Y

for $DO

$DO[16].X through $DO[31].X, $DI[00].Y through $DI[31].Y
