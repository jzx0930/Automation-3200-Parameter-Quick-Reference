IFOV: Verifying the Hardware Configuration

After you configure the parameters and commands and you connect the servo drive to the GL4 or the Nmark CLS, GCL, or SSaM, verify the hardware configuration before you try to run a program in IFOV mode.

TO VERIFY THE HARDWARE CONFIGURATION for FireWire drives
  1. Open Motion Composer.
  2. Open the Status Utility. Then select the Diagnostics tab.
  3. Npaq only. If you are using an Npaq, issue the ENCODER OUT ON/OFF Command. Refer to the example applicable to your Npaq:

  4. Example: Npaq with Quadrature

    ' ENCODER OUT X ON <Channel1>, <Channel2>, <Channel3>
' <Channel1> Argument=1: Echo encoder channel 0 to J8 (outputs 8 and 9)
' <Channel2> Argument=2: Echo encoder channel 1 to J8 (outputs 10 and 11)
' <Channel3> Argument=0: Not used
ENCODER OUT X ON 1, 2

    Example: Npaq with -MXR Feedback

    ' ENCODER OUT X ON <Channel1>, <Channel2>, <Channel3>
' <Channel1> Argument=7: Echo encoder channel 6 to J8 (outputs 8 and 9)
' <Channel2> Argument=8: Echo encoder channel 7 to J8 (outputs 10 and 11)
' <Channel3> Argument=0: Not used
ENCODER OUT X ON 7, 8
  5. In Motion Composer, enable the servo axes.
  6. Jog the servo axes.
  7. In the Status Utility, look at the Auxiliary Position Feedback signal on the galvo axes for the Nmark CLS, GCL, or SSaM.
  8. Verify that the Auxiliary Position Feedback signal on the galvo axes counts in the same direction as the Position Feedback signal on the servo axes:
    • If the value of the Auxiliary Position Feedback signal on the galvo axes does not change, then there is a problem with the hardware configuration or the wiring.
    • If the Auxiliary Position Feedback signal on the galvo axes counts in the opposite direction, then the wiring polarity is reversed. Refer to IFOV: Verifying the Wiring Polarity for more information.
TO Verify the Hardware Configuration FOR HyperWire Drives
  1. Open Motion Composer.
  2. Open the Status Utility. Then select the Diagnostics tab.
  3. XC4, XC4e, and XR3 only. Issue the ENCODER OUT ON/OFF Command. Refer to the example that follows:

  4. Example: XC4 and XC4e

    ' Echo X-axis primary feedback to the SYNC A port.
ENCODER OUT X ON 0
				
' Echo Y-axis primary feedback to the SYNC A port.
ENCODER OUT Y ON 0

  5. Example: XR3

    ' Echo X-axis primary feedback to HOUT (X-axis high-speed outputs 0 and 1)
ENCODER OUT X ON 0
				
' Echo Y-axis primary feedback to HOUT (Y-axis high-speed outputs 0 and 1)
ENCODER OUT Y ON 0
  6. In Motion Composer, enable the servo axes.
  7. Jog the servo axes.
  8. In the Status Utility, look at the Auxiliary Position Feedback signal on the galvo axes for the GL4.
  9. Verify that the Auxiliary Position Feedback signal on the galvo axes counts in the same direction as the Position Feedback signal on the servo axes:
    • If the value of the Auxiliary Position Feedback signal on the galvo axes does not change, then there is a problem with the hardware configuration or the wiring.
    • If the Auxiliary Position Feedback signal on the galvo axes counts in the opposite direction, then the wiring polarity is reversed. Refer to IFOV: Verifying the Wiring Polarity for more information.
