Controller Configuration

This section supplies information on configuring the A3200 controller. This configuration is done primarily through the use of parameters and axis calibration files, which are described in detail in this section. It also contains instructions for configuring some less commonly used controller features as well as general reference information about the A3200.

Parameters

On the A3200 most controller configuration takes place through the use of parameters. Parameters contain settings that affect system, axis, and task behavior, and parameters generally establish the default behavior for the controller.

The A3200 stores parameter settings in parameter files. You can create and edit parameter files and change parameter settings in the Configuration Manager. For an overview of the parameters that are available on the A3200, refer to Parameter File Configuration.

Axis Calibration

It is sometimes necessary to correct the position of an axis to correct for errors in the mechanics, feedback device, or alignment of the stages. You can use the axis calibration feature of the A3200 to apply position correction values from a calibration table to the axis position. For more information about this feature, refer to Axis Calibration.

Configuration Examples

To simplify the setup of systems with uncommon or unique configurations, Aerotech offers several examples of how to configure parameters for these systems. The topics under Configuration Examples explain the necessary parameter settings, and they include additional information to assist you in configuring your system.

Additional Information

For reference, this section includes diagrams and other information that explain various aspects of controller behavior.
