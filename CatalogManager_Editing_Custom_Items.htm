Editing Custom Items in the Catalog Manager

You can edit custom items that are in your catalog. Refer to the procedure that follows.

To Edit Custom Items in the Catalog

  1. In the Catalog section of the Catalog Manager dialog, select the custom item that you want to edit.
  2. On the Edit menu, select Edit Item. Or click the Edit Item button on the toolbar.
  3. In the Mechanics section of the dialog, edit the properties of the item that you selected. If it is necessary to cancel your changes, click the Discard button .
    4. Figure: Editing a Custom Item in the Catalog Manager

  4. NOTE: If you open a parameter file in the Axis Parameter Calculator and that file uses catalog items that you edited in the Catalog Manager, the calculator tells you to apply those changes to the parameter file. This includes changes that you make to the properties and the name of a custom item.
  5. To apply the changes to the item, click the Done button . The Catalog section of the dialog shows an asterisk adjacent to this item to identify changes that you must save to the catalog file.
  6. Optional. If you want to undo the changes to a custom item, refer to the steps that follow.
    1. In the Catalog section of the dialog, select an edited item for which you want to undo the changes.
    2. On the Edit menu, select Undo Item Changes. Or click the Undo Item Changes button on the toolbar. The Catalog Manager restores the properties of the item to the previous changes that you applied.
  7. Click Apply to save all of your changes to the catalog file.
