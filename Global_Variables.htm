Global Variables

Syntax:

 
$global[<Expression>]
 
$strglob[<Expression>]
 
$aptglob[<Expression>]

Example:

 
$global[4] = 5
 
$strglob[4] = "Mystring"
 
$aptglob[4+9 ] = X4 Y6.8

Global variables are accessible from any program on any task and are initialized to zero (or NULL ) when the controller resets. Global variable names are not case sensitive.

Global variables occur in three types: doubles, strings, and axis points.  The number of global variables available is defined by the GlobalDoubles, GlobalStrings and GlobalAxisPoints global parameters. If you increase these numbers, new global variables are allocated.

Global variables begin with index 0 and may be referenced by expression. This must be a positive integer (or zero) or a runtime error will occur.
