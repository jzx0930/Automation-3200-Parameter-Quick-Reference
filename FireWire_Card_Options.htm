FireWire or HyperWire™ Card Options

The following sections describe the optional FireWire or HyperWire card configurations.

Operation without a FireWire or HyperWire Card 

For testing (compiling and writing programs only), you can operate without a FireWire or HyperWire card (no drives are connected).

To configure the system to run without a FireWire or HyperWire card (virtual)

  1. In the Network Explorer of Configuration Manager, right-click Controller and then click Connection Settings.
  2. The Connection Settings dialog comes into view. Select the Drive Communication tab.
  3. Select Operate without a card.
  4. Click OK and then close Configuration Manager. Close all other A3200 applications.
  5. Shut down the A3200. Click Start → All Programs → Aerotech → Automation 3200 → Shutdown A3200. A black Command Prompt window comes into view momentarily.
  6. Open Configuration Manager or any other A3200 application. All axes then operate as virtual.
Multiple FireWire or HyperWire Cards in Your System

If you have multiple FireWire or HyperWire cards in your PC, you need to specify which card you want the Automation 3200 to use.

To select a FireWire or HyperWire card

  1. In the Network Explorer of Configuration Manager, right-click Controller and then click Connection Settings.
  2. The Connection Settings dialog comes into view. Select the Drive Communication tab.
  3. Click Select the card to use and select the applicable card.
    • If the card is listed with a blue check mark, then that card is configured for use under the INtime real-time operating system.
    • If the card is listed with a yellow warning, then that card cannot be used with A3200. Refer to Compatible FireWire or HyperWire™ Cards .
    • If the card is not listed, then refer to Configuring and Verifying the FireWire or HyperWire™ Card to put the card under INtime control.
  4. Click OK and then close Configuration Manager. Close all other A3200 applications.
  5. Shut down the A3200. Click Start → All Programs → Aerotech → Automation 3200 → Shutdown A3200. A black Command Prompt window comes into view momentarily.
  6. Open Configuration Manager and try to connect to the controller. If you see an error message about FireWire or HyperWire cards, record the text of the message and contact Aerotech Global Technical Support.

Note: If you selected a specific FireWire or HyperWire card in Connection Settings and then you make changes to your PC peripherals, such as installing a network card, you might need to repeat the steps to select a FireWire or HyperWire card to correct your configuration.
