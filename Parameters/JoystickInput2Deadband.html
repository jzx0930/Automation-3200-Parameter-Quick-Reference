JoystickInput2Deadband Parameter

Default Value

0.1

Minimum Value

0.0

Maximum Value

1.0

Units

Volts

Type

64-bit IEEE floating point

Use the JoystickInput2Deadband parameter to specify a deadband voltage range for the axis after you specify an analog input channel with the JoystickInput2 parameter. The deadband range is ± the value to which you set this parameter around the central voltage. Use the following formula to calculate the central voltage.

The deadband has two main functions.

  • The deadband defines a voltage range where the joystick is centered and no axis motion is directed, which prevents small offsets from generating unwanted motion.
  • After you issue the JOYSTICK command, the joystick will not operate until the joystick enters the deadband voltage range a minimum of one time. All axes must be in the deadband for the joystick to operate. This is a safety feature and prevents the joystick from commanding high axes velocities when it activates.

This parameter is used only when you use the 3D option in the JOYSTICK command.
