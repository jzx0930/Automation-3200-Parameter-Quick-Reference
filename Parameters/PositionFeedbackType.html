None (Open-Loop)

The position feedback is set equal to the position command, thus the position error is forced to zero. This effectively bypasses the servo loop and is typically used only with a stepper motor or to configure a Null Axis.

1

Local Encoder Counter

The axis gets position feedback from its local encoder input. This setting is used with RS-422 compatible digital quadrature encoders.

2

Encoder Multiplier

The axis uses the optional primary or auxiliary encoder multiplier for position feedback. You can use this option with an encoder that produces analog signals to increase the effective encoder resolution of the axis. When you use this feedback type, you must configure the parameters in the Feedback > Multiplier Parameter Category.

3

Analog Input

The axis uses one of its available analog inputs for position feedback. When you use this feedback type, the analog input is treated as an absolute position transducer. Specify the analog input number by using the PositionFeedbackChannel parameter.

4

EnDat Absolute Encoder

The axis uses an EnDat absolute encoder for position feedback. The drive communicates serially with the encoder in order to initialize the servo position, thus eliminating the need to home the axis. You must configure the EnDAT serial interface by using the parameters in the Feedback > EnDat Encoder Parameter Category.

After initialization, this feedback type behaves similarly to the Encoder Multiplier type by using the analog signals from the encoder to incrementally update the feedback position. Because this feedback type uses the primary encoder multiplier, you must configure the parameters in the Feedback > Multiplier Parameter Category.

5

Hall-Effect Switches

The axis uses Hall-effect switches for position feedback. When you use this feedback type, you must set the CyclesPerRev parameter to 0 to force the motor commutation to operate in six-step mode, using only the Hall-effect switches.

6

Resolver

The axis uses the optional resolver interface for position feedback. When you use this feedback type, you must configure the parameters in the Feedback > Resolver Parameter Category.

7

Parallel Laser Interface

The axis uses the optional parallel laser interface for position feedback. This setting is available only on the Npaq.The drive communicates with a ZYGO laser interferometer through the parallel laser interface to read the position feedback.

8

Serial Encoder

This is a custom feedback type that is not supported in all configurations. Do not use this setting unless you are instructed to do so by Aerotech personnel.

9

Resolute Absolute Encoder

The axis uses a Resolute absolute encoder for position feedback. The drive communicates serially with the encoder in order to initialize the servo position, eliminating the need to home the axis. You must configure the Resolute serial interface by using the parameters in the Feedback > Resolute Encoder Parameter Category.

10

Position Detector Feedback

The axis uses the internal position detector of the galvo head. This setting is available only on the Nmark CLS.

11

Capacitance Sensor

The axis uses the optional capacitance sensor interface for position feedback. When you use this feedback type, you must configure the parameters in the Feedback > Cap Sensor Parameter Category.
