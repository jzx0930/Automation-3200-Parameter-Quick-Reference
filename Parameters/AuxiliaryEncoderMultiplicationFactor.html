AuxiliaryEncoderMultiplicationFactor Parameter

Default Value

1.0

Minimum Value

1.0

Maximum Value

262144.0

Units

None

Type

32-bit IEEE floating point

Hardware: This parameter applies only when used with an applicable drive type and multiplier option. To see the drive types and multiplier options that are available for this parameter, refer to the table that follows.

Table:     Applicable Drive Types and Multiplier Options

Drive Type Multiplier Option

XC4e

Using an Encoder Multiplier with -MX3

XR3

Using an Encoder Multiplier with -MX4

NOTE: This parameter applies only when you set one of these parameters to the specified value:

  • If you set the PositionFeedbackType Parameter to Encoder Multiplier.
  • If you set the VelocityFeedbackType Parameter to Encoder Multiplier.

Note: If you change this parameter, you must adjust other parameters. Refer to Adjusting Parameters Based on System Changes for more information.

Use the AuxiliaryEncoderMultiplicationFactor parameter to specify the multiplication factor of the optional auxiliary encoder multiplier module of the drive. If you set this parameter to a value greater than 1, then the effective resolution of the encoder increases by the factor that you specify. The multiplier module is designed to work only with encoders that generate an analog output. The auxiliary encoder multiplier module applies only when you connect an encoder to the auxiliary encoder input.

The table that follows gives the maximum values of this parameter by drive type.

Table:     AuxiliaryEncoderMultiplicationFactor Maximum Values

Drive Type Absolute Maximum

Multiplier Option

XC4e

16,384

Using an Encoder Multiplier with -MX3

XR3

65,536

Using an Encoder Multiplier with -MX4

NOTE: If your HyperWire drive is configured to use an encoder multiplier with an -MX4 multiplier option, quadrature is generated only if the effective multiplication factor is greater than 4. The effective multiplication factor is equal to the value of the AuxiliaryEncoderMultiplicationFactor parameter divided by the value of the AuxiliaryEmulatedQuadratureDivider parameter.
