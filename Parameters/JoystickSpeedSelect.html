JoystickSpeedSelect Parameter

Default Value

0

Minimum Value

0

Maximum Value

N/A

Units

None

Type

32-bit signed integer

Use the JoystickSpeedSelect and JoystickAxesSelect axis parameters to specify which digital inputs are connected to the joystick buttons. Joystick Button A, Joystick Button B, and Joystick Button C are pressed by the defined digital input states of the JoystickSpeedSelect and JoystickAxesSelect parameters as shown in the following table.

Use Joystick Button A to change between axis pairs as defined by the JOYSTICK AXISGROUP command. Use Joystick Button B to change between axis speeds as specified by the JoystickLowSpeed and JoystickHighSpeed axis parameters. Use Joystick Button C to cancel or deactivate the joystick.

The default axis is 0 (typically X) (not used when specifying a Virtual Binary input type).

The input number is zero-based. The default value is input zero.
