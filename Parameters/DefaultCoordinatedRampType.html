DefaultCoordinatedRampType Parameter

Default Value

2

Minimum Value

0

Maximum Value

2

Units

None

Type

32-bit signed integer

Parameter Settings and Parameter File Values

The following table gives the value that is assigned in the parameter file when you select a parameter setting.

Table:     Parameter Setting Values

Option Settings Value

Ramp Type

Linear

0

SCurve

1

Sinusoidal

2

Note: To set the ramp type from within an AeroBasic program, use the RAMP TYPE Command.

Use the DefaultCoordinatedRampType parameter to set the default coordinated ramp type for coordinated motion such as BEZIER, CCW, CW, and LINEAR commands.

When you select Linear, the acceleration and deceleration during coordinated motion use a linear ramp type that has a constant rate.

When you select Sinusoidal, the acceleration and deceleration during coordinated motion use a sinusoidal ramp type.

When you select SCurve, the acceleration and deceleration during coordinated motion use an SCURVE ramp type.
