Use the ServoFilterSetup parameter to specify the second-order digital filters that you want to enable in the servo loop. To enable the digital filters, select the box for each filter that you want to use. Most A3200 drives have eight digital filters in the servo loop. The Ndrive FLS and Nmark CLS have four digital filters in the servo loop. The GL4, the XC2, the XC4, the XC4e, the XR3, and the Nmark GCL have sixteen digital filters in the servo loop.

The value of this parameter is set automatically when you use the Digital Filter Calculator in Configuration Manager or Digital Scope to calculate filter coefficients. The section that follows gives the option for this parameter.

Filter Enables

Servo Filter 0 Enabled is the default. Select the second-order digital filters that you want to enable on the controller.
