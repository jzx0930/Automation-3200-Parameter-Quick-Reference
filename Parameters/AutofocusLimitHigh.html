AutofocusLimitHigh Parameter

Default Value

0.0

Minimum Value

N/A

Maximum Value

N/A

Units

Primary Units

Type

64-bit IEEE floating point

Note: When you reference this parameter in an AeroBasic program, the parameter value is in the units specified by the unit mode of the task. If the task is in primary units mode, this parameter is in primary units. If the task is in secondary units mode, this parameter is in secondary units.

Hardware: Autofocus does not apply when used with the GL4, the Ndrive FCL or FLS, the Nmark CLS, GCL, or SSaM, or the XL4s.

Use the AutofocusLimitHigh parameter to set the autofocus loop upper limit. This limit is the maximum position to which the autofocus loop will command the axis to move. You can use the Autofocus Limits setting of the AutofocusSetup parameter to change how this value is used by the autofocus loop. The Autofocus Loop Block Diagram shows the structure of the autofocus loop.

The AutofocusLimitLow parameter must always have a lower numbered value than the AutofocusLimitHigh parameter. The arithmetic sign of the autofocus parameters is determined by the encoder direction. The following example shows you how to set the AutoFocusLimitLow and the AutoFocusLimitHigh parameters.

For Example

Your programmed autofocus range is -10 mm to 20 mm.

If ReverseMotionDirection is set to No on this axis, set the parameters to the following values.

Display Position Encoder Position Parameter Parameter Value

-10 mm

-10 mm

AutofocusLimitLow

-10

20 mm

20 mm

AutofocusLimitHigh

20

If ReverseMotionDirection is set to Yes on this axis, set the parameters to the following values.

Display Position Encoder Position Parameter Parameter Value

-10 mm

10 mm

AutofocusLimitHigh

10

20 mm

-20 mm

AutofocusLimitLow

-20

Note: Do not change the value of the AutofocusLimitHigh parameter or the AutofocusLimitLow parameter while the autofocus mode is active.
