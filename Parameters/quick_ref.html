Parameter Quick Reference

Each A3200 controller that has stage and motor assembly hardware must have a unique configuration with parameters that are kept in a file on your PC.

At the factory, the A3200 is initially configured with the default values of the parameters. If you purchased integration and the stage and motor hardware is assembled by Aerotech, a parameter file with the extension *.prma is included on your software installation device.

If your drive and your stage support FlashConfig, the axis parameters that you need are kept on the stage. When you connect a stage that has FlashConfig to a drive that supports FlashConfig, the controller automatically identifies the stage each time the controller is powered on or is reset. You can always restore the controller with the data used at the factory to initially configure the axis. For more information about FlashConfig parameters, refer to FlashConfig Overview.

System settings can have an effect on the values of your parameters. Use the calculators supplied by A3200 Configuration Manager to calculate the correct values. For information on calculators, refer to Using Calculators to Set Parameters.

You can also edit parameters in Configuration Manager.
