DecimalPlaces Parameter

Default Value

3

Minimum Value

0

Maximum Value

14

Units

None

Type

32-bit signed integer

Use the DecimalPlaces parameter to set the number of decimal places that show in the Axis Manager of Motion Composer and CNC Operator Interface. This parameter has an effect on only the Axis Manager. The Diagnostic Display of Motion Composer always shows values in counts.

If you set the CountsPerUnit parameter to 1, the position shows in counts with no decimal places, regardless of this value.

Motion Composer and CNC Operator Interface read the value of this parameter at startup. If you change the value of this parameter, you must restart the application for the changes to have an effect.
