Note: This parameter should only be set to a nonzero value if the axis is intended to operate in dual-loop mode, that is, if the axis has two different feedback devices.

Use the VelocityFeedbackType parameter to specify the type of feedback device that is used for velocity feedback. Not all feedback types are supported by all drive types. The table that follows shows the valid combinations.

Commutation information for brushless motors is typically derived from the velocity feedback device. This means that the CountsPerRev parameter value should be calculated based on the feedback device specified by this parameter.

0

None (Use Position Feedback)

The velocity feedback is set equal to the position feedback, thus the system operates in single-loop mode.

1

Local Encoder Counter

The axis gets velocity feedback from its local encoder input. This setting is used with RS-422 compatible digital quadrature encoders.

2

Encoder Multiplier

The axis uses the optional primary or auxiliary encoder multiplier for velocity feedback. You can use this option with an encoder that produces analog signals to increase the effective encoder resolution of the axis. When you use this feedback type, you must configure the parameters in the Feedback > Multiplier Parameter Category.

3

Analog Input

The axis uses one of its available analog inputs for velocity feedback. When you use this feedback type, the analog input is treated as a tachometer input. Specify the analog input number by using the VelocityFeedbackChannel parameter.

6

Resolver

The axis uses the optional resolver interface for velocity feedback. When you use this feedback type, you must configure the parameters in the Feedback > Resolver Parameter Category.

7

Parallel Laser Interface

The axis uses the optional parallel laser interface for velocity feedback. This setting is available only on the Npaq. The drive communicates with a ZYGO laser interferometer through the parallel laser interface to read the velocity feedback.

8

Serial Encoder

This is a custom feedback type that is not supported in all configurations. Do not use this setting unless you are instructed to do so by Aerotech personnel.
