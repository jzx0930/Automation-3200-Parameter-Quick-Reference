AutofocusInput Parameter

Default Value

0

Minimum Value

-1

Maximum Value

9

Units

None

Type

32-bit signed integer

Hardware: Autofocus does not apply when used with the GL4, the Ndrive FCL or FLS, the Nmark CLS, GCL, or SSaM, or the XL4s.
NOTE: To use autofocus, the drive doing the autofocus must have an analog input available.

Use the AutofocusInput parameter to select the analog input to use as the input to the autofocus loop. The input must correspond to a physical input on this axis. The Autofocus Loop Block Diagram shows the structure of the autofocus loop.

Output 0

When you select Output 0, the value of the autofocus input becomes equal to the value of analog output 0. You can use this setting to test the autofocus response to analog output voltages that you want to specify. To specify a voltage for analog output 0, use the $AO[0].<Axis>=<Voltage> command. Analog output 0 is channel 0 of the $AO Function.
