WARNING: Aerotech recommends that you do not use this parameter because it decreases system safety.

Hardware: This parameter does not apply when used with the Ndrive FCL or FLS, the Nmark CLS, GCL, or SSaM.

Note: This parameter applies only if you set the EnableBrakeControl Parameter to Yes and you set the BrakeDisableDelay Parameter to a nonzero value.

Use the FaultMaskDisableDelay parameter to specify the fault conditions that obey the BrakeDisableDelay Parameter.

If the controller disables an axis because of a fault condition, it ignores the BrakeDisableDelay parameter. The controller also disables the servo loop at the same time that it engages the brake. This behavior occurs by default when you do not set the corresponding bit of the FaultMaskDisableDelay parameter. In conditions where this behavior is not permitted for a given fault, you can set the corresponding bit of this parameter.

If you set the corresponding bit, the controller engages the brake and then waits for a specified interval before it disables the servo loop. You can specify the quantity of time for the delay by using the BrakeDisableDelay Parameter.

For more information about axis faults, refer to Fault and Error Overview.
