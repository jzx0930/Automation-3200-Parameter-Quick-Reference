EncoderMultiplicationFactor Parameter

Default Value

1.0

Minimum Value

1.0

Maximum Value

262144.0

Units

None

Type

32-bit IEEE floating point

Hardware: This parameter applies only if you have purchased the correct hardware option.

NOTE: This parameter applies only when you set one of these parameters to the specified value:

  • If you set the PositionFeedbackType Parameter to Encoder Multiplier.
  • If you use an EnDat absolute encoder that supplies incremental signals and set the PositionFeedbackType Parameter to EnDat Absolute Encoder.
  • If you set the VelocityFeedbackType Parameter to Encoder Multiplier.

Note: If you change this parameter, you must adjust other parameters. Refer to Adjusting Parameters Based on System Changes for more information.

Use the EncoderMultiplicationFactor parameter to specify the multiplication factor of the optional primary encoder multiplier module on the drive. If you set this parameter to a value greater than 1, then the effective resolution of the encoder increases by the factor that you specify. The multiplier module is designed to work only with encoders that generate an analog output.

For single-axis FireWire drives and all HyperWire drives, the primary encoder multiplier module applies only when you connect an encoder to the primary encoder input.

Note: When you have a FireWire drive that uses -MXH or -MXR feedback or a HyperWire drive that uses an encoder multiplier with the -MX2, -MX3, or -MX4 multiplier option for primary feedback, quadrature is generated only if the effective multiplication factor is greater than 4. The effective multiplication factor is equal to the value of the EncoderMultiplicationFactor parameter divided by the value of the EmulatedQuadratureDivider parameter.
