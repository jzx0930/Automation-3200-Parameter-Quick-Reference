ESTOPFaultInput Parameter

Default Value

-2

Minimum Value

-2

Maximum Value

23

Units

None

Type

32-bit signed integer

NOTE: The controller generates an axis fault only if you set the corresponding bit in the FaultMask parameter. When an axis fault occurs, you can configure what the axis does by changing the state of the corresponding bit in the FaultMaskDecel and FaultMaskDisable parameters.

Use the ESTOPFaultInput parameter to select the digital input that is used as the Emergency Stop (ESTOP) sense input. An ESTOP switch or circuit can be connected to a dedicated ESTOP input or to one of the standard digital inputs. The ESTOP input is always fail safe, which means that the ESTOP condition exists if the optocoupler is not driven, or when the digital input is not connected.

Hardware: Although the Nservo has more than 8 digital inputs, it is limited to using only the first 8 inputs for the ESTOP input.
