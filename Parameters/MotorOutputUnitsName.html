MotorOutputUnitsName Parameter

Default Value

N m

Minimum Value

N/A

Maximum Value

N/A

Units

None

Type

String

Optional Purchase Required: This parameter applies only when used with the A3200Motion Simulator software. For more information, refer to License Options.

Use the MotorOutputUnitsName parameter to set the name of the unit that the Motion Simulator will display for the motor output. The value of this parameter should correspond to the units of the scale factor you entered for the MotorOutputConstant parameter. For example, if the value you enter for the MotorOutputConstant parameter is in newtons per ampere, enter newtons (or N) for this parameter.

The application reads the value of this parameter at startup. If you change the value of this parameter, you must restart the Motion Simulator for the changes to take effect.

To view the motor output in Motion Simulator:

  1. Click Utilities.
  2. Select Preferences.
  3. On the left side of the Preferences dialog, click Axis Manager.
    4. Figure: Motion Simulator — Axis Manager: Display Type 4 Visible

  4. On the right side of the Preferences dialog, click Display Type 4 Visible.
  5. Use the drop down to select True.
  6. It might be necessary to expand the top part of the Motion Simulator screen to include Motor Output.
    7. Figure: Motion Simulator — Motor Output

Note: This parameter is a string variable that controls only the display in the Motion Simulator. Changing the value of this parameter will not convert the value of the MotorOutputConstant parameter.
