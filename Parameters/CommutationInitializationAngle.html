CommutationInitializationAngle Parameter

Default Value

0

Minimum Value

0

Maximum Value

359

Units

Degrees

Type

32-bit signed integer

Warning: Setting this parameter to an incorrect value can cause the motor to fail and might cause damage to the motor.

Hardware: This parameter applies only when used with HyperWire drives.

NOTE: This parameter applies only when you set the parameters that follow to the specified values:

  • When you set the Override Angle on First Home option of the CommutationInitializationSetup Parameter to Enabled.
  • When you set the HomeType Parameter to Home Past Limit to Marker, Home to Limit and Reverse to Marker, or Home to Marker Only.
  • When you set the MotorType Parameter to AC Brushless (Auto-MSET) or AC Brushless (Commutation Search).

Use the CommutationInitializationAngle parameter to specify the commutation initialization angle that you want to use with the Override Angle on First Home option of the CommutationInitializationSetup Parameter. You can specify two types of values:

A Value Based on the Mechanics and Configuration of Your System

Set the CommutationInitializationAngle parameter to the value of the motor commutation angle when the motor is at the marker position. If you know the relationship between the motor, the motor commutation angle, and the marker, Aerotech recommends that you use this information to determine the value for this parameter.

The value that you specify must be a positive integer between 0° - 359°. Adjust negative values when necessary. For example if you have a -30° commutation initialization angle, set the value of the CommutationInitializationAngle parameter to 330°.

After you set this value, save the parameter file. Then reset the controller for these changes to have an effect.

The drive uses the Automatic Motor Set (Auto-MSET) or Commutation Search method to initialize motor commutation. It selects a search method based on the value that you specified for the MotorType Parameter.

A Measured Value

If you do not know the relationship between the motor, the motor commutation angle, and the marker, use the procedure that follows to measure a value for the CommutationInitializationAngle parameter.

The value that you specify must be a positive integer between 0° - 359°. Adjust negative values when necessary. For example if you have a -30° commutation initialization angle, set the value of the CommutationInitializationAngle parameter to 330°.

The drive uses the Automatic Motor Set (Auto-MSET) or Commutation Search method to initialize motor commutation. It selects a search method based on the value that you specified for the MotorType Parameter.

TO MEASURE A VALUE FOR THE COMMUTATIONINITIALIZATIONANGLE PARAMETER
  1. Open Configuration Manager.
  2. Find the current value of the HomeOffset Parameter. Record this value. You will set the parameter back to this value at the end of the procedure.
  3. Change the value of the HomeOffset Parameter to 0.
  4. Set the Override Angle on First Home option of the CommutationInitializationSetup Parameter to Disabled.
  5. Save the parameter file. Then reset the controller for these changes to have an effect.
  6. Open Digital Scope.
  7. On the Configuration menu, select Configure Data Collection. The Configure Data Collection dialog comes into view. Do the steps that follow:
    1. In the Signals section, select the Motor Commutation Angle signal.
    2. Set Period (msec) and Points so that data collection measures one full home cycle.
    3. Click Apply. Then click OK to close the dialog.
  8. On the Action menu, select Step Forward. The Enter Forward Motion Command box comes into view. Do the steps that follow:
    1. Set the Step Forward motion command to HOME <Axis>.
    2. Click OK to close the box.
  9. On the Axis toolbar, click the Enable button to enable the axis.

    10. NOTE: The drive uses the Automatic Motor Set (Auto-MSET) or Commutation Search method to initialize motor commutation.
  10. On the Action menu, click Step Forward to run the Step Forward motion command.
  11. After the home cycle completes, find the value of the Motor Commutation Angle signal.
  12. In Configuration Manager, set the CommutationInitializationAngle parameter to the integer nearest to the value of the Motor Commutation Angle signal.
  13. Set the Override Angle on First Home option of the CommutationInitializationSetup parameter back to Enabled.
  14. Set the HomeOffset parameter back to the value that you recorded in Step 2.
  15. Save the parameter file. Then reset the controller for these changes to have an effect.
