AutofocusInitialRampTime Parameter

Default Value

0

Minimum Value

0

Maximum Value

1000

Units

Milliseconds

Type

32-bit signed integer

Hardware: Autofocus does not apply when used with the GL4, the Ndrive FCL or FLS, the Nmark CLS, GCL, or SSaM, or the XL4s.

Use the AutofocusInitialRampTime parameter to control the behavior of the axis when autofocus is enabled. If you specify zero as the value for this parameter, the controller performs no ramping. With this setting, sudden changes in position command might occur when autofocus is enabled if the value of the analog input specified by the AutofocusInput is significantly different than the AutofocusTarget. Due to this difference between the analog input and the AutofocusTarget, a large error is instantaneously introduced into the autofocus loop, and the output of the autofocus loop is controlled only by the magnitude of the error and the AutofocusGainKi and AutofocusGainKp parameters.

By setting this parameter to a nonzero value, sudden changes in the position command caused by enabling autofocus control can be reduced or eliminated. To avoid instantaneously introducing a large error into the autofocus loop, the axis will ramp its internal autofocus target value from the current value of the analog input to the value specified by the AutofocusTarget parameter. This ramping occurs over the time specified in this parameter and only when autofocus is enabled. The value of this parameter is not used if the AutofocusTarget value is changed while autofocus is enabled.
