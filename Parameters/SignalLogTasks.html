Note: You must reset the controller for these changes to have an effect. Setting this parameter in an AeroBasic program does not set the active value and does not have an effect on program execution.

Use the SignalLogTasks parameter to specify the mask of tasks that Signal Log monitors. When Signal Log is enabled, it collects a circular buffer of data from the tasks that you specify for this parameter.

Note: This parameter requires memory allocation. If the memory allocation fails, a memory allocation error occurs. If memory resources are low, you can decrease the value of this parameter to release memory for other A3200 features.
