AbortDecelRate Parameter

Default Value

500.0

Minimum Value

0.0

Maximum Value

N/A

Units

Primary Units /Second /Second

Type

64-bit IEEE floating point

Note: When you reference this parameter in an AeroBasic program, the parameter value is in the units specified by the unit mode of the task. If the task is in primary units mode, this parameter is in primary units. If the task is in secondary units mode, this parameter is in secondary units.

Use the AbortDecelRate parameter to specify the deceleration rate used by an axis during an abort. There are several circumstances under which an axis will abort using this rate, including but not limited to:

  • when an axis fault occurs that is selected in both the FaultMask and FaultMaskDecel parameters
  • when you issue an ABORT command during asynchronous motion (FREERUN, MOVEINC, or MOVEABS)
  • after a loss of communications

Note: Aborting an axis that is part of a coordinated move will cause the motion to be aborted along the coordinated path. The motion will not use the AbortDecelRate parameter. (Refer to ABORT Command for more information.)

Note: If an End of Travel limit fault or a Software limit fault triggers an abort, the LimitDecelDistance parameter might determine the deceleration rate instead of the AbortDecelRate. This can cause faster deceleration than specified by AbortDecelRate. Refer to LimitDecelDistance parameter topic for more information.
