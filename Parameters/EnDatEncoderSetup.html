Hardware: This parameter applies only when used with the Ndrive CP, HLe, or HPe, the XC2, the XC4, the XC4e, or the XR3.

Use the EnDatEncoderSetup parameter to configure options related to the EnDat encoder interface of the drive. The sections that follow give the possible options for this parameter.

Commutation Initialization

This option specifies if absolute position information from the encoder is used to initialize the motor commutation.

When you select Do Not Use Absolute Position, the motor commutation is initialized through the standard method using either Hall-effect switches or Auto-MSET mode depending on the setting of the MotorType parameter.

Use Encoder Absolute Position is the default. The motor commutation is initialized using the absolute position information that is read from the EnDat encoder and assumes that the zero position of the encoder corresponds to the motor zero. To compensate for misalignment between the encoder and the motor, use the CommutationOffset parameter.

Position Initialization

This option specifies if absolute position information from the encoder is used to initialize the position feedback.

When you select Do Not Use Absolute Position, the information from the encoder is not used to initialize the position feedback, which by default is set to zero at startup.

Use Encoder Absolute Position is the default. The position feedback is initialized using the absolute position information that is read from the EnDat encoder.

Encoder Configuration

Hardware: This setting applies only when used with HyperWire drives.

This option specifies if the drive will use the controller parameters that follow or the manufacturer settings to configure the EnDat encoder interface.

  • EnDatEncoderIncrementalResolution
  • EnDatEncoderResolution
  • EnDatEncoderTurns
  • Encoder Type option of the EnDatEncoderSetup parameter
  • Feedback Mode option of the EnDatEncoderSetup parameter

Use Controller Parameters is the default. When you select this option, the drive uses the values specified in the controller parameter file to configure the EnDat encoder interface.

When you select Use Manufacturer Settings, the drive uses the manufacturer settings that are stored in the encoder to configure the EnDat encoder interface.

Encoder Type

Note: This parameter option does not apply if the Encoder Configuration option for the EnDatEncoderSetup parameter is set to Use Manufacturer Settings.

This option specifies the type of EnDat encoder that you are using. Refer to Configuring Parameters for an EnDat Absolute Encoder for configuration instructions that are specific to each type of encoder.

Rotary is the default. The controller is configured to communicate with a rotary EnDat encoder.

When you select Linear, the controller is configured to communicate with a linear EnDat encoder.

Feedback Mode

Note: This parameter option does not apply if the Encoder Configuration option for the EnDatEncoderSetup parameter is set to Use Manufacturer Settings.

This option specifies if the incremental feedback signals are used for calculating the position feedback.

Use Incremental Signals is the default. The absolute position is read one time, at startup, from the EnDat encoder to initialize the position feedback. After the absolute position is read, the incremental signals are used to calculate the position. For correct operation with incremental signals, the EnDat encoder must supply incremental feedback signals and the drive must have an applicable encoder multiplier option.

When you select Use Absolute Position Only, the absolute position is read from the EnDat encoder at each servo cycle and no incremental feedback signals are necessary. Use this mode only with encoders that do not supply incremental feedback signals.

Initial Position

This option specifies how the drive initializes the position feedback of an axis when you use an EnDat absolute encoder for feedback.

Always Positive is the default. When the drive initializes the position of the axis, it interprets the value from the encoder as an unsigned number. Thus, the initial position is always positive. For example, if you set the EnDatEncoderResolution Parameter to 16, the drive initializes the position feedback to a value between 0 and 65,535.

When you select Positive or Negative, the drive interprets the value from the encoder as a signed number when it initializes the position of the axis. Thus, the initial position can be positive or negative. For example, if you set the EnDatEncoderResolution Parameter to 16, the drive initializes the position feedback to a value between -32,768 and 32,767.
