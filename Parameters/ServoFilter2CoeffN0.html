ServoFilter2CoeffN0 Parameter

Default Value

1.0

Minimum Value

-4.0

Maximum Value

4.0

Units

None

Type

32-bit IEEE floating point

Warning: Disable the axis before you make changes to this parameter.

Most A3200 drives have eight digital filters in the servo loop. The Ndrive FLS and Nmark CLS have four digital filters in the servo loop. The GL4, the XC2, the XC4, the XC4e, the XR3, and the Nmark GCL have sixteen digital filters in the servo loop. Each filter contains the parameters that follow.

  • ServoFilter2CoeffN0
  • ServoFilter2CoeffN1
  • ServoFilter2CoeffN2
  • ServoFilter2CoeffD1
  • ServoFilter2CoeffD2

The coefficients for the first filter default to a low-pass filter with a 1 kHz bandwidth. You can use the Digital Filter Calculator in Configuration Manager or Digital Scope to calculate the coefficients for some standard filter types (low-pass, notch, lead-lag, and resonant). You can calculate coefficients for other filters externally, but the coefficient values must be greater than -4.0 and less than 4.0.

Note: You must enable each filter in the ServoFilterSetup parameter for the filter to have an effect.
