Hardware: The following information applies only if you are using the Ndrive HLe or HPe, the Npaq, or the Nservo with the resolver option. This information does not apply to the Npaq MR.

Use the ResolverSetup parameter to configure options related to the optional resolver interface of the drive. The sections that follow give the possible options for this parameter.

Resolver Type

This option specifies the mode of operation of the resolver feedback.

Single Speed is the default. A single resolver channel is used to derive feedback.

When you select Dual Speed, two resolver channels are used to derive feedback. For this setting, you must also configure the ResolverCoarseChannel, ResolverFeedbackRatio, and ResolverFeedbackOffset parameters.

Resolution Select

This option specifies the resolution of the resolver feedback device.

Dynamic Resolution is the default. The controller adjusts the resolution of the resolver feedback device automatically based on the speed of the motor.

Table:     Dynamic Resolution Setting

Applicable Drives Resolution

Ndrive HLe and HPe
  • The resolution of the resolver feedback device starts at 16-bit.
  • When the speed of the motor exceeds 12.2 revolutions per second, the controller decreases the resolution to 14-bit.
  • When the speed of the motor exceeds 48.8 revolutions per second, the controller decreases the resolution to 12-bit.

Npaq and Nservo
  • The resolution of the resolver feedback device starts at 16-bit.
  • When you command the axis to move, the controller decreases the resolution to 14-bit.

When you select 16-bit Fixed Resolution, 14-bit Fixed Resolution, or 12-bit Fixed Resolution, the controller configures the resolver feedback device to the specified resolution. The following table shows the feedback counts and the maximum speed of each setting.

Reference Select

This option specifies which of the two available reference signals is used to process the resolver feedback.

Sine (Resolver) is the default. The reference signal output to the resolver is used to process the feedback. This setting is typically used for interfacing with a resolver device.

When you select Cosine (Inductosyn), a 90 degree phase shifted version of the reference signal is used to process the feedback. This setting is typically used for interfacing with an Inductosyn device due to the inherent phase shift introduced by the feedback device.

Encoder Emulation

This option enables the generation of emulated quadrature signals based on the resolver feedback information.

Disabled is the default.

When you select Enabled, the resolver channel generates emulated quadrature signals at the resolution that you specified for the Resolution Select option.

Refer to the Resolution Select option for the Dynamic Resolution setting. The resolver channel generates emulated quadrature signals at the lowest resolution for this setting.

Commutation Initialization

This option specifies the method of initializing the motor commutation when using resolver feedback.

Initialize Using Resolver is the default. The motor commutation is initialized using the absolute angle of the resolver and assumes that the zero angle of the resolver corresponds to the motor zero. To compensate for misalignment between the resolver and the motor, use the CommutationOffset parameter.

When you select Initialize Using Hall Signals, the motor commutation is initialized using Hall-effect switches on the motor.

Initial Position

This option specifies how the drive initializes the position feedback of an axis when you use resolver feedback.

Zero to one revolution is the default. The drive initializes the position of the axis to a value that is between zero and one full revolution of the resolver. For example, for a 16-bit resolver the drive initializes the position feedback to a value between 0 and 65,535.

When you select +/- Half of one revolution, the drive initializes the position of the axis to a value that is between negative one-half and positive one-half of one revolution of the resolver. For example, for a 16-bit resolver the drive initializes the position feedback to a value between -32,768 and 32,767.
