Analog1Filter1CoeffD1 Parameter

Default Value

0.0

Minimum Value

-4.0

Maximum Value

4.0

Units

None

Type

32-bit IEEE floating point

All A3200 drives support two second-order digital filters on the analog inputs. Each filter contains the parameters that follow.

  • Analog1Filter1CoeffN0
  • Analog1Filter1CoeffN1
  • Analog1Filter1CoeffN2
  • Analog1Filter1CoeffD1
  • Analog1Filter1CoeffD2

On the first filter of each input, the coefficients default to a low-pass filter with a 250 Hz bandwidth. On the second filter of each input, the coefficients default to no filtering. You can change any of the coefficients, but you need to calculate the values externally. Analog inputs are sampled at a rate of 24 kHz on the Ndrive QL and QLe. On all other drives, analog inputs are sampled at a rate of 8 kHz.

Hardware: The Ndrive HLe and HPe, and the XC4 and XC4e with the optional I/O board have two more analog inputs (Input 2 and Input 3). These inputs are programmed with a 250 Hz low-pass filter that you cannot change. You can enable or disable these inputs by using the AnalogFilterSetup parameter in Configuration Manager.
