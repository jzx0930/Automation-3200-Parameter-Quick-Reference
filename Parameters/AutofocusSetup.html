Use the AutofocusSetup parameter to configure options related to autofocus behavior. The sections that follow give the possible options for this parameter.

Loop Polarity

Normal is the default. The autofocus loop polarity is normal (positive error causes positive motion).

When you select Inverted, the autofocus loop polarity is reversed (negative error causes positive motion).

Autofocus Limits

Disabled is the default. There are no limits. The autofocus loop can command the axis to move to any position.

When you select Enabled (Absolute), the autofocus position limits are relative to the axis home position. Use the AutofocusLimitHigh Parameter and AutofocusLimitLow Parameter to specify the autofocus position limits.

For Example - Enabled (Absolute)

You select Enabled (Absolute), which sets the AutofocusSetup Parameter to 0x2 on the X axis. Then you set the values of the AutofocusLimitLow Parameter to -127 mm and the AutofocusLimitHigh Parameter to 20 mm. The home position of the axis is at 0 mm. When the axis position is at -50 mm, you issue the AUTOFOCUS ON Command to enable autofocus.

Because the autofocus position limits are referenced to the home position and not the current position, the controller clamps the position command at -127 mm or 20 mm.

When you select Enabled (Relative), the autofocus position limits are relative to the position where autofocus was enabled. Autofocus is enabled when the AUTOFOCUS ON Command is issued.

For Example - Enabled (Relative)

You select Enabled (Relative), which sets the AutofocusSetup Parameter to 0x4 on the X axis. Then you set the values of the AutofocusLimitLow Parameter to -127 mm and the AutofocusLimitHigh Parameter to 20 mm. The home position of the axis is at 0 mm. When the axis position is at -50 mm, you issue the AUTOFOCUS ON Command to enable autofocus.

Because the autofocus position limits are referenced to the current position and not the home position, the controller clamps the position command at -177 mm or -30 mm.

Focus Mode

Continuous Focus is the default. The autofocus routine runs continuously until you issue the AUTOFOCUS OFF command to manually disable this routine.

When you select Single Focus, the autofocus routine automatically turns off when the analog input that is specified by the AutofocusInput parameter is within the specified deadband of the AutofocusDeadband parameter.

Hold Input Polarity

Active Low is the default. The autofocus loop is held when the digital input specified by the AutofocusHoldInput parameter is low. For a differential input this corresponds to a low input level, and for an opto-isolated input this corresponds to the optocoupler not being driven.

When you select Active High, the autofocus loop is held when the digital input specified by the AutofocusHoldInput parameter is high. For a differential input this corresponds to a high input level, and for an opto-isolated input this corresponds to the optocoupler being driven.

Motion Mode

Normal is the default. The controller uses the output of the autofocus loop as the position command for the axis. In this mode, if you enable autofocus control then you cannot issue other Motion commands to the axis while the autofocus is active.

When you select Additive, the controller adds the output of the autofocus loop to the position command for the axis.

InPosition Mode

Use Autofocus Deadband is the default. The drive uses the Autofocus Error to set the In Position bit of the Drive Status. When the Autofocus Error is less than the value that you specify for the AutofocusDeadband Parameter, the drive sets the In Position bit to TRUE.

NOTE: You must turn on autofocus for the Use Autofocus Deadband setting to have an effect. If autofocus is off, the results look as if you have selected the Use Position Error setting.

When you select Use Position Error, the drive uses the absolute value of the Position Error to set the In Position bit of the Drive Status. When the absolute value of the Position Error is less than the value that you specify for the InPositionDistance Parameter for a minimum of the quantity of time that you specify for the InPositionTime Parameter, the drive sets the In Position bit to TRUE.

When you select Use Autofocus Error, the drive uses the Autofocus Error to set the In Position bit of the Drive Status. When the Autofocus Error is less than the value that you specify for the InPositionDistance Parameter, the drive sets the In Position bit to TRUE. You must specify the value of the InPositionDistance parameter in volts when you use this setting.

NOTE: If you use the Use Autofocus Error setting while the autofocus mode is off or while you issue the AUTOFOCUS OFF Command, the In Position bit stays set to TRUE.
InPosition2 Mode

Use Position Error is the default. The drive uses the absolute value of the Position Error to set the In Position 2 bit of the Drive Status. When the absolute value of the Position Error is less than the value that you specify for the InPosition2Distance Parameter for a minimum of the quantity of time that you specify for the InPosition2Time Parameter, the drive sets the In Position 2 bit to TRUE.

When you select Use Autofocus Error, the drive uses the Autofocus Error to set the In Position 2 bit of the Drive Status. When the Autofocus Error is less than the value that you specify for the InPosition2Distance Parameter, the drive sets the In Position 2 bit to TRUE. You must specify the value of the InPosition2Distance parameter in volts when you use this setting.

NOTE: If you use the Use Autofocus Error setting while the autofocus mode is off or while you issue the AUTOFOCUS OFF Command, the In Position 2 bit stays set to TRUE.
Stability0 Mode

Use Position Error is the default. The controller uses the absolute value of the Position Error to set the Stability 0 bit of the Axis Status. When the absolute value of the Position Error is less than the value that you specify for the Stability0Threshold Parameter for a minimum of the quantity of time that you specify for the Stability0Time Parameter, the controller sets the Stability 0 bit to TRUE.

When you select Use Autofocus Error, the controller uses the Autofocus Error to set the Stability 0 bit of the Axis Status. When the Autofocus Error is less than the value that you specify for the Stability0Threshold Parameter, the controller sets the Stability 0 bit to TRUE. You must specify the value of the Stability0Threshold parameter in volts when you use this setting.

NOTE: If you use the Use Autofocus Error setting while the autofocus mode is off or while you issue the AUTOFOCUS OFF Command, the controller sets the Stability 0 bit to TRUE after the quantity of time that you specify for the Stability0Time parameter occurs.
Stability1 Mode

Use Position Error is the default. The controller uses the absolute value of the Position Error to set the Stability 1 bit of the Axis Status. When the absolute value of the Position Error is less than the value that you specify for the Stability1Threshold Parameter for a minimum of the quantity of time that you specify for the Stability1Time Parameter, the controller sets the Stability 1 bit to TRUE.

When you select Use Autofocus Error, the controller uses the Autofocus Error to set the Stability 1 bit of the Axis Status. When the Autofocus Error is less than the value that you specify for the Stability1Threshold Parameter, the controller sets the Stability 1 bit to TRUE. You must specify the value of the Stability1Threshold parameter in volts when you use this setting.

NOTE: If you use the Use Autofocus Error setting while the autofocus mode is off or while you issue the AUTOFOCUS OFF Command, the controller sets the Stability 1 bit to TRUE after the quantity of time that you specify for the Stability1Time parameter occurs.
