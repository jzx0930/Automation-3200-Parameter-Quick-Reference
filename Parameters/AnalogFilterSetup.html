Use the AnalogFilterSetup parameter to configure options related to the second-order digital filters on the analog inputs. The sections that follow give the possible options for this parameter.

Input 0 Filters

Filter 0 Enabled is the default.

When you select Filter 1 Enabled, filter 1 is enabled on analog input 0.

Input 1 Filters

Filter 0 Enabled is the default.

When you select Filter 1 Enabled, filter 1 is enabled on analog input 1.

Input 2 Filter

Hardware: This setting applies only to the Ndrive HLe and HPe, the XC4, and the XC4e.

The filter is enabled on analog input 2 by default. The value of this filter is programmed as a 250 Hz low-pass filter. You cannot change the value of this filter. To disable this filter, clear the Fixed 250Hz Low-Pass box.

Input 3 Filter

Hardware: This setting applies only to the Ndrive HLe and HPe, the XC4, and the XC4e.

The filter is enabled on analog input 3 by default. The value of this filter is programmed as a 250 Hz low-pass filter. You cannot change the value of this filter. To disable this filter, clear the Fixed 250Hz Low-Pass box.
