ModeStackSize Parameter

Default Value

10

Minimum Value

0

Maximum Value

100

Units

None

Type

32-bit signed integer

Use the ModeStackSize parameter to specify the maximum number of mode stack elements available for this task. Task modes can be pushed onto the stack using the PUSHMODES command and removed from the stack using the POPMODES command.

Note: This parameter requires memory allocation. If the memory allocation fails, a memory allocation error occurs. If memory resources are low, you can decrease the value of this parameter to release memory for other A3200 features.

WARNING: Do not change this parameter from within an AeroBasic program. Change this parameter only in Configuration Manager.

Warning: Do not change this parameter while the axes are moving. Discontinuities can occur in the motion if the parameter change requires large quantities of memory allocation.
