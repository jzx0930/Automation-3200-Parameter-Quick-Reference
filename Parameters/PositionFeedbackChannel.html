Use the PositionFeedbackChannel parameter to select the channel of the position feedback device that you want to use. If your feedback device does not support multiple channels, the controller ignores the value of this parameter.

For single-axis drives, the only channels that you can select for this parameter are Default, Channel 0, or Channel 1. They do the operations that follow:

  • Default - Configures the axis to use the primary encoder input.
  • Channel 0 - Configures the axis to use the primary encoder input.
  • Channel 1 - Configures the axis to use the auxiliary encoder input.
  • The other channels do not have an effect.

For the XR3 multi-axis drive, each axis operates like a single-axis drive. When you use an XR3, the only Position Feedback Channels that you can select for this parameter are Default, Channel 0, or Channel 1.

For other multi-axis drives, such as the Npaq and Nservo, you can use this parameter to let any axis of the drive use feedback from any feedback channel. If you select Default, the axis gets feedback from the channel that corresponds to its axis index within the unit. For example, Axis 0 uses Channel 0, Axis 1 uses Channel 1, and so on.

The Npaq has two auxiliary encoder inputs on connector J8 that you can use by selecting Channel 6 or Channel 7. This information does not apply to the Npaq MR.

The Nservo has a single auxiliary encoder input on connector J107 that you can use by selecting Channel 4.

To configure the Feedback Averaging option of the FeedbackSetup Parameter for multi-axis drives, refer to Configuring Parameters for Position Feedback Averaging on Multi-Axis Drives.
