DefaultCoordinatedSpeed Parameter

Default Value

50.0

Minimum Value

0.0

Maximum Value

N/A

Units

Primary Units /Second

Type

64-bit IEEE floating point

Note: This parameter applies only to dominant axes. For information about axis types, refer to Axis Type Overview.

NOTE: To set the coordinated speed from within an AeroBasic program, use the F command.

Use the DefaultCoordinatedSpeed parameter to define the default feedrate of dominant axes along the path of a coordinated motion. This parameter initializes the value that is specified by the F command of the current task.
