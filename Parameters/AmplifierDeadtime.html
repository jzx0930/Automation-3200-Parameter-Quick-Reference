AmplifierDeadtime Parameter

Default Value

3.0

Minimum Value

1.0

Maximum Value

5.0

Units

Microseconds

Type

32-bit IEEE floating point

Hardware: This parameter does not apply to drives that have linear amplifiers, which include the GL4, Ndrive FCL, HLe, ML, QL, and QLe, the Nmark GCL, the Npaq with DL4010, the XL4s, and the XR3 with XSL3.

Use the AmplifierDeadtime parameter to set the Pulse Width Modulation (PWM) switching dead time of the amplifier power stage. Aerotech does not recommend changing the value of this parameter in most situations.

By default, the minimum value for this parameter is set to 3 microseconds and is programmed into non-volatile memory on the drive. This limit is intended to protect the amplifier from incorrect parameter settings and can be configured for a different value at the factory. If you need to set this parameter to a value less than 3 microseconds, you must contact Aerotech Global Technical Support.
