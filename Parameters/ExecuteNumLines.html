ExecuteNumLines Parameter

Default Value

1

Minimum Value

N/A

Maximum Value

N/A

Units

None

Type

32-bit signed integer

Use the ExecuteNumLines parameter to control program execution priority of tasks, prioritize task program execution, and speed up execution of a high priority task. The ExecuteNumLines parameter determines the number of program lines that the controller attempts to execute every millisecond.

If the program contains a command that requires more an one millisecond to execute (such as RAPID, LINEAR, CW, CCW, DWELL, ENABLE, DISABLE, SETPOSCMD, WAIT, or a Callback command) the task stops trying to execute lines and releases execution to the next task, regardless of the value of this parameter.

Refer to theTask Scheduling Overview topic for more information.

Warning: Be careful when you change the value of this parameter. Setting this parameter too high can cause the A3200 to starve all other Windows processes, which might cause the PC to stop responding.
