Command List

The A3200 AeroBasic programming commands, including mathematical functions, motion and I/O commands, and special compiler commands, are listed in the following table.

Some AeroBasic commands do actions that were accomplished in previous versions of the A3200 using G-Code or M-Code commands. You can use G-Codes and M-Codes if you have purchased the CNC controller option. Refer to G-Code and M-Code Equivalents for a complete list of G-Codes and M-Codes and their AeroBasic equivalents.
