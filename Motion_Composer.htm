Motion Composer Overview

A3200 Motion Composer is an environment for developing and debugging AeroBasic programs for A3200 controllers. It also functions as a basic interface and can be used to do the following tasks.

  • Command basic motion
  • Inspect diagnostic information
  • do basic configuration (For advanced configuration options, use Configuration Manager.)

Figure: Motion Composer

You can use the Alt key to access menus in the menu toolbar of the application. Press Alt. Then press the letter that is underlined in the name of the menu that you want. In the menu that comes into view, press the letter that is underlined in the name of the option that you want.
