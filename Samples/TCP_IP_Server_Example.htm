TCP/IP Server Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation. 

'                       TCPServer.pgm
'
' An example chat server.
'
' This program listens on port 8000 for connections. Any content sent 
' from a client is sent to all other clients. Up to 5 clients are 
' supported. Content sent has to be new line terminated.
'
' This program implements rudimentary error handling: if an error
' occurs, the program restarts and closes the connection to any
' clients.
'
' See TCPClient.pgm for an example client.

#define MAX_CLIENTS 5
#define SERVER_PORT 8000
#define BUFFER $strtask[0]

DVAR $hListen
DVAR $hClient[MAX_CLIENTS]
DVAR $clientCount
DVAR $pendingClients
DVAR $pendingOctets
DVAR $currentReadClient
DVAR $currentWriteClient

ONGOSUB (TASKSTATUS(DATAITEM_TaskErrorCode) != 0) FARCALL "" "HandleTaskError" PRIORITY_FAULT

AFTER_ERROR:
' Close any open sockets.
NETWORK CLOSE
$clientCount = 0

' Open the port for listening.
$hListen = NETWORK LISTEN SERVER_PORT

' Forever loop.
WHILE 1
	' Check to see if there is a new client that wants to connect.
	$pendingClients = NETWORK PENDING CLIENT $hListen
	IF $pendingClients > 0 && $clientCount < MAX_CLIENTS THEN
		$hClient[$clientCount] = NETWORK ACCEPT $hListen
		' Set to autoflush and not have a timeout.
		NETWORK AUTOFLUSH $hClient[$clientCount] ON
		NETWORK TIMEOUT $hClient[$clientCount], -1
		
		$clientCount = $clientCount + 1
	ENDIF
	
	' Check all the clients for data.
	FOR $currentReadClient = 0 TO $clientCount-1
		$pendingOctets = NETWORK PENDING OCTETS $hClient[$currentReadClient]
		IF $pendingOctets > 0 THEN
			' Read the data until a new line.
			BUFFER = NETWORK READ $hClient[$currentReadClient] STRING "\x0D\x0A"
			IF STRLEN(BUFFER) > 0 THEN
				' Send the buffer to all clients.
				FOR $currentWriteClient = 0 TO $clientCount-1
					IF $currentWriteClient != $currentReadClient THEN
						' Send the string.
						NETWORK WRITE $hClient[$currentWriteClient], STRING BUFFER, STRING "\x0D\x0A"
					ENDIF
				NEXT $currentWriteClient
			ENDIF
		ENDIF
	NEXT $currentReadClient
WEND

' Lines below are not called.  They are used to avoid a compiler error.
GOTO AFTER_ERROR
GOTO HandleTaskError

DFS HandleTaskError
	ACKNOWLEDGEALL
	FARGOTO "" "AFTER_ERROR"
ENDDFS
