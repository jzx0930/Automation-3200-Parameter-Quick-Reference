Display User Messages Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation. 

'				Display.pgm
'
'
' This program shows how to use callbacks to 
' print messages to the Motion Composer and 
' Operator Interface.  Not all commands are 
' supported exactly the same in each application.
' Refer to the individual commands documentation
' for specific details.
' 

DVAR $Count
DVAR $Response       

' Clear the Msg Tab
MSGCLEAR -1                

MSGDISPLAY 0, "DEMO STARTED"   

' Ask for user input - OK Button only
' note: if user presses OK, then $info0 is returned as 1
$global[5] = 78.9887
$global[5] = MSGINPUT (DF_MSGBOX_OKONLY), "Guess a number;Enter any number;", $global[5]     
' Echo back to the Msg Tab
MSGDISPLAY 0, "You entered " $global[5] " Key pressed(" $info0 ")"

' Ask the User for a string
$strglob[0] = MSGINPUT (DF_INPUT_STRING), "Program;Enter a program;", "Default string value"
' Echo back to the Msg Tab
MSGDISPLAY 0, "You entered " $strglob[0] " Key pressed(" $info0 ")"

' Ask for user input - OK/Cancel Button (Integer values only)
' note: if user presses OK, then $info0 is returned as 1
' note: if user presses cancel, then $info0 is returned as 0
$global[5] = 78.9887
$global[5] = MSGINPUT (DF_INPUT_INTEGER + DF_MSGBOX_OKCANCEL), "Guess a number;Enter any number;", $global[5]
' Echo back to the Msg Tab
MSGDISPLAY 0, "You entered " $global[5] " Key pressed(" $info0 ")"

' Blinks "Fault" in all the availibale panels
$Count = 1
REPEAT 6
   MSGLAMP $Count, RED, "Fault"
   DWELL 0.5
   MSGLAMP $Count, GRAY, ""    
   DWELL 0.5
   $Count=$Count+1
ENDRPT

'
' Hide Msg Tab
MSGHIDE

' Display Msg Tab, Clear its contents, Make sure it is visible
MSGCLEAR -1
MSGSHOW

' Display User Type 2 messages in list 
MSGDISPLAY 2, "I am a user type 2 message"

' Display User Type 1 messages in list 
MSGDISPLAY 1, "User type 1 message"
MSGDISPLAY 1, "Another user type 1 message"

' Display List , Clear all user type 1 messages
MSGCLEAR 1, "I just cleared type 1 messages"

' Ask the user a question
$Response = MSGBOX (DF_MSGBOX_YESNO + DF_ICON_QUESTION), "Are you Sleepy?"
MSGDISPLAY 0, "Resp = ", $Response
IF $Response == YES_BUTTON THEN
   MSGDISPLAY 0, "Me Too!!"
ELSE
   MSGDISPLAY 0, "You must be Sneezy then"
ENDIF

' Display menu of choices, allow user to pick one
MSGMENU (DF_MENU_REMOVE), -1 ""
MSGMENU (DF_MENU_ADD),   1  "Cut"
MSGMENU (DF_MENU_ADD),  2,  "Scribe"
MSGMENU (DF_MENU_ADD), 3, "Drill (Default)"
MSGMENU (DF_MENU_ADD), 4, "Weld"
$global[5] = MSGMENU (DF_MENU_SHOW), (DF_MSGBOX_OKCANCEL), "Select your Process;Which Option ?;", 3
' Echo back to the Msg Tab
MSGDISPLAY 0, "You selected option: " $global[5] " Key pressed(" $info0 ")"
      
' Get user to pick a file
$strglob[0] = MSGFILESELECT DF_FILESELECT_OPEN,  "Browse for a program file;C:\;Display.pgm;Program File(*.pgm)|*.pgm|All Files(*.*)|*.*"
' Echo back to the Msg Tab
MSGDISPLAY 0, "You selected file: " $strglob[0]

MSGDISPLAY 0, "DEMO DONE"    

END PROGRAM
