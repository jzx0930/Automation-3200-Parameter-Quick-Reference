Program Flow Control Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation. 

'				ProgramFlow.pgm
'
'
' This example shows you the structure of a basic program.
'

' You can use #define and #include commands.
#define MY_CONSTANT    6

' Variable Declarations
' All variables are declared before executable program commands.
DVAR $Counter1, $Counter2
DVAR $ArrayOfDoubles[MY_CONSTANT * MY_CONSTANT]

' A FOR-NEXT Loop Example
' Loop nesting is limited by available memory.
FOR $Counter2 = 0 TO MY_CONSTANT STEP 1
    FOR $Counter1 = 0 TO MY_CONSTANT STEP 1
        $ArrayOfDoubles[$Counter2 + $Counter1] = $Counter1 * $Counter2
    NEXT $Counter1
NEXT $Counter2

' You can use IF-THEN-ELSEIF commands.
IF $Counter1 == 4 THEN
    $Counter1 = 8
ELSEIF $Counter1 == 3 THEN
    $Counter1 = 9
ELSE 
    $Counter1 = 5
END IF

' A WHILE/ENDWHILE Example
WHILE $Counter1 < 40
    $Counter1 = $Counter1 + 1
ENDWHILE

' Call a user-defined subroutine.
CALL SqrValue 

' You can use GOTO commands.
' This GOTO command creates an infinite loop.
'TAG:
'    $Counter1 = $Counter1 - 1
'GOTO TAG

' Use an END PROGRAM command before you define subroutines or you will
' get a compiler warning. If you do not use subroutines in your program,
' the END PROGRAM command is not necessary. 
END PROGRAM

' Define a subroutine.
DFS SqrValue                      
	' Raise $Counter1 to the power of 2.
	$Counter1 = $Counter1 ** 2
ENDDFS
