Array Based, Pulse Mode Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation.

Program Summary

Hardware: This example program does not apply to the Nmark SSaM and the Nstep.

This program demonstrates the use of the PSO for single-axis firing based on variable distances. An array of firing distances is created and downloaded to the PSO hardware. The primary feedback for the X axis is tracked and a PSO firing event is generated each time the feedback increments by the distance specified in the array. At each firing event a set of two pulses is generated, with each pulse being four milliseconds long and having a 50 percent duty-cycle.

'                     ArrayDistancePulse.pgm
'
' This sample demonstrates the use of the PSO for single-axis firing based on variable
' distances.  An array containing the distance values is downloaded to the hardware and 
' used in conjunction with single-axis distance tracking.  At each firing event, a user-
' configured output pulse stream is generated and a new firing distance is loaded from
' the array.  After all of the array values are used, the PSO stops firing.

' NOTE: All PSO samples are written assuming 1000 counts per unit.
' NOTE: PSO pulse outputs are intentionally long so that the PSO output state can
'       be easily viewed using the "PSO Status" item in the Digital Scope.


' Define the number of distances we want to put in our array.  Using a #define statement
' makes it easier to change the number of distances later without having to remember
' all of the places in the program that would need to be changed.
#define NUM_DISTANCES 10

DVAR $Index
DVAR $ArrayDistances[NUM_DISTANCES]

' Fill the local array with the incremental firing distances.
' In this case, we are increasing the firing distance by a fixed quantity for each array
' element.  The distance increments are 1000, 1400, 1800, 2200, ... so the PSO should
' fire at positions 1000, 2400, 4200, 6400, etc.
FOR $Index = 0 TO (NUM_DISTANCES - 1)
   $ArrayDistances[$Index] = 1000 + ($Index * 400)
NEXT $Index

' Write the distance values from the local variables into the drive array.
ARRAY X WRITE $ArrayDistances[0] 0 NUM_DISTANCES

' Enable and home the axis.
ENABLE X
HOME X

' Reset the PSO.  This resets the internal state of the PSO hardware, including the distance
' tracking counters and the window counters.
PSOCONTROL X RESET

' Uncomment the following line to configure the PSO hardware to send the PSO signal out the
' auxiliary marker.  If using the dedicated PSO output, this command should not be used.
' Refer to your drive hardware manual for more information.
'PSOOUTPUT X CONTROL 0 1

' Configure the PSO for single axis tracking from the primary encoder input.
PSOTRACK X INPUT 0

' Configure the PSO to generate a firing event based on distances in the array.
' In this case the distances will be used starting at location 0 in the drive array.
PSODISTANCE X ARRAY 0 NUM_DISTANCES

' Configure the pulse generator by setting the total and off times and the number of cycles.
PSOPULSE X TIME 4000, 2000 CYCLES 2

' Specify the output of the PSO to come from the pulse generator.
PSOOUTPUT X PULSE

' Arm the PSO to start the distance tracking.
PSOCONTROL X ARM

' Move the axis so as to cause the PSO to fire.
LINEAR X 30 F 10

' Disable the PSO pulse generator.
PSOCONTROL X OFF

END PROGRAM
