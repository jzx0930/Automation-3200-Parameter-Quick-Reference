End-Point Modification Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation. 

'				EndPointModification.pgm
'
'
' This program uses end-point modification to illustrate
'  high-speed position capture via the DATAACQ command.
'
' The axis is commanded to an absolute position, which is
'  the distance that is traveled if the registration input
'  is not found.
'
' If the registration input is found, end-point
'  modification is used to move to an absolute position
'  past the registration point.
'
' Note: In this example, the same feedrate is used for
'  both moves. This is not a requirement. The registration
'  move can be made at a faster speed to make sure that the
'  motion is completed within a time limit.
' -------------------------------------------------

' Declare variables used in the program
DVAR $MoveDistance, $DistancePastRegistration, $MotionVelocity
DVAR $Done, $CapturedPosition, $NumberOfValues

ENABLE X
HOME X

' Initialize the value of the variables
' Move distance = 200 mm
$MoveDistance = 200			
' Move 50 mm past registration
$DistancePastRegistration = 50      
' 20 mm per second
$MotionVelocity = 20		
' Not done
$Done = 0                       
' Rate-based accel/decel
RAMP MODE RATE    
' Accel/decel = 1,000 mm/sec/sec
RAMP RATE 1000                  

' Capture X axis position from a user interrupt
' Input 12  (User Interrupt 1) Rising or 
' opto-isolator energizing is the trigger
DATAACQ X 1 TRIGGER 0   
 
' Capture X axis Primary position Feedback
DATAACQ X 1 INPUT 0  

' Collect 1 data point into array element 0
DATAACQ X 1 ARRAY 0 1

' Start X axis moving toward/past the registration.
MOVEABS X $MoveDistance $MotionVelocity

' Loop and test for two conditions 
'  Position capture or motion complete 
' 1. If a position is captured, move to the desired point 
' 2. If the motion completes, the program is finished 
WHILE( $Done == 0 )
	' Read # of captured values
    $NumberOfValues = DRIVEINFO X 5 WORD 0		
    IF( $NumberOfValues > 0 ) THEN
		' Stop collecting data. 
        DATAACQ X 1 OFF   			                            
                             
		' Read one captured position value from the drive array
		ARRAY X READ $CapturedPosition 0 1 UNITS

        MOVEABS X ( $CapturedPosition + $DistancePastRegistration ) $MotionVelocity
        WAIT MODE INPOS
        $Done = 1                                                                  
    ' Test for move done
	ELSEIF( AXISSTATUS(X, DATAITEM_AxisStatus) BAND AXISSTATUS_MoveDone ) THEN
        $Done = 1
    END IF
WEND

END PROGRAM
