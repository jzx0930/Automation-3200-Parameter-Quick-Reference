File Access Example

Before you run an AeroBasic example program, copy the example to your User Files directory. Copying the example makes sure that you have access to compile and run the program and prevents changes that you make from being overwritten if you install updated software.

View the example program in A3200 Motion Composer. Motion Composer adds color in the code which makes it easier to read. For best results, do one of the following.

  • Open a new file in Motion Composer. Copy and paste the code into the file.
  • OR
  • To examine the source code, open the example program from your User Files directory. You might have to copy the example program from the \A3200\Samples\AeroBasic folder first.

This example is one of many example programs. For a complete list of examples, refer to Example Programs.

To see a list of available hardware manuals, refer to Other Documentation. 

'				FileAccess.pgm
'
'
' This program reads & writes to files. To 
' observe the effects of this program, use 
' the Watch window to view variable values. 
' Also, commands can be added to use the 
' MSGDISPLAY command to display the data.

' Define the variables used by this program
DVAR $TempInt, $TempFloat, $TempDouble,  $TempLong
DVAR $InputFile1 , $InputFile2, $OutputFile1, $OutputFile2 
DVAR $FilePresent, $EndFileStatus, $ReadString 

' Close all open file handles for this task
FILECLOSE

' Test if the output files exists, and if so, delete it.
$FilePresent = FILEEXISTS "C:\TestOutput.txt" 			

IF( $FilePresent == 1 ) THEN
	' Erases the file.
	$strglob0 = "cmd /c del C:\TestOutput.txt" 
    EXEMODAL $strglob0
END IF

$FilePresent = FILEEXISTS "C:\TestFile.txt"		

IF( $FilePresent == 1) THEN
	' Erases the TestFile.txt file.
    EXEMODAL "cmd /c del C:\TestFile.txt"		 
END IF

' Open a file for writing data.
$OutputFile1 = FILEOPEN "C:\TestOutput.txt", 0
CALL TestErrCode

' Initialize the variables with values.
$TempInt = 10
$TempFloat = 4.257
$TempDouble = 100.6
$TempLong = 47

' Write data to the output file.
FILEWRITE $OutputFile1, $TempInt
CALL TestErrCode				
FILEWRITE $OutputFile1, $TempDouble
CALL TestErrCode			
FILEWRITE $OutputFile1, $TempLong
CALL TestErrCode 		
FILEWRITE $OutputFile1, $TempFloat
CALL TestErrCode

' Close the and save the file
FILECLOSE $OutputFile1
CALL TestErrCode

' Open a file for output
$OutputFile2 = FILEOPEN "C:\TestFile.txt", 0
CALL TestErrCode

' Write data into the output file stream
FILEWRITE $OutputFile2, "This is line 1"
CALL TestErrCode
FILEWRITE $OutputFile2, "This is line 2"
CALL TestErrCode
FILEWRITE $OutputFile2, "This is line 3"
CALL TestErrCode
FILEWRITE $OutputFile2, "This is line 4"
CALL TestErrCode

' Close and save the file
FILECLOSE $OutputFile2
CALL TestErrCode

END PROGRAM

' test error code
DFS TestErrCode
   IF( $info0 != 0 )      		
        TASKERROR $info0, $info1
   ENDIF
ENDDFS
