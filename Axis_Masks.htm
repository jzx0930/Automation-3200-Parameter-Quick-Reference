Axis Masks

Syntax:

 
<axisNameConstants>

Example:

 
X Y Z
 
X

Axis masks specify one or more axes and consist of a series of one or more axis names, where each letter must be separated by white space or a comma (for example, X Y Z and X,Y,Z are accepted, but XYZ is not). Axis masks can be specified as variables in some cases.

Use axes masks in ENABLE and HOME commands.

In the case of commands requiring axes masks (refer to example below), you cannot use constants. You must use a variable and special syntax (percent signs) is required.

Example

$global[0] = 3

' This line enables the first axis and the second axis.
ENABLE %$global[0]%
