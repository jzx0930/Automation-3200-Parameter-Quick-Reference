Valid Digital Binary Input

Digital inputs are specified as a numeric value, which represents the input type, axis, and an input bit number as listed in the following table.

These values are provided only for reference. If you change the value of a parameter from within a program or edit parameters in the parameter file directly, it must be coded in this format. However, if you use the Configuration Manager to change parameter values, this coding is done for you automatically.

Refer to the I/O Overview to view the available I/O on Aerotech drives.
