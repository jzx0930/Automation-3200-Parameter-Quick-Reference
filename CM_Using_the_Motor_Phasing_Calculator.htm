Using the Motor Phasing Calculator

Use the Motor Phasing Calculator to calculate the correct values for the CommutationOffset Parameter, FeedbackSetup Parameter, and ServoSetup Parameter. By calculating the values of these axis parameters, you can compensate for motor wiring problems. This calculator is available only when you are using an AC Brushless (Auto-MSET) motor or an AC Brushless (Hall-Effect Switches) motor.

Figure: Configuration Manager — Motor Phasing Calculator

To Use the Motor Phasing Calculator

  1. Make sure that it is safe for motion to occur on the selected axis.
    • Make sure that the selected axis is away from any end-of-travel limits or software limits.
    • Make sure that there is nothing blocking the selected axis.
  2. Open the Motor Phasing Calculator in Configuration Manager. Do one of the following.
    • On the Tools menu, select Other Calculators → Motor Phasing Calculator. Make sure that the correct Axis is selected in the calculator.
    • In the Viewer, select one of the following parameters. Then go to the Editor pane and click the Calculator button.
      • CommutationOffset Parameter
      • FeedbackSetup Parameter
      • ServoSetup Parameter
  3. In the Options section of the Motor Phasing Calculator, make sure that the value of the Test Current is correct.
  4. The Motor Phasing Calculator automatically sets the Test Current to 50% of the value of the AverageCurrentThreshold axis parameter. If the Motor Phasing Calculator fails to calculate results, the Test Current value might be too low.
  5. Click the Start button . While the axis travels one full electrical cycle, the calculator collects data and calculates the parameter results.
  6. Examine the calculation results to see if your parameters require changes.
    • If parameter changes are required, the calculator shows the results for the parameters in the New Value section of the Results pane.
    • If parameter changes are not required, the calculator shows a message in the Results pane that tells you the motor is phased correctly and no changes are required.
      • Figure: Motor Phasing Calculator — No Changes Required

  7. When you are done with the calculation, click OK.

To Use Advanced Options

You can use the Advanced Options of the calculator to increase the effectiveness of the Motor Phasing calculation. Make sure that you use the default options for the first time that you use the Motor Phasing Calculator on an axis. If the results that you get from using the default options are not satisfactory, you can specify a larger Step Interval or smaller Step Size to get more accurate results.

  1. In the Options section of the Motor Phasing Calculator, select the Use Advanced Options box.
    2. Figure: Motor Phasing Calculator — Advanced Options

  2. Enter a value for the Step Interval. This is the quantity of time, in milliseconds, that the calculator waits to let the encoder settle after the axis moves to a new angle in the electrical cycle.
  3. Enter a value for the Step Size. This is the number of degrees in the electrical cycle that the axis steps. If you specify a smaller value for Step Size, the calculator collects higher-resolution data.
  4. Click the Start button . The Motor Phasing Calculator uses the Advanced Options while it travels one full electrical cycle and collects data.

Approximating the Value of the CountsPerRev Parameter

The Motor Phasing Calculator records the total number of Encoder Counts that the axis travels during one full electrical cycle. The calculator uses this number to calculate the approximate value of the CountsPerRev axis parameter.

If the actual value of the CountsPerRev axis parameter is significantly different from the approximate value of this parameter, the Motor Phasing Calculator shows a warning. If you see this warning, do the following.

  • Make sure that you correctly configured the CountsPerRev axis parameter while using the Axis Parameter Calculator. Refer to Using the Axis Parameter Calculator for more information.
  • Refer to the hardware manual that is applicable to your motor or stage.
