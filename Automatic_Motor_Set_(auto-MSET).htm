Automatic Motor Set (Auto-MSET)

WARNING: If you use the Automatic Motor Set (Auto-MSET) method of commutation initialization, the motor moves a maximum of one-half of an electrical cycle when you enable the axis.

For AC brushless motors that do not have Hall-effect switches, the A3200 can use the Automatic Motor Set (Auto-MSET) method to initialize motor commutation. The drive uses this method when you set the MotorType Parameter to AC Brushless (Auto-MSET).

NOTE: The A3200 can also use the Commutation Search method to initialize motor commutation. Refer to Commutation Search for more information.

The Automatic Motor Set (Auto-MSET) method uses the steps that follow:

  1. The controller sends an ENABLE Command to the axis.
  2. The drive outputs current at a known, fixed, electrical vector that you specify for the AutoMsetAngle Parameter. This current ramps up from zero to the specified value of the AutoMsetCurrent Parameter for the quantity of time that is specified by the AutoMsetRampTime Parameter.
  3. The drive holds the level of current specified by the AutoMsetCurrent parameter for the quantity of time specified by the AutoMsetTime Parameter. This makes sure that the motor moves to the specified angle. The quantity of current that the drive holds must be high enough so that loading, friction, and other opposing forces have a minimum effect on the motor angle.
  4. The drive initializes motor commutation by assuming that the motor is now at a known angle. Then motor commutation continues normally. The drive uses the feedback device specified by the VelocityFeedbackType and VelocityFeedbackChannel parameters.
