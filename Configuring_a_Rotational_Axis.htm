Configuring a Rotational Axis

An axis that generates rotational motion can be configured to have a modulo position display by setting the RolloverCounts parameter to a value other than zero. Typically, the RolloverCounts parameter is set to the number of counts traversed during one rotation of the axis.

An axis that has the RolloverCounts parameter set to a nonzero value can use the shortest distance method to determine the direction of motion. Use the RolloverMode parameter to enable shortest distance motion.

When the shortest distance method is enabled, an axis will travel the shortest distance to get to the absolute position commanded by a move. Shortest distance motion also applies to all normalcy alignment moves of the axis when it is configured as a normalcy axis.

For example

Say that the units of an axis is set to degrees. The axis is at position 0° and is commanded to move to 355°. The axis will rotate -5° instead of rotating +355° and extra revolutions are removed.

Typically, you should configure a rotational axis as dependent with the AxisType parameter. Refer to Axis Type Overview for more information about dominant and dependent axes.
