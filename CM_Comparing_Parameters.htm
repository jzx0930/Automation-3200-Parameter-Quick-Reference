Comparing Parameters

In Configuration Manager you can compare one parameter file with another parameter file. You can also compare the parameters from an axis or task with parameters in other axes or tasks in the same parameter file.

Selecting the Parameters to Compare

  1. In Configuration Manager, open the Compare Parameters dialog box.
  2. On the Tools menu, select Compare Parameters or click the Compare Parameters button on the toolbar. When you open the Compare Parameters dialog box, if a parameter file is selected in the Network Explorer that file becomes the Source.
  3. Figure: Compare Parameters Dialog Box

  4. To select or change the Source, click the Source arrow and select an item from the tree or click Browse to select a parameter file from the file system.
  5. Click the Compare With arrow and do one of the following.
    • To compare the parameters from an axis or task with parameters in other axes or tasks in the same parameter file, select Axes/Tasks within Source. This is the default setting.
    • To compare the Source parameter file with another parameter file, select a file from the tree or click Browse to select a parameter file from the file system. When Axes/Tasks within Source is selected, the file tree is not available.
  6. Verify your selections and click Compare.

Parameter Comparison Results

The Parameter Comparison dialog box shows the results of your comparison.

When you select two different parameter files for your comparison, the application compares all axes and tasks from the first file with all axes and tasks from the second file.

When you use the same parameter file for your comparison by selecting Axes/Tasks within Source in the Compare Parameters dialog box, the application compares the first two axes with each other and the first two tasks with each other.

Parameter Comparison Options

To make an analysis of the results from your comparison, use the following options in the Parameter Comparison dialog box.

Figure: Parameter Comparison Options

Table:     Parameter Comparison Options

Figure Label Name Description

Parameter Categories

Shows all of the parameter categories that contain differences.

  • When you select a parameter category in this section, the Results section shows the results of your comparison only for the parameters in that category.

    • For Example

    If you select the Current Loop node, the Results section shows the differences in only the parameters that apply to the Current Loop Parameter Category.

  • When you select the Results node in this section, the Results section shows the results for all of the parameters in your comparison.

Compare Axis

NOTE: The Compare Axis option is available only when you compare two parameter files.

Configures the comparison results for one or two axes in two parameter files. For example, you can configure a custom comparison to compare the X axis from the Source file with the Y axis from the file that you compare with the Source file.

To See Comparison Results for One Axis
  1. In the Options section, click the Compare Axis arrow. The application shows a list of axes that you can compare.
  2. Select one axis that you want to compare from the Source file and the file that you compare with the Source file.
To See Custom Comparison Results for Two Axes
  1. In the Options section, click the Compare Axis arrow and select Custom Comparison.
  2. Click the First Axis arrow and select the axis that you want to compare from the Source file.
  3. Click the Second Axis arrow and select a different axis that you want to compare from the file that you compare with the Source file.

Compare Task

NOTE: The Compare Task option is available only when you compare two parameter files.

Configures the comparison results for one or two tasks in two parameter files. For example, you can configure a custom comparison to compare Task 1 from the Source file with Task 2 from the file that you compare with the Source file.

To See Comparison Results for One Task
  1. In the Options section, click the Compare Task arrow. The application shows a list of tasks that you can compare.
  2. Select one task that you want to compare from the Source file and the file that you compare with the Source file.
To See Custom Comparison Results For Two Tasks
  1. In the Options section, click the Compare Task arrow and select Custom Comparison.
  2. Click the First Task arrow and select the task that you want to compare from the Source file.
  3. Click the Second Task arrow and select a different task that you want to compare from the file that you compare with the Source file.

First Axis and Second Axis

You can use these options to see the comparison results for two axes in the following locations.

  • The same parameter file.
  • The Source file and the file that you compare with the Source file.
To See Comparison Results for Two Axes
  1. In the Options section, click the First Axis arrow and select the axis that you want to compare.
  2. Click the Second Axis arrow and select a different axis that you want to compare.

First Task and Second Task

You can use these options to see the comparison results for two tasks in the following locations.

  • The same parameter file.
  • The Source file and the file that you compare with the Source file.
To See Comparison Results for Two Tasks
  1. In the Options section, click the First Task arrow and select the task that you want to compare.
  2. Click the Second Task arrow and select a different task that you want to compare.

Highlight Differences box

Highlights the pieces of parameter values that are different. The Results section does not highlight these differences by default because the Highlight Differences box is not selected by default.

When you select this box, the Results section highlights the differences between the values of the axes and the values of the tasks in the two parameter files that you compared.
