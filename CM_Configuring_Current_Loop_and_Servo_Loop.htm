Configuring the Current Loop and the Servo Loop

Use the Axis Parameter Calculator in Configuration Manager to calculate the digital current loop gains. The calculator is the best way to configure the current loop because the gains do not change and they depend on some properties of the motor, including the resistance and inductance. If you are using an axis for the first time, you must configure the digital current loop gains.

You can also use the Axis Parameter Calculator to calculate servo loop gain values that change because of the mechanics of the system and the quantity of weight that is applied to the axis. If you change the load on the axis, you must adjust the servo loop gains. The calculator supplies you with servo loop gains that you can use to start your calculations. If you want to do more servo tuning, Aerotech recommends that you use Autotune or EasyTune.

To Configure the Current Loop

NOTE: Before you continue, make sure that you complete the Mechanics and Drive sections of the Axis Parameter Calculator.
  1. Expand the Current Loop section. If you want to make changes for only the current loop parameters, clear the Calculate boxes in the Results pane for all categories except for Current Loop Parameters.
    2. Figure: Current Loop Section

  2. Table:     Current Loop Section

    Field Name

    Description

    Phase Margin (Deg)

    If Phase MarginThe phase margin of a system is defined as the difference, in degrees, of the phase of the open-loop system at the crossover frequency and -180 degrees. The phase margin is a measure of system stability and provides a measure of robustness against variations in system dynamics. Increasing the phase margin increases system stability while decreasing the phase margin decreases stability and can eventually lead to an unstable system. If the phase margin is less than or equal to zero then the system is unstable. A larger phase margin results in a less oscillatory and better-damped system. If an open-loop system has multiple crossover frequencies, the overall phase margin is taken as the worse case or lowest phase margin. Typically, a phase margin between 30 and 60 degrees results in a reasonable trade-off between system performance and stability. is not at the default value of 45 degrees, change it to 45 degrees.

    Crossover Frequency (Hz)

    If Crossover FrequencyThe open-loop crossover frequency is the frequency at which the open-loop gain is equal to one (0 dB). The simplest model for motor has a single open-loop crossover frequency. The open-loop crossover frequency is a measure of system performance. A higher crossover frequency implies better system performance through improved position tracking and decreased response time. Too high of a crossover frequency can however degrade system performance by decreasing damping in the system and causing instabilities in the servo-loop. is not at the default value of 2000 Hz, change it to 2000 Hz.

    Actual Crossover Frequency (Hz)

    The calculator determines the crossover frequency based on the values you enter in other fields and information from various loop models. You cannot set this value.
  3. The Results pane of the calculator shows the Current Value and the New Value of all the parameters that change as a result of the settings that you specify in the calculator. Review these values. Make necessary corrections to the values in the Current Loop section and click OK. The Axis Parameter Calculator closes.
To Configure the Servo Loop Gains

NOTE: Before you continue, make sure that you complete the Mechanics, Drive, and Feedback sections of the Axis Parameter Calculator.
  1. Expand the Servo Loop section. If you want to make changes for only the servo loop parameters, clear the Calculate boxes in the Results pane for all categories except for Servo Loop Parameters.

  3. The Results pane of the calculator shows the Current Value and the New Value of all the parameters that change as a result of the settings that you specify in the calculator. Review these values. Make necessary corrections to the values in the Servo Loop section and click OK. The Axis Parameter Calculator closes.
  4. Save the active parameter file. If the parameter file you are working on is not the active parameter, set it as the active parameter file. Refer to Working with Parameters.
  5. Move the motor with the newly calculated servo loop gains.
    • If the motor is moving correctly, but you want to continue fine-tuning, refer to Autotuning an Axis.
    • If the motor does not move correctly, makes noises, vibrates, or oscillates, then the Calculator was unable to calculate the correct servo loop gains for the particular configuration. Verify that all of the settings in the Calculator are correct. Recommit the parameters and move the motor again. If the motor still does not move correctly, then you need to manually tune the axis.
