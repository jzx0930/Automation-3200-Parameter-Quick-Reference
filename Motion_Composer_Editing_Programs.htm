Editing Programs in Motion Composer

Use the following editors in Motion Composer to write AeroBasic programs.

Task Tabs

Each controller task is identified by a tab. You can open files in a specific task tab and use the files to debug programs that are running in that task. Aerotech recommends that you open files for a program in the tab for the task in which the program is running. If you open files in multiple task tabs, the files are linked together. If you make changes to one file, the changes are applied to all linked files.

Table:     Motion Composer Task Tabs

Task Tab State Indicators

Description

White

Shows that the application is not connected to the A3200 or the task is inactive on the current controller. Use the EnabledTasks parameter to set which tasks are active.

Blue

Shows that the task on the current controller is active.

Red

Shows that the task on the current controller has generated a fault.

File Tabs

The file tabs have the following features.

Figure: File Tab Features

Table:     Motion Composer File Tabs

File Tab Features

Description

Locked Indicator

While a program is actively being debugged, all files that are part of that program (*.pgm) will be locked. Locked files are identified by an icon in the file tab that comes before the file name. You cannot make changes to those files until the files are unlocked.

Modified Indicator

Files identified with an asterisk (*) have been changed after they were opened.

Save

Right-click the file tab of an open file that you want to save and click Save.

Close

Right-click the file tab of an open file that you want to close and click Close.

Move To

Right-click the tab of an open file that you want to move to a different task, select Move To, and click on the task to which you want to move the file. Since files that are the same are linked, this is equivalent to opening the file in the Destination task and closing the file in the Source task.

Copy To

Right-click the tab of an open file that you want to copy to a different task, select Copy To, and click on the task to which you want to copy the file. Since files that are the same are linked, this is equivalent to changing to the destination task and manually opening the file.

File Navigator

Click the Open Files tab to browse through all open files.

Note: When you try to close a file that you changed but did not save, the application asks you if you want to save the file.

Editor Features

The following features are available in the editor.

  • Line number indicators Line number indicators show the number of each line in the file.
  • Indicator margin
  • Breakpoints Breakpoints in files are identified by the breakpoint icon . When you debug programs, lines that contain breakpoints cause the program to pause before those lines are executed.
  • Current Line indicator When a program is paused, the next line to be executed is identified by the current line icon .
  • Status bar
  • File path Shows the full path of the file you are editing.
  • Line number Shows the line number of the cursor.
  • Column number Shows the column number of the cursor.
  • Character number Shows the character number of the cursor.
  • Syntax highlighting Syntax highlighting includes keywords, strings, and built-in commands and functions that are highlighted with corresponding colors to make programming easier.

Network Explorer

User Files

The User Files node of the Network Explorer shows all of the files in the User Files directory, which can be changed as a user preference.

File Nodes

In the User Files node, files and directories in the User Files directory are shown as nodes.

  • Open Opens the file in the current instance of Motion Composer if the file is identified as an AeroBasic file type. If the file is not an AeroBasic file type, it is sent to the shell and Windows tries to find the correct program to open the file.
  • Rename Renames the file in place on the file system.
  • Delete Deletes the file from the file system.
  • New Folder Makes a new directory in the selected directory.
