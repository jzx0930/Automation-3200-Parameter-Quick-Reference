AsynchronousCommunicationError

An error occurred during an asynchronous communication: <Location>

Table:     Location Codes

Location Code Interpretation

1

A timeout occurred while waiting for an asynchronous receive event.

2

An ack_complete error occurred during an asynchronous transmit.

3

An ack_complete error occurred during an asynchronous receive.

4

An incorrect node responded during an asynchronous communication.

5

A timeout occurred while waiting for the asynchronous transmit active bit.

6

A timeout occurred while waiting for the asynchronous receive active bit.

7

A drive did not echo the release command.

8

An unidentified FireWire or HyperWire bus node member was detected.

9

An incorrect device number setting was returned from a drive.

10

An asynchronous command was sent to an invalid node.

Cause

An error occurred during asynchronous communications.

Asynchronous communications are used during the startup and initialization procedure. An error in these communications usually identifies a problem with the hardware.

Solution

To Correct Hardware Problems

  1. Close all of the A3200 applications.
  2. Shut down the A3200. Click Start → All Programs → Aerotech → Automation 3200 → Shutdown A3200. A black Command Prompt window comes into view momentarily.
  3. Power off all of the hardware for 30 seconds.
  4. Power on all of the hardware.
  5. Run all of the A3200 applications again.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
