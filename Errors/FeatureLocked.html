The sections that follow tell you what might cause the error and give you solutions.

You must have the Fiber Optic option to use this feature.

Cause

You tried to use a Fiber Optic command, and your license key does not include the Fiber Optic option.

Solution

To use Fiber Optic features, purchase the Fiber Optic option.

You must have the CNC Option to use this feature.

Cause

You tried to use a CNC command, and your license key does not include the CNC Option.

Solution

To use CNC features, purchase the CNC Option.

You must have the Professional Option to use tasks 5-31.

Cause

You tried to enable or run a program on tasks 5-31, and your license key does not include the Professional Option.

Solution

To use tasks 5-31, purchase the Professional Option.

To prevent this error, open Configuration Manager and change the value of the System → Enabled Tasks parameter to the default value.

To be able to change parameters that are not included in your controller license, on the Tools menu in Configuration Manager, click Preferences. In the Preferences dialog, click Parameters and set the Show Only Licensed Parameters to False.

You must have the Dynamic Controls Toolbox option to use this feature.

Cause

You tried to set a Dynamic Controls Toolbox parameter, and your license key does not include the Dynamic Controls Toolbox option.

Solution

To use Dynamic Controls Toolbox features, purchase the Dynamic Controls Toolbox option. To prevent this error, open Configuration Manager and change the values of the following Dynamic Controls Toolbox parameters to the default values.

Axis → Dynamic Controls Toolbox → Command Shaping → CommandShaperSetup

Axis → Dynamic Controls Toolbox → Harmonic Cancellation → HarmonicCancellationSetup

To be able to change parameters that are not included in your controller license, on the Tools menu in Configuration Manager, click Preferences. In the Preferences dialog, click Parameters and set Show Only Licensed Parameters to False.

You must have the Professional Option to use this feature.

Cause

You tried to collect more than one PC Modbus Master connection, and your license key does not include the Professional Option.

Solution

To collect more than one PC Modbus Master connection, purchase the Professional Option.

To prevent this error, configure only one PC Modbus Master connection.

The time limit for enabling demo mode is expired.

Cause

You tried to enable demo mode and the time limit for demo mode is expired.

Solution

Demo mode is available for two weeks after it is first enabled. After two weeks, you must purchase the correct license feature to continue using the features that are supplied in demo mode.

You entered an incorrect renewal code.

Cause

You entered an incorrect demo code in Feature Manager.

Solution

Contact your Aerotech sales representative and ask about the license options that are available for your controller.

You must have the EtherCAT fieldbus option to use this feature.

Cause

You tried to configure an EtherCAT device, and your license key does not include the EtherCAT fieldbus option.

Solution

To use an EtherCAT device, purchase the EtherCAT fieldbus option.

You must have the PROFINET fieldbus option to use this feature.

Cause

You tried to configure a PROFINET device, and your license key does not include the PROFINET fieldbus option.

Solution

To use a PROFINET device, purchase the PROFINET fieldbus option.

You must have the Enhanced Tracking Control option to use this feature.

Cause

You tried to enable the Enhanced Tracking Control feature by setting the EnhancedTrackingSetup parameter, and your license key does not include the Enhanced Tracking Control option.

Solution

To use the Enhanced Tracking Control feature, purchase the Enhanced Tracking Control option. To prevent this error, open Configuration Manager and change the value of the following Enhanced Tracking Control parameter to the default value.

Axis → Enhanced Tracking Control → EnhancedTrackingSetup

To be able to change parameters that are not included in your controller license, on the Tools menu in Configuration Manager, click Preferences. In the Preferences dialog, click Parameters and set Show Only Licensed Parameters to False.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
