GantryMotionError

An error occurred while performing gantry motion: <Location>

Table:     Location Codes

Location Code Interpretation

1

You cannot command motion on a gantry slave axis.

2

The HomeOffset configuration violates the GantrySeparationThreshold parameter.

3

You cannot command this type of motion on a gantry.

The sections that follow tell you what might cause the error and give you solutions.

You cannot command motion on a gantry slave axis.

Cause

You cannot command motion on a gantry slave axis.

Solution

Change your program to command the gantry master axis as an alternative to the gantry slave axis.

The HomeOffset configuration violates the GantrySeparationThreshold parameter.

Cause

The HomeOffset parameters configured on the gantry axes violate the GantrySeparationThreshold parameter. This usually shows that the HomeOffset parameters on the gantry axes are incorrectly configured.

Solution

Recalculate the HomeOffset parameters on the gantry axes so that they do not violate the GantrySeparationThreshold parameter. Refer to Configuring Parameters for Gantry Mode for information about how to correctly configure the HomeOffset parameter values on a gantry.

You cannot command this type of motion on a gantry.

Cause

You tried to execute the specified motion command on an axis that is configured as a gantry axis.

Solution

Change your program to execute the motion command on an axis that is not a gantry master axis or a gantry slave axis.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
