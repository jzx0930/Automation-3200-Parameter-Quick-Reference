CallbackTimeout

A timeout occurred while responding to a callback.

A timeout occurred while the SMC was waiting for a callback command to issue a continue. The sections that follow tell you what might cause the error and give you solutions.

Cause

The A3200 is configured in Remote Server mode, and you are running a program that contains a callback command on the server PC.

Solution

When you operate in Remote Server mode, you must use the client PC to run AeroBasic programs or use applications that register for callbacks. If you do not want to use a Remote Server configuration, change the Connection Mode to Local by using the Connection Settings dialog in Configuration Manager.

 

Cause

You are using a callback command, but there are no applications registered to handle the callback.

Solution

  • Change the AeroBasic program so that it does not include the callback command.
  • Start the application that handles the callback.

 

Cause

You are using Remote Desktop to control the PC that is running the A3200.

Solution

  • Log into Windows directly before you connect with Remote Desktop.
  • Use the PC directly instead of using Remote Desktop.

 

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
