AxisNotEnabled

The axis is not enabled.

Cause

You tried to do an action, such as motion, on an axis that is not enabled.

Solution

Enable the axis and try the operation again. If the axis does not enable, it is typically caused by of one of these problems:

  • The controller generated an axis fault, such as a Position Error Fault or a Position Feedback Fault, that you configured in the FaultMask parameter.
    • Click the Acknowledge All button to try to clear the fault. Then try to enable the axis again.
  • The drive is not powered on or is not connected to the FireWire or HyperWire bus.
    • Examine all cables to make sure that they are connected correctly.
  • An external emergency stop circuit (ESTOP) is open, which prevents the axis from enabling.
    • Correct the emergency stop condition.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
