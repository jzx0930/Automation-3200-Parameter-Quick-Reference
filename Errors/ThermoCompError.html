ThermoCompError

An error occurred using the ThermoComp feature: <Location>

Table:     Location Codes

Location Code Interpretation

1

The calibration cannot be reset while ThermoComp is turned on.

2

The operation mode is not valid.

3

The range of the thermal state of the stage is not valid.

4

The update interval is not valid.

5

The configuration is not valid.

6

The analog input number is out of range.

7

Manual updates are not permitted when the feature is off.

Optional Purchase Required: The information in this topic applies only if you purchased the ThermoComp™ feature on your Aerotech positioning product.

The sections that follow tell you what might cause the error and give you solutions.

The calibration cannot be reset while ThermoComp is turned on.

Cause

A THERMOCOMP CONFIG RESET command was issued while ThermoComp™ was turned on.

Solution

Before you issue a THERMOCOMP CONFIG RESET command, issue the THERMOCOMP OFF Command.

The operation mode is not valid.

Cause

A value that is not valid was specified for the <Mode> argument of the THERMOCOMP CONFIG MODE Command.

Solution

To operate ThermoComp™ on a linear stage, specify 0 for the <Mode> argument of the THERMOCOMP CONFIG MODE Command.

To operate ThermoComp™ on a ball-screw stage, specify 1 for the <Mode> argument of the THERMOCOMP CONFIG MODE Command.

The range of the thermal state of the stage is not valid.

Cause

A minimum range was specified for the thermal state of the stage. But the specified minimum range is greater than the specified maximum range of the thermal state of the stage.

Solution

In the THERMOCOMP CONFIG VALUES Command, make sure that the value of the <MinTemp> argument is less than the value of the <MaxTemp> argument.

The update interval is not valid.

Cause

An interval at which thermal compensation is updated was specified and is not valid.

Solution

Make sure that you specify the value of the <UpdateInterval> argument of the THERMOCOMP CONFIG UPDATEINTERVAL Command in seconds. The minimum value is 0.1 second.

The configuration is not valid.

Cause

An unknown configuration error occurred.

Solution

Make sure that the values are correct for all the arguments of each command that configures the ThermoComp™ feature. To see a list of the commands, refer to ThermoComp Command Category.

The analog input number is out of range.

Cause

An analog input number that does not exist was specified.

Solution

Find the analog input number of the axis to which the sensor that reads the thermal state is connected on the drive. Use that number to specify the <AnalogChannel> argument of the THERMOCOMP CONFIG MODE Command.

Manual updates are not permitted when the feature is off.

Cause

A THERMOCOMP UPDATE ONCE command was issued while ThermoComp was turned off.

Solution

Before you issue a THERMOCOMP UPDATE ONCE command, you must turn on the ThermoComp feature with the THERMOCOMP ON/OFF Command.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
