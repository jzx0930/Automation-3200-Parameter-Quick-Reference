Verify that the sample time, number of points and signals are correctly configured.

If this problem occurred because of insufficient data collection memory, increase the value of the DataCollectionPoints and DataCollectionItems parameters.

The sections that follow tell you what might cause the error and give you solutions.

An invalid number of data collection records was requested.

Cause

You tried to configure data collection with an invalid number of data collection records (data collection items). You must configure at least one record when configuring data collection.

Solution

Change your data collection configuration to configure at least one data collection item.

An invalid number of data collection samples was requested.

Cause

You tried to configure data collection with an invalid number of data collection samples. You must configure at least one sample when configuring data collection.

Solution

Change your data collection configuration to configure at least one data collection sample.

An invalid data collection sample time was requested.

Cause

You tried to configure data collection with an invalid sample time.

Solution

Change your data collection configuration to specify a valid sample time. (Refer to DATACOLLECT START command for a list of valid data collection sample times.)

An attempt was made to configure more data collection records than specified.

Cause

An error occurred configuring data collection records.

Solution

Reapply the data collection configuration. If you are using the DATACOLLECT commands, use the DATACOLLECT ITEM RESET command before applying your data collection configuration.

There is not enough memory allocated for data collection.

Cause

You exceeded the quantity of memory available for data collection when configuring data collection items or data collection samples.

Solution

Data collection memory is allocated on a controller reset and is determined by the DataCollectionPoints and DataCollectionItems parameters. Increase the DataCollectionPoints and DataCollectionItems parameters to match the number of items or number of points that you are allocating in your program.

An invalid data item was requested.

Cause

You tried to add a data collection item that was outside of the range of valid items.

Solution

Change your program to configure valid data collection items.

The drive cannot be configured for this many data items.

Cause

You exceeded the maximum number of items that can be collected on a particular drive.

Solution

Change your data collection configuration to collect fewer items. This error is caused by a drive limitation and increasing the DataCollectionItems parameter will not prevent this error from occurring.

The drive is out of data collection memory.

Cause

You exceeded the maximum quantity of memory available for data collection on a particular drive.

Solution

Change your data collection configuration to collect fewer items or fewer number of points. This error is caused by a drive limitation and increasing the DataCollectionPoints and DataCollectionItems parameters will not prevent this error from occurring.

The drive did not return a valid data collection response.

Cause

An error occurred applying your data collection configuration.

Solution

Reapply the data collection configuration. If this error continues to occur, power cycle all drives, reset the controller, and try applying the data collection configuration again.

An invalid index was requested.

Cause

You configured an item for data collection but the index of that item is invalid. This can occur when the index specified for an axis item or task item is not between 0 and 31.

Solution

  • Change the data collection configuration so that axis items specify one of the 32 axes. The axis index must be greater than or equal to 0 and less than or equal to 31.
  • Change the data collection configuration so that task items specify one of the 32 tasks. The task index must be greater than or equal to 0 and less than or equal to 31.
An invalid sample trigger was specified.

Cause

You specified a sample trigger that is not supported.

Solution

Specify a sample trigger that is part of the DataCollectionSampleTrigger enumeration. Refer to the A3200 Programming Help file for more information.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
