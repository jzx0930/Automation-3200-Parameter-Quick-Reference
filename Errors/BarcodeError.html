BarcodeError

An error occurred encoding a barcode: <Location>

Table:     Location Codes

Location Code Interpretation

1

The input string size is not valid.

2

The input string is too large to fit in the encoding format that you specified.

3

The encoding type is not valid.

4

The encoding format is not valid.

The sections that follow tell you what might cause the error and give you solutions.

The input string size is not valid.

Cause

The input string size is zero.

Solution

You must specify a string with a non-zero length.

The input string is too large to fit in the encoding format that you specified.

Cause

You tried to encode a string that is too large to fit in the encoding format that you specified.

Solution

If you specified an encoding format that is not Auto, increase the size of the encoding format or use the Auto encoding format. If you are using Auto for the encoding format, then the string is too large to be included.

The encoding type is not valid.

Cause

You specified an encoding type that is not in the valid range of values.

Solution

Specify an encoding type in the valid range of values.

The encoding format is not valid.

Cause

You specified an encoding format that is not in the valid range of values.

Solution

Specify an encoding format in the valid range of values.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
