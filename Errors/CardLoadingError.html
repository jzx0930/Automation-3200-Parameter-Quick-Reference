The sections that follow tell you what might cause the error and give you solutions.

The HyperWire card was not identified.

Cause

The controller did not identify the version of the HyperWire card before it tried to update the firmware.

Solution

Make sure that you issue the CARDLOADER ID Console command before you issue the CARDLOADER ALL Console command. If this error continues to occur, contact Aerotech Global Technical Support.

An error occurred while the controller tried to read the load file.

Cause

An error occurred while the controller tried to read the load file.

Solution

Close Console, open Console, and then try to load the HyperWire card again. If this error continues to occur, contact Aerotech Global Technical Support.

The controller did not allocate sufficient memory to the load file.

Cause

A CRC checksum error occurred during data transmission.

Cause

An error occurred while the controller tried to transmit the load image.

Solution

Warning: Before you restart your PC, use the procedure in this Solution to make sure that you correctly reload the HyperWire card.

Close Console, open Console, and then try to load the HyperWire card again. If this error continues to occur, contact Aerotech Global Technical Support.

No HyperWire card is configured.

Cause

An error occurred while the controller tried to identify a HyperWire card.

Solution

Make sure that a HyperWire card is configured and is the selected card in the Connection Settings dialog of Configuration Manager. To access this dialog, select Controller → Connection Settings in the application.

The controller was unable to re-establish PCIe configuration after upgrading the firmware.

Cause

The controller did not rewrite the PCIe configuration header after the firmware card was upgraded.

Solution

Restart your PC to repopulate the PCIe header. The HyperWire card should now use the new firmware to communicate.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
