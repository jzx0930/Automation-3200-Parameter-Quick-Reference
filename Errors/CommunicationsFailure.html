CommunicationsFailure

A FireWire or HyperWire communication error occurred: <Location>

Table:     Location Codes

Location Code Interpretation

1

The SMC received an unsolicited response from the drive.

2

The SMC could not set the isochronous transmit event.

3

The SMC could not set the isochronous receive event.

Cause

An error occurred with the FireWire or HyperWire communications.

Solution

Power cycle all drives and restart the PC.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
