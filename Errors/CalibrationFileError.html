The sections that follow tell you what might cause the error and give you solutions.

An invalid argument was specified.

Cause

You specified an invalid argument after :START, :START2D, :GALVOPOWER2D, or :GALVO2D in an axis calibration file.

Solution

  • Make sure that all of the arguments in the axis calibration file are spelled correctly.
  • Make sure that all of the arguments in the axis calibration file are valid for the type of calibration that you are using.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
