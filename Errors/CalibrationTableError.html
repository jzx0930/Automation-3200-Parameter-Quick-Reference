CalibrationTableError

An error occurred with a calibration table: <Location>

Table:     Location Codes

Location Code Interpretation

1

The axis calibration table item is invalid.

2

The axis calibration table number is invalid.

3

The axis calibration table point is invalid.

4

The axis calibration mode is invalid.

5

An error occurred while allocating the axis calibration table.

6

The axis calibration table is not allocated.

7

The axis calibration table is in use.

8

The maximum number of axis calibration tables was exceeded.

9

The 2D axis calibration row number is invalid.

10

The 2D axis calibration column number is invalid.

11

The gain calibration table number is invalid.

12

The gain calibration table point is invalid.

13

The gain calibration mode is invalid.

14

An error occurred while allocating the gain calibration table.

15

The gain calibration table is not allocated.

16

The gain calibration table is in use.

Cause

An axis calibration file is invalid or is not configured correctly.

Solution

Verify that you are using the correct axis calibration file for your system. The axis calibration file is configured by using the Network Explorer in Configuration Manager. Refer to Axis Calibration for more information.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
