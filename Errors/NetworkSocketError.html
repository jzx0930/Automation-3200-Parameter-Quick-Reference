NetworkSocketError

An error occurred while doing a network socket operation: <Location>

Table:     Location Codes

Location Code Interpretation

1

A timeout occurred while waiting for a network socket operation to complete.

2

A connection with the specified ID does not exist.

3

An error occurred when you passed an invalid argument to a network socket command.

4

The remote end of the connection terminated the connection.

5

An error occurred when you tried to convert a value to another data type.

6

The requested operation was canceled.

The sections that follow tell you what might cause the error and give you solutions.

A timeout occurred while waiting for a network socket operation to complete.

Cause

The requested operation could not complete before the timeout occurred. This error might have been caused if the timeout that you specified is too low and does not allow the requested operation to complete.

Solution

Use the NETWORK TIMEOUT command to increase the timeout.

A connection with the specified ID does not exist.

Cause

You specified a connection ID or server socket ID that does not correspond to a connection that is currently open.

Solution

Change the specified connection ID or server socket ID to a valid identifier.

An error occurred when you passed an invalid argument to a network socket command.

Cause

You specified an invalid value as an argument to a network socket command. For example, if you specify a TCP/IP port number that does not exist (outside the range of 0-65535), this is considered passing invalid arguments to network socket commands.

Solution

Change the specified value to a valid value.

The remote end of the connection terminated the connection.

Cause

The remote machine closed the connection unexpectedly. This can be caused by a network service interruption, hardware fault, or software fault on the remote machine.

Solution

Examine the networking hardware that connects the local and remote machines and examine the networking software configuration.

An error occurred when you tried to convert a value to another data type.

Cause

You specified a value that cannot be converted to the specified type. For example, the string, "foo" cannot be converted to a byte.

Solution

Change the value that you specified so it can be converted to the data type that you specified.

The requested operation was canceled.

Cause

The task that made the network connection or service socket has terminated and has requested to close the connection or server socket. As a result of the connection or server socket closing, the task that made the network connection or server socket has been terminated.

Solution

Determine the cause for why the task terminated. If the task does not continue to terminate, then the connections and server sockets that the task makes do not close.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
