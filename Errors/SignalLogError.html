SignalLogError

An error occurred with Signal Log: <Location>

Table:     Location Codes

Location Code Interpretation

1

Signal Log cannot be triggered in the current state.

2

Data cannot be retrieved from Signal Log because it is not enabled or not ready.

3

A Signal Log sample that is not valid was requested.

4

An internal error occurred during Signal Log initialization.

The sections that follow tell you what might cause the error and give you solutions.

Signal Log cannot be triggered in the current state.

Cause

You issued the SIGNALLOG TRIGGER command while Signal Log was writing data for a previous trigger event.

Solution

Signal Log must be enabled and collecting new data before you can use the SIGNALLOG TRIGGER command. Use the SignalLogSetup parameter to enable Signal Log. If you previously triggered Signal Log and the controller is writing that data to a .DAT file, you must wait for Signal Log to start collecting new data before you can trigger Signal Log again.

Data cannot be retrieved from Signal Log because it is not enabled or not ready.

Cause

The controller cannot retrieve data from Signal Log. This is an internal error that does not normally occur.

Solution

Reset the controller. If the problem continues to occur, record the text of the error message and contact Aerotech Global Technical Support.

A Signal Log sample that is not valid was requested.

Cause

The controller cannot retrieve data from Signal Log. This is an internal error that does not normally occur.

Solution

Reset the controller. If the problem continues to occur, record the text of the error message and contact Aerotech Global Technical Support.

An internal error occurred during Signal Log initialization.

Cause

Signal Log cannot initialize because of an internal error.

Solution

Record the text of the error message and contact Aerotech Global Technical Support.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
