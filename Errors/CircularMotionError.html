CircularMotionError

An error occurred while performing circular motion: <Location>

Table:     Location Codes

Location Code Interpretation

1

A circular command with the same start and end positions cannot be specified by using the R method.

2

The circular center was not specified.

3

The circular coordinate plane does not match the cutter radius compensation coordinate plane.

4

The circular coordinate plane does not match the normalcy coordinate plane.

5

The circular center is over-specified.

The sections that follow tell you what might cause the error and give you solutions.

A circular command with the same start and end positions cannot be specified by using the R method.

Cause

You tried to specify a full circle using the R circular method.

Solution

You must use the IJ method when commanding a full circle. Refer to CW or CCW commands for more information about the R and IJ methods.

The circular center was not specified.

Cause

You did not specify circular offsets as arguments to a G2 or G3 command.

Solution

You must specify the appropriate arguments when commanding a circle. Refer to G2 or G3 commands for more information about required arguments.

The circular coordinate plane does not match the cutter compensation coordinate plane

Cause

Cutter compensation was active and you executed a circular command that is not in the same coordinate plane as the cutter compensation axes.

Solution

Verify that you have set the correct cutter compensation axes (G44 command) and are using the correct coordinate plane (G17, G18, G19) for the circular move.

The circular coordinate plane does not match the normalcy coordinate plane.

Cause

Normalcy was active and you executed a circular command that is not in the same coordinate plane as the normalcy axes.

Solution

Verify that you have set the correct normalcy axes (NORMALCY AXES command) and are using the correct coordinate plane (G17, G18, G19) for the circular move.

The circular center is over-specified.

Cause

You tried to command a circle using both the R and IJ methods.

Solution

Correct your program to specify the correct arguments to the circular command. Refer to CW or CCW commands for more information.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
