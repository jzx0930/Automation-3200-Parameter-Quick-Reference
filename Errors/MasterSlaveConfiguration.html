MasterSlaveConfiguration

A master/slave configuration error occurred: <Location>

Table:     Location Codes

Location Code Interpretation

1

Master/slave motion cannot be configured if two axes have different position update rates.

2

Master/slave motion cannot be configured on a gantry slave.

3

An invalid master axis index was specified.

4

An axis cannot be its own master.

The sections that follow tell you what might cause the error and give you solutions.

Master/slave motion cannot be configured if two axes have different position update rates.

Cause

You tried to configure gearing, camming, or gantry motion between axes that have different position update rates. For more information, refer to the table that follows.

Table:     Position Update Rates

Drive Type Position Update Rate

GL4

48 kHz

Ndrive FCL

48 kHz

Ndrive FLS

48 kHz

Ndrive QL and QLe

24 kHz

Nmark CLS, GCL, and SSaM

48 kHz

Servo Drives

8 kHz

XL4s

48 kHz

XR3-XSL3

8 kHz

XR3-XSP3

8 kHz

Solution

Change your configuration so that the master and slave axes in gearing, camming, or gantry configuration use the same position update rate.

Master/slave motion cannot be configured on a gantry slave.

Cause

You tried to configure gearing or camming motion on a gantry slave axis.

Solution

Change your program to do gearing or camming on the gantry master axis. Refer to Master and Slave Motion Overview for more information about configuring Master and Slave motion.

An invalid master axis index was specified.

Cause

You tried to enable gearing on an axis that is not configured for gearing.

Solution

Use the GEAR MASTERCONFIG command to configure gearing before you issue the GEAR ON command.

An axis cannot be its own master.

Cause

You tried to configure gearing, camming, or gantry motion and one or more axes references itself as its master.

For example

This error can occur if you configure Axis Y as a slave to Axis X. Then you configure Axis X as a slave to Axis Y. This creates a circular reference.

Solution

Change your gearing, camming, and gantry configuration settings to make sure that an axis does not reference itself as its master.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
