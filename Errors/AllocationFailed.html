AllocationFailed

An attempt to allocate memory on the SMC failed: <Location>

Table:     Location Codes

Location Code Interpretation

1

The controller was unable to allocate the memory required for a specific operation.

2

The controller was unable to allocate the memory required for data collection.

3

The controller was unable to allocate the memory required for lookahead.

The sections that follow tell you what might cause the error and give you solutions.

The controller was unable to allocate the memory required for a specific operation.

Cause

The controller was unable to allocate the requested quantity of memory. This could be caused by any of the following situations.

  • You have too many AeroBasic programs loaded on the controller.
  • Your AeroBasic programs are too large.
  • You have too many AeroBasic program variables.
  • A cam table or calibration file is too large.
  • The value of one or more of the following parameters is set too high. If the error appears on reset, the problem is an incorrect parameter value.
    • CallStackSize
    • CannedFunctions
    • DataCollectionItems
    • DataCollectionPoints
    • GlobalAxisPoints
    • GlobalDoubles
    • GlobalStrings
    • MaxLookaheadMoves
    • ModeStackSize
    • MonitorStatements
    • SignalLogAxes
    • SignalLogPointsAfterTrigger
    • SignalLogPointsBeforeTrigger
    • SignalLogTasks
    • TaskAxisPoints
    • TaskDoubles
    • TaskStrings

Solution

A small quantity of memory is available to the controller. Increasing the quantity of RAM in your PC will probably not solve this problem because the controller uses non-paged pool memory, which is limited by Windows. You need to decrease the quantity of memory that is required by your parameter settings and/or AeroBasic programs.

If this errors occurs on reset, it is probably a parameter problem and you need to lower the value of the parameters that require memory allocation. If this error occurs when downloading an AeroBasic program, it is probably caused by the size of the AeroBasic program and you need to decrease the size of the program and program variables. You can also make more memory available by issuing a Remove Program to remove programs that are loaded on the controller.

The controller was unable to allocate the memory required for data collection.

Cause

The controller failed to allocate the memory required for data collection.

Solution

Decrease the value of the DataCollectionItems parameter and/or the DataCollectionPoints parameter.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
