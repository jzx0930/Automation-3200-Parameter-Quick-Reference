An unknown or invalid drive type was detected.

Cause

An error occurred while the controller tried to read the load file or tried to communicate with one or more drives.

Solution

You must close Firmware Loader before connecting.

Cause

Solution

No FireWire or HyperWire card is configured.

Cause

You tried to use Firmware Loader while the A3200 is configured to operate without a drive communication interface card.

Solution

  • If you do not want to use a FireWire or HyperWire card, you cannot use Firmware Loader. Close Firmware Loader.
  • If you want to use a FireWire or HyperWire card and you did not configure the card, do the procedure that follows.
    1. Install the FireWire or HyperWire card. Refer to Installing the FireWire or HyperWire™ Card .
    2. Configure the FireWire or HyperWire card for the A3200. Refer to Configuring and Verifying the FireWire or HyperWire™ Card.
  • If you want to use a FireWire or HyperWire card and you previously configured the card, use the procedure that follows to disable the Operate without a Card selection.
    1. Close Firmware Loader.
    2. Open Configuration Manager.
    3. On the Controller menu, select Connection Settings. The Connection Settings dialog comes into view.
    4. Select the Drive Communication tab.
    5. In the Drive Communication Interface section, select one of the following.
      • Automatically select a card
      • Select the card to use
    6. Click OK.
The load file is not compatible with this version of the controller.

Cause

You tried to use a load file that is not compatible with your version of the controller.

Solution

Use the load file that was installed with your software. If this error continues to occur, contact Aerotech Global Technical Support.

One or more drives reported an incorrect controller family.

Cause

One or more drives reported a controller family that is not A3200.

Solution

Make sure that you are only connected to A3200 drives. After you do this, close Firmware Loader, power cycle all drives, and open Firmware Loader again. If this error continues to occur, contact Aerotech Global Technical Support.

One or more drives do not support this firmware version.

Cause

Solution

One or more drives have an obsolete hardware version.

Cause

Solution

One or more drives are not authorized to use this load file.

Cause

You tried to load a firmware version that is not authorized to be used with one or more drives.

Solution

Use a newer version of the controller software. If this error continues to occur, contact Aerotech Global Technical Support.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
