Accuracy Correction

Accuracy correction of mechanics compensates for the errors within the mechanics, such as a ball screw. It is the most common type of axis calibration. Accuracy correction precisely adjusts and improves the performance of the mechanics, but does not solve mechanical problems.

Use a precision reference source such as a laser interferometry system to map errors. Then use that data to produce a calibration table to compensate for the errors. The axis that is corrected is the master axis.

Note: To see the calibration file data, use the DUMPAXISCAL Console command in Console.
Note: All corrections are bidirectional, meaning that the correction value applied for a given position of the master axis is the same regardless of the sign or value of its velocity.
