Controller Options

Aerotech offers optional A3200 features. You can try some of these optional features by using them in Feature Demonstration. Refer to Unlocking Optional Features for more information. If you have questions about how to use a feature, contact Aerotech Global Technical Support. To purchase an optional feature, contact your Aerotech sales representative.

Note: Controller Options and Motion Composer Options are sold separately from the A3200.

CNC Option

Supplies G-Code and M-Code execution as well as advanced motion control, program control, and I/O capabilities.

Dynamic Controls Toolbox Option

A value-added collection of algorithms that you can use for Gain Scheduling, Harmonic Cancellation, Command Shaping, and Cross-Axis Feedforward.

Enhanced Throughput Module Option

Improves machine throughput by measuring base or frame vibration.

Enhanced Tracking Control Option

Improves machine performance by decreasing tracking errors that occur.

Fiber Optic Option

Aligns fiber optic devices.

Professional Option

Supplies more tasks and Modbus connections.

Fieldbus Options

Connect to multiple fieldbus slave devices and access the I/O points directly from AeroBasic.

EtherCAT Option

The fieldbus module that accesses EtherCAT enabled devices from the SMC.

PROFINET Option

The fieldbus module that accesses PROFINET enabled devices from the SMC.
