Subroutine Argument Existence Testing

Syntax:

 
$<axis>.DEFINED

Example:

 
IF $X.DEFINED == 1 THEN
    $global[1] = $X            ; $X argument is passed to subroutine
ENDIF

Use the .DEFINED keyword to determine if a particular parameter was passed to the subroutine or program.

Value

State

Definition

1

TRUE

The parameter was passed.

0

FALSE

The parameter was not passed.

If you reference a parameter that was not passed into a program or subroutine, you will receive a ParameterNotPassed error.

For additional information, refer to the example program in Using Subroutines Example.
