Straightness Correction

Straightness correction compensates for the variation in straightness of table travel. The axis to be corrected must be mounted orthogonally to another table.

The errors are mapped with a precision reference source, such as a laser interferometry system. A calibration table is produced and entered into the 1D axis calibration file.

For Example

You have an X Y axis pair. To correct the straightness of travel of the X axis:

  1. Specify the axis_number argument to be the Y axis.
  2. Specify the master axis as the X axis (the axis you want to correct).

NOTE: Typically, axis indices are specified as zero-based integers. But calibration files specify axis indices as one-based integers.

Note: To see the calibration file data, use the DUMPAXISCAL Console command in Console.
Note: All corrections are bidirectional, meaning that the correction value applied for a given position of the master axis is the same regardless of the sign or value of its velocity.
