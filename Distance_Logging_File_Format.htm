Distance Logging File Format

When you create a distance logging file, you must follow very specific rules to make the format of this file correct.

The Distance Logging File

Create a text file to use as the distance logging file. Make sure that you save all changes that you make to this file. The following example shows a distance logging file that the controller uses to track the distances traveled by each motor on each axis. You can copy this example into your distance logging file so that the controller tracks the distances traveled by the motors on the first four axes. You can also add more axes to this file.

For Example

0: 0
1: 0
2: 0
3: 0

Formatting Rules

Make sure the format of the distance logging file obeys all of the following rules. If you create a distance logging file that does not obey these rules and you enable distance logging, an error occurs when you reset the controller.

The Rules - Each Line

  • Each line must contain the format a:d.
  • a is the axis that you want to track.
  • d is the distance, in counts, that the motor has currently traveled.
  • Each line can contain only one axis and one distance.
  • Each line can contain a maximum of 64 characters.

The Rules - Characters and Spacing

  • You must separate the axis and the distance with one colon.
  • You can put any number of spaces before and after the axis and the distance. Make sure that you do not exceed the 64 character limit.
  • Do not include commas, decimals points, or spaces in the numbers of the axis and the distance.
  • Do not use other characters that are not permitted in these rules.

Important Notes

  • The controller tracks only the axes and the distances that you include in the distance logging file.
  • In the distance logging file, it is not necessary to put the axes and the distances in chronological order. But each time that you reset the controller, the axes and the distances default to chronological order.

What's next?

After you create the distance logging file, refer to Distance Logging Overview for more information about how to enable distance logging.
