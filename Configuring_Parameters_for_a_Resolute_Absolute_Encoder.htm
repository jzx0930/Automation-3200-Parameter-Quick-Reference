Configuring Parameters for a Resolute Absolute Encoder

Hardware: You can use a Resolute encoder only if you are using the Ndrive CP, HLe, or HPe, the XC2, the XC4, the XC4e, or the XR3.

Resolute Encoder Interface with Aerotech Position Feedback

To configure the parameters to use the Resolute encoder interface with Aerotech Position Feedback, use the Axis Parameter Calculator. For more information, refer to Using Calculators to Set Parameters.

  • If the stage does not have hardware limits, you must configure the applicable Software Limits. For more information, refer to Limits Overview.
  • If the motor does not have Hall-effect sensors, the Axis Parameter Calculator configures the ResoluteEncoderSetup Parameter to use absolute position information from the encoder to initialize the motor commutation. The commutation assumes that the zero position of the encoder corresponds to the motor zero. To compensate for misalignment between the encoder and the motor, use the CommutationOffset Parameter.

Resolute Encoder Interface on a Custom Stage

To configure the parameters to use the Resolute encoder interface on a custom stage, you must know this information about the encoder.

  • What is the absolute position resolution (the width of the serial communication word) in bits?

Note: In the ResoluteEncoderSetup Parameter, if you set the Commutation Initialization setting to Use Encoder Absolute Position, then you must determine the commutation offset of the Resolute encoder. Use Encoder Absolute Position is the default setting. Refer to CommutationOffset Parameter for more information.

To configure the parameters

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

  1. Set the value of the ResoluteEncoderResolution Parameter to the absolute position resolution in bits. For example, type 32 for a 32-bit absolute encoder.
  2. Set the value of the ResoluteEncoderUserResolution Parameter to be less than or equal to the value of the ResoluteEncoderResolution Parameter.
  3. If you set the value of the ResoluteEncoderUserResolution Parameter to be less than the value of the ResoluteEncoderResolution Parameter, the controller decreases the effective resolution of the feedback from the Resolute encoder.
  4. Open the Axis Parameter Calculator by selecting Tools → Axis Parameter Calculator. Then do the steps that follow. Refer to Using Calculators to Set Parameters for more information.
    1. For the Position Feedback, select Other Encoder.
    2. Enter the resolution that is applicable to your encoder. Make sure this value is based on the value that you specified for the ResoluteEncoderUserResolution parameter.
      • For a linear absolute encoder, type a value for Resolution (mm/count).
      • For a rotary absolute encoder, type a value for Resolution (counts/rev).
    3. Configure the remaining axis parameters that are applicable to your project.
    4. Click OK to accept the changes and close the calculator.
  5. Change the PositionFeedbackType Parameter to Resolute Absolute Encoder.
  6. To use the absolute position information to initialize the motor commutation, you must determine the correct setting for the CommutationOffset Parameter.
  7. Click the Save Parameter File button to save all of your changes to the parameter file.
