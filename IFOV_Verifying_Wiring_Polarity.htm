IFOV: Verifying the Wiring Polarity

The wiring polarity is correct when the Auxiliary Position Feedback signal on the galvo axes counts in the same direction as the Position Feedback signal on the servo axes. If the Auxiliary Position Feedback signal on the galvo axes counts in the opposite direction, then the wiring polarity is reversed.

Reversed Wiring Polarity

You can correct reversed wiring polarity without making changes to the wire configuration. To do this, set the Auxiliary Feedback Direction setting of the FeedbackSetup Parameter to Inverted on the galvo axes. You must set this parameter in the active parameter file of Configuration Manager.

If the counts are inverted for your IFOV system and the Auxiliary Feedback Direction setting of the FeedbackSetup Parameter is currently set to Inverted, change this setting back to Normal to correct the inverted counts.

Figure: FeedbackSetup Parameter

For more information about this setting, refer to FeedbackSetup Parameter.
