Position Error Fault Axis Fault

The absolute value of the difference between the position command and the position feedback exceeded the threshold specified by the PositionErrorThreshold parameter.

Name

Position Error Fault

AeroBasic Constant

AXISFAULT_PositionError

NOTE: The controller generates an axis fault only if you set the corresponding bit in the FaultMask parameter. When an axis fault occurs, you can configure what the axis does by changing the state of the corresponding bit in the FaultMaskDecel and FaultMaskDisable parameters.

Position error fault

The difference between the commanded position and the actual encoder position has exceeded the PositionErrorThreshold axis parameter. This safety feature is monitored at the servo loop level and disables the axis in the event of an abnormal operating condition.

Possible Causes and Solutions

  • Commanded speed or acceleration/deceleration rates exceed system limitations.
    • Decrease the commanded acceleration rates. Refer to Acceleration and Deceleration Ramping Overview.
    • Decrease the commanded speed. Refer to Feedrate Overview.
  • System tuning needs to be improved.
    • Refer to EasyTune.
    • Autotune (refer to Autotuning an Axis) the axis in question with increased bandwidth. Use Digital Scope to observe the position error during a typical move sequence.
  • The PositionErrorThreshold axis parameter is set too low.
    • Increase the PositionErrorThreshold axis parameter. Use Digital Scope to observe the position error during a typical move sequence.
  • The Motor Supply does not have power.
    • Plot current feedback during a typical move command. If current feedback stays near 0 during the move, examine the motor connections, the motor power supply connections, and the fuses on the amplifier, power supply, or transformer.
  • The axis has an excessive friction or mechanical problem.
    • Disconnect the power to the stage and examine the stage for mechanical friction or interference of parts.
  • On the Nservo, if the OP500 cable is not connected all axes indicate that a Position Error fault has occurred.
    • Connect the OP500 cable.
