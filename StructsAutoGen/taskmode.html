Task Mode

This value provides status information about this task. The following table describes each bit of this value. More status information for the task is reported in Task Status 0, Task Status 1, and Task Status 2.

You can use the TASKSTATUS function to retrieve this value. You can also view the status of these bits in the Task Mode tab of the Status Utility.

Table:     Task Mode Bit Descriptions
