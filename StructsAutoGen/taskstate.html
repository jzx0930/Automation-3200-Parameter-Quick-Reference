Task State

This value represents information about the state of this task.

You can use the TASKSTATUS function to retrieve this value.

For more information about task errors, refer to the Fault and Error Overview.

Table:     Task State Values

Value AeroBasic Constant Name Description

0

TASKSTATE_Unavailable

Unavailable

The Professional Option is not installed. You cannot use this task.

1

TASKSTATE_Inactive

Inactive

This task was disabled by the EnabledTasks parameter.

2

TASKSTATE_Idle

Idle

No program is associated or running.

3

TASKSTATE_ProgramReady

Program Ready

A program is associated but not running.

4

TASKSTATE_ProgramRunning

Program Running

A program is associated and running.

5

TASKSTATE_ProgramFeedheld

Program Feedheld

A program is associated, was run, and was feedheld. This state is active from the moment the motion begins to decelerate due to a feedhold until the moment the motion begins to accelerate back to speed due to a feedhold release.

6

TASKSTATE_ProgramPaused

Program Paused

A program is associated, was run, and was paused.

7

TASKSTATE_ProgramComplete

Program Complete

A program is associated, was run, and completed.(1)

8

TASKSTATE_Error

Error

A task error occurred on this task.

9

TASKSTATE_Queue

Queue

This task is running in queue mode.

(1) A program is completed after it runs the last line of the program, or runs the END PROGRAM command.
