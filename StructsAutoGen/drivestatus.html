Drive Status

This value provides status information from the drive connected to this axis. The following table describes each bit of this value. More status information for the axis is reported in Axis Status.

You can use the AXISSTATUS function to retrieve this value.

Table:     Drive Status Bit Descriptions

Bit Value AeroBasic Constant Name Description

0

0x00000001

DRIVESTATUS_Enabled

Enabled

The axis is enabled.

1

0x00000002

DRIVESTATUS_CwEOTLimit

CW End Of Travel Limit Input Level

This represents the state of the CW end of travel limit input. It is not affected by the active polarity, which is configured by the EndOfTravelLimitSetup parameter.

2

0x00000004

DRIVESTATUS_CcwEOTLimit

CCW End Of Travel Limit Input Level

This represents the state of the CCW end of travel limit input. It is not affected by the active polarity, which is configured by the EndOfTravelLimitSetup parameter.

3

0x00000008

DRIVESTATUS_HomeLimit

Home Limit Input Level

This represents the state of the home limit input. It is not affected by the active polarity, which is configured by the EndOfTravelLimitSetup parameter.

4

0x00000010

DRIVESTATUS_MarkerInput

Marker Input Level

This represents the state of the marker input.

5

0x00000020

DRIVESTATUS_HallAInput

Hall A Input Level

This represents the state of the Hall-effect sensor A input.

6

0x00000040

DRIVESTATUS_HallBInput

Hall B Input Level

This represents the state of the Hall-effect sensor B input.

7

0x00000080

DRIVESTATUS_HallCInput

Hall C Input Level

This represents the state of the Hall-effect sensor C input.

8

0x00000100

DRIVESTATUS_SineEncoderError

Sine Encoder Input Error

An error condition is present on the Sine encoder input of the position feedback device.

9

0x00000200

DRIVESTATUS_CosineEncoderError

Cosine Encoder Input Error

An error condition is present on the Cosine encoder input of the position feedback device.

10

0x00000400

DRIVESTATUS_ESTOPInput

Emergency Stop Input Level

This represents the state of the emergency stop sense input.

11

0x00000800

DRIVESTATUS_BrakeOutput

Brake Output Level

This bit shows the state of the dedicated brake output. The GL4, the Ndrive QL and QLe, the Nservo, the Nstep, and the XL4s support the BRAKE Command in the firmware, but do not have a dedicated brake output.

12

0x00001000

DRIVESTATUS_GalvoPowerCorrection

Galvo Power Correction is configured on this axis.

Galvo Power Correction is configured on this axis.

14

0x00004000

DRIVESTATUS_NoMotorSupply

No Motor Supply

The drive detected that no motor supply voltage is present.

15

0x00008000

DRIVESTATUS_CurrentClamp

Current Clamp

For piezo drives, the controller clamps the motor output to the value of the PiezoVoltageClampLow or the PiezoVoltageClampHigh parameter. For all other drives, the controller clamps the motor output to the value of the MaxCurrentClamp parameter.

16

0x00010000

DRIVESTATUS_MarkerLatch

Marker Latch

The position of the marker is latched.

17

0x00020000

DRIVESTATUS_PowerLimiting

Power Limiting

The motor output is being limited to prevent damage to the amplifier.

18

0x00040000

DRIVESTATUS_PSOHaltLatch

PSO Halt Latch

Internal drive state was latched by the PSOHALT logic.

19

0x00080000

DRIVESTATUS_HighResMode

High Resolution Mode

The amplifier is operating in high resolution current feedback mode.

20

0x00100000

DRIVESTATUS_GalvoCalEnabled

Galvo Calibration Enabled

The specified 2D calibration file contains a galvo calibration table that corrects this axis.

21

0x00200000

DRIVESTATUS_AutofocusActive

Autofocus Active

The axis is operating under control of the AUTOFOCUS loop.

22

0x00400000

DRIVESTATUS_ProgramFlash

Programming Flash

The drive is programming its internal flash memory.

23

0x00800000

DRIVESTATUS_ProgramMXH

Programming MXH

The on-board encoder multiplier is performing a programming operation.

24

0x01000000

DRIVESTATUS_ServoControl

Servo Control

The axis is operating under servo control.

25

0x02000000

DRIVESTATUS_InPosition

In Position

The axis is considered to be in position as configured by the InPositionDistance and InPositionTime parameters.

26

0x04000000

DRIVESTATUS_MoveActive

Move Active

The axis is performing drive generated motion.

27

0x08000000

DRIVESTATUS_AccelPhase

Acceleration Phase

The axis is accelerating.

28

0x10000000

DRIVESTATUS_DecelPhase

Deceleration Phase

The axis is decelerating.

29

0x20000000

DRIVESTATUS_EncoderClipping

Encoder Clipping

The on-board encoder multiplier detected that the input signals may be exceeding the maximum input range, which results in clipping of the encoder signals.

30

0x40000000

DRIVESTATUS_DualLoopActive

Dual Loop Active

The axis is operating in dual-loop mode using two different feedback devices.

31

0x80000000

DRIVESTATUS_InPosition2

In Position 2

The axis is considered to be in position as configured by the InPosition2Distance and InPosition2Time parameters.
