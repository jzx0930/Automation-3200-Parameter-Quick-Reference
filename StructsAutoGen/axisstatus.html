Axis Status

This value provides status information about this axis. The following table describes each bit of this value. More status information for the axis is reported in Drive Status.

You can use the AXISSTATUS function to retrieve this value.

Table:     Axis Status Bit Descriptions

Bit Value AeroBasic Constant Name Description

0

0x00000001

AXISSTATUS_Homed

Homed

The axis is homed.

1

0x00000002

AXISSTATUS_Profiling

Profiling

The axis is performing coordinated (LINEAR, CW, CCW, BEZIER), RAPID, or PVT motion.

2

0x00000004

AXISSTATUS_WaitDone

Wait Done

The controller finished waiting for motion on this axis to complete. The behavior of this bit depends on the selected wait mode. When in WAIT MODE MOVEDONE, this bit will mimic the Move Done bit, but when in WAIT MODE INPOS, this bit will not be active until both the Move Done bit and the In Position bit are both active.

3

0x00000008

AXISSTATUS_CommandValid

Command Valid

Motion on the axis is controlled from the SMC.

4

0x00000010

AXISSTATUS_Homing

Homing

The axis is currently homing.

5

0x00000020

AXISSTATUS_Enabling

Enabling

The axis is currently enabling.

7

0x00000080

AXISSTATUS_JogGenerating

Jog Generating

This bit represents internal status.

8

0x00000100

AXISSTATUS_Jogging

Jogging

The axis is performing asynchronous motion (MOVEINC, MOVEABS, FREERUN).

9

0x00000200

AXISSTATUS_DrivePending

Drive Pending

The SMC sent a command to the drive that will cause the drive to take control of the motion, but the drive has not yet done so.

10

0x00000400

AXISSTATUS_DriveAbortPending

Drive Abort Pending

The SMC sent an abort command to the drive, but the drive has not yet started the abort.

11

0x00000800

AXISSTATUS_TrajectoryFiltering

Trajectory Filtering

Trajectory filtering is enabled for this axis using either the TrajectoryIIRFilter or TrajectoryFIRFilter parameters.

12

0x00001000

AXISSTATUS_IFOVEnabled

IFOV Enabled

Infinite Field of View (IFOV) is enabled for this axis. Enable IFOV by issuing the IFOV ON command. Disable IFOV by issuing the IFOV OFF command.

13

0x00002000

AXISSTATUS_NotVirtual

Not Virtual

A physical drive is associated with this axis. Axes with no drive attached will clear this bit and operate as virtual axes.

14

0x00004000

AXISSTATUS_CalEnabled1D

Calibration Enabled 1D

The specified 1D calibration file contains a calibration table that corrects this axis. The state of this bit is not affected by the CALENABLE or CALDISABLE commands.

15

0x00008000

AXISSTATUS_CalEnabled2D

Calibration Enabled 2D

The specified 2D calibration file contains a calibration table that corrects this axis. The state of this bit is not affected by the CALENABLE or CALDISABLE commands.

16

0x00010000

AXISSTATUS_MasterSlaveControl

Master/Slave Control

The axis is currently performing motion under master/slave control (gearing, camming, or handwheel).

17

0x00020000

AXISSTATUS_JoystickControl

Joystick Control

The axis is currently performing motion under control of the JOYSTICK command.

18

0x00040000

AXISSTATUS_BacklashActive

Backlash Active

Backlash compensation is enabled for this axis using the BacklashDistance parameter or the BACKLASH ON command.

19

0x00080000

AXISSTATUS_GainMappingEnabled

Gain Mapping Enabled

A Gain Mapping table was specified for this axis.

20

0x00100000

AXISSTATUS_Stability0

Stability 0

The axis is considered to be stable as configured by the Stability0Threshold and Stability0Time parameters.

21

0x00200000

AXISSTATUS_MotionBlocked

Motion Blocked

Motion on this axis is being prevented by the BLOCKMOTION command.

22

0x00400000

AXISSTATUS_MoveDone

Move Done

Motion on this axis is done, meaning that the velocity command reached zero.

23

0x00800000

AXISSTATUS_MotionClamped

Motion Clamped

Motion on this axis is being clamped due to a software limit clamp or safe zone. Refer to the SoftwareLimitSetup parameter, and the Safe zone overview.

24

0x01000000

AXISSTATUS_GantryAligned

Gantry Aligned

This axis is part of a gantry pair and the gantry is correctly aligned. This bit will not be active until the gantry axes have been homed.

25

0x02000000

AXISSTATUS_GantryRealigning

Gantry Realigning

The axis is currently performing gantry realignment motion.

26

0x04000000

AXISSTATUS_Stability1

Stability 1

The axis is considered to be stable as configured by the Stability1Threshold and Stability1Time parameters.

27

0x08000000

AXISSTATUS_ThermoCompEnabled

ThermoComp Enabled

The ThermoComp feature is currently turned on.
