Task Status 0

This value provides status information about this task. The following table describes each bit of this value. More status information for the task is reported in Task Status 1, Task Status 2, and Task Mode.

You can use the TASKSTATUS function to retrieve this value. You can also view the status of these bits in the Task Status 0 tab of the Status Utility.

Table:     Task Status 0 Bit Descriptions

Bit Value AeroBasic Constant Name Description

0x00000001

TASKSTATUS0_ProgramAssociated

Program Associated

A program is associated with this task.

Immediate Concurrent

An immediate command is executing concurrently with a program.

3

TASKSTATUS0_ImmediateExecuting

Immediate Executing

An immediate command is executing.

Program Reset

The program is reset.

0x00000200

TASKSTATUS0_PendingAxesStop

Pending Axes Stop

One or more axes are decelerating due to an abort, task stop, task error, feedhold, ongosub, or retrace direction reversal that occurs during a RAPID, LINEAR, CW, CCW, or BEZIER motion.

10

0x00000400

TASKSTATUS0_SoftwareESTOPActive

Software Emergency Stop Active

A Software Emergency Stop is active as configured by the SoftwareESTOPInput parameter.

11

0x00000800

TASKSTATUS0_FeedHoldActive

FeedHold Active

Bit turns on as soon as motion begins to decelerate due to a feedhold. Bit turns off when motion begins to accelerate back to speed due to a feedhold release. The FeedHeldAxesStopped bit of Task Status 1 indicates when deceleration due to a feedhold stops.
