Conditional Program Execution

Use the preprocessor directives that follow for conditional program execution in a program.

Table:     Preprocessor Directive Commands

Preprocessor Directive

Description

#ifdef

Execute program lines if a #define statement has been made.

#ifndef

Execute program lines if a #define statement has not been made.

#else

Execute program lines if a #define statement test is false (#ifdef statement or #ifndef statement).

#undef

Undefine replacement text.

#endif

End of #ifdef statement or #ifndef statement conditional program block.

Example

#define TestMode 1
#ifdef TestMode 
    $global[1] = 2
    #define InTestMode 1
#else
    $global[1] = 7888
    #define InTestMode 0 
#endif 
#undef TestMode
