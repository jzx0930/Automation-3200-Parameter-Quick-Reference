Immediate Commands

An immediate command is a one line program that executes on a task. Simple immediate commands, which include asynchronous motion, setting parameters, setting variables, and setting modes (for example, ABSOLUTE command and INCREMENTAL command), can execute on a task at the same time a program is running on that task. More complicated immediate commands, which include synchronous motion, can only execute if a program is not running.

You can issue immediate commands from the Command box of Aerotech applications which include Motion Composer, CNC Operator Interface, and Digital Scope. You can also issue immediate commands by using the IMMEDIATE EXECUTE AeroBasic command from a program.
