PC Modbus Installation

This topic tells you how to select and install a supported PCI-Express Ethernet card.

Selecting an Ethernet Card

Note: You must use a PCI-Express Network Interface Card. Standard PCI cards do not work on every platform.

Select an PCI-Express Ethernet card with available real-time drivers. This card will be used only for PC Modbus and cannot be used for any other purpose (such as Internet access). Two or more Ethernet cards can be installed in one PC, but one must be dedicated for real-time use.

The A3200 supports these families of network cards.

  • Broadcom Gigabit (bge1g.rsl)
  • Intel 10/100Mbps (ie100m.rsl)
  • Intel Gigabit (ie1g.rsl)
  • RealTek 10/100 Mbps (rtl100m.rsl)
  • RealTek Gigabit (rtl1g.rsl)

For a list of each device that is supported, refer to Compatible Network Interface Cards. Or, you can purchase a supported card from Aerotech.

To Install the Ethernet Card

  1. Make sure that your PC has at least one free PCI-Express slot.
  2. Install the Ethernet card in the PCI-Express slot in your PC.
  3. Use Windows to load the device driver for the card. If necessary, use the driver that is supplied by the manufacturer.
  4. Add INtime support for the NIC.
    1. Open INtime Device Manager. (Start → All Programs → INtime → INtime Configuration. Then double-click INtime Device Manager)
    2. Figure: INtime Configuration Panel — Device Manager
    3. In the Windows devices tree, find your network adapter. Right-click your adapter and select Pass to INtime using MSI.
    4. Click File and then select Save the Configuration. The card will now be listed under the INtime devices tree.
    5. Close INtime Device Manager.

Proceed to PC Modbus Configuration.
