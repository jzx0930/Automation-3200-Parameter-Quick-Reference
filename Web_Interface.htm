Web Interface

You can connect to the A3200 controller from a device that has a recent Web browser. Smartphones and tablets are supported.

To connect to your controller through the Web interface

  1. In Configuration Manager, set the WebServerSetup parameter to Enabled.
  2. Find the name or the IP address of the PC. To find the IP address, open a Windows command prompt and run ipconfig. If you are using Client and Server mode, you must use the name or the IP address of the client PC.
  3. On your device, type the name or the IP address of the PC in the address bar of the browser.

The Web interface looks almost the same on all devices and it is almost the same as the Axis Manager in Motion Composer.

Signal Display

The Servo Off icon shows while the servo loop is off. You can turn off the servo loop only on the Ndrive QL or QLe. Refer to the SERVO ON/OFF command for more information.

The signal display shows the Program Position Feedback signals and the Program Velocity Feedback signals.
Jog Speed

Click the positive or negative jog button to jog the axis in that direction.

Refer to MaxJogSpeed Parameter for more information.

Error Messages

Errors that are caused by immediate commands show at the bottom of the Web interface.
