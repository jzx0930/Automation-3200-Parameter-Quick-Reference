IFOV: Configuring the Software

The information in this topic uses these axis configurations:

  • X and Y are servo axes.
  • A and B are galvo axes.
  • X and A move in the same direction.
  • Y and B move in the same direction.

TO CONFIGURE THE IFOV SOFTWARE

  1. In Configuration Manager, set the IFOVConfigurations Parameter to 1 or 2.
  2. Reset the controller.
  3. Set the task-based speeds and ramp rates. Do the steps that follow:
    1. Use the RAMP RATE Command to set the task-based acceleration and deceleration rates. To get the best performance from your IFOV system, use a large ramp rate or an infinite ramp rate of RAMP RATE 0.
    2. Use the RAMP MODE Command to set the task-based acceleration and deceleration modes to rate-based ramping. Rate-based ramping is the default.
    3. Use the F Command to set the coordinated speed. To calculate the maximum permitted speed, refer to IFOV: Calculating the Maximum Permitted Speed.
  4. Set the axis-based speeds and ramp rates on each of the two galvo axes. Do the steps that follow:

    5. NOTE: You can use IFOV without changing the axis-based speeds and ramp rates of the servo axes. In IFOV mode, when you program motion on the galvo axes the controller automatically limits the speed and acceleration of the servo axes.
    1. Use the RAMP RATE Command with the <Axis> argument to set the ramp rate for each of the two galvo axes. To get the best performance from your IFOV system, use a large ramp rate or an infinite ramp rate of RAMP RATE 0.
    2. Use the RAMP MODE Command with the <Axis> argument to set the axis-based acceleration and deceleration modes to rate-based ramping for each of the two galvo axes.
    3. Use the <AxisName>F<Speed> syntax to set the axis-based speed command for each of the two galvo axes. To calculate the maximum permitted speed, refer to IFOV: Calculating the Maximum Permitted Speed.

      4. For Example

      If one of the galvo axes has the name A, then AF<Speed> sets the axis-based speed on the A axis. If the other galvo axis has the name B, then BF<Speed> sets the axis-based speed on the B axis.

  5. Configure IFOV by issuing the commands that follow.

    6. Table:     IFOV Configuration Commands

    Command Description

    IFOV AXISPAIR Command

    Issue this command two times to map the two galvo axes to the corresponding servo axes.

    IFOV SYNCAXES Command

    Specify more axes to command in IFOV. If you do not want to synchronize more axes, specify NONE for this command.

    IFOV SIZE Command

    Specify, in user units, the field-of-view size of the galvo scan head.

    IFOV TIME Command

    Specify the maximum search time, in milliseconds, that the controller looks ahead. Aerotech recommends that you start with a value of 200 milliseconds for the <SearchTime> argument.

    IFOV TRACKINGSPEED Command

    Specify the maximum speed of the servo axes.

    IFOV TRACKINGACCEL Command

    Specify the maximum acceleration of the servo axes.
