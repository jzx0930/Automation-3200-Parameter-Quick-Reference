IFOV: Calculating the Maximum Permitted Speed

The controller requires you to command motion that does not exceed the maximum permitted speed in IFOV mode. Use the equation that follows to calculate the maximum permitted speed that you can command to the galvo axes.

For Example

If you specify 100.0 mm to the IFOV SIZE Command, you can command a maximum permitted speed of 10,000 mm/s to the galvo axes in your IFOV program. If you command motion that exceeds this speed, a task error occurs.
