Drive Configuration Overview

NOTE: The Drive Configuration dialog applies to HyperWire drives only. If you are using FireWire drives, refer to the FireWire information in the Getting Started section of the Help file.

Use the Drive Configuration dialog to complete the tasks that follow for the HyperWire drives that are connected to your PC:

  • Assign communication channels
  • Load firmware

TO OPEN THE Drive Configuration DIALOG

WARNING: When you open the Drive Configuration dialog, Configuration Manager disables all of the enabled axes and stops all of the AeroBasic programs that are running. It also stops programs that monitor safety-type conditions.

On the Controller menu in Configuration Manager, select Drive Configuration.

Rescanning the HyperWire Network

When the Drive Configuration dialog comes into view, Configuration Manager scans the HyperWire network. Then the dialog shows a list of all the HyperWire drives that are currently connected to your PC. But there might be times when you must manually rescan the network because the dialog does not show a connected drive in the Drive Topology section, Firmware tab, and the Communication Channels tab. For example, this can occur if a drive is not powered on when you initially open the Drive Configuration dialog. To see the connected drive, you must power on the drive. Then manually scan the HyperWire network by clicking the Refresh button at the top of the dialog.

Identifying Drives by Physical Identifiers

You can select a HyperWire drive in the Drive Topology section, the Firmware tab, or the Communication Channels tab of the Drive Configuration dialog. When you select a drive its physical identifier starts to flash, which is usually an LED. When the physical identifier of the drive flashes, you can determine which physically connected drive corresponds to the drive that you select. Before you assign communication channel numbers, you can do this for all of the connected drives to see how the drives in the Drive Configuration dialog match the physically connected drives.
