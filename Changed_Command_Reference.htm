Changed Command Reference

Some commands are different in different software versions. The sections that follow give you the changes in command names and functionality. They also show commands that are no longer available for each software version.

6.01.000 Changed Commands

Command Name

New Functionality

LOOKAHEAD FAST

Functionality removed. The task will always run in a high-speed lookahead mode.

LOOKAHEAD NORMAL

 Functionality removed. The task will always run in a high-speed lookahead mode.
