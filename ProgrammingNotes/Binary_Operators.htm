Logical and Bitwise Operators

Syntax:

 
[<Operand>] <Operator> <Operand>

Example:

 
$Var1 = $Var2 >> 2    ; Shift right by two bits.
  
$Var1 = ~$Var2        ; Do a bitwise NOT.

Logical operators evaluate an operation on one or two operands. The result of the operation is 1 or 0.

Bitwise operators evaluate an operation on one or two 32-bit integer operands. Before the operation is evaluated, all operands are truncated to 32-bit unsigned integers. The result of the operation is a 32-bit unsigned integer.

(1) The BNAND and BNOR operators are a mixture of logical and bitwise operators. For example, $global[0] BNAND $global[1] is the same as NOT($global[0] BAND $global[1]). $global[0] BNOR $global[1] is the same as NOT($global[0] BOR $global[1]).

(2) Bits that are shifted off of on of the two ends of a 32-bit integer value are discarded. For example, (1>>3) is 0.

These examples show correct syntax that uses logical and bitwise operators and summarize their results in binary. 0h or 0x prefixes a hexadecimal constant. The comment that follows the code shows the binary description of the operation.

Examples

$var1 = 12                  ' 1100 in binary or 0hC in
                            ' hexadecimal.

' Logical operators
$var2 = $var1 OR 0hA        ' 1
$var2 = $var1 AND 0hA       ' 1
$var2 = !$var1              ' 0
$var2 = $var1 BNAND 0h1A    ' 0 (NOT($var1 BAND 0h1A))
$var2 = $var1 BNOR 0h1A     ' 0 (NOT($var1 BOR 0h1A))

' Bitwise operators
$var2 = $var1 BOR 0h1A      ' 30         (0h1E)
$var2 = $var1 BAND 0h1A     ' 8          (0h8)
$var2 = BNOT $var1          ' 4294967283 (0hFFFFFFF3)
$var2 = $var1 BXOR 0h1A     ' 22         (0h16)
$var2 = $var1 << 2          ' 48         (oh30)
$var2 = $var1 >> 3          ' 1          (0h1)
