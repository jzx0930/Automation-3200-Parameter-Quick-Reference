Galvo Command Reference

Hardware: These commands apply only when used with the GL4 or the Nmark CLS, GCL, or SSaM.

Table:     Laser Configuration Commands

Command

Description

GALVO LASERMODE

Specifies the mode in which the laser output signals operate.

GALVO LASEROUTPUTPERIOD

Specifies the period, in microseconds, of the O1 and O2 signals.

GALVO LASER1PULSEWIDTH

Specifies the pulse width, in microseconds, of the O1 signal.

GALVO LASER2PULSEWIDTH

Specifies the pulse width, in microseconds, of the O2 signal.

GALVO STANDBYPERIOD

Specifies the period, in microseconds, of the standby signals.

GALVO STANDBYPULSEWIDTH

Specifies the pulse width, in microseconds, of the standby signals.

GALVO SUPPRESSIONPULSEWIDTH

Specifies the pulse width, in microseconds, of the suppression signal.

Table:     Laser Delay Commands

Command

Description

GALVO LASEROFFDELAY

Specifies when the GL4 or the Nmark CLS, GCL, or SSaM stops firing the laser relative to when you command the laser to power off.

GALVO LASERONDELAY

Specifies when the GL4 or the Nmark CLS, GCL, or SSaM fires the laser relative to when you command the laser to power on.

Table:     Laser Motion and I/O Commands

Command

Description

GALVO ENCODERSCALEFACTOR

Enables "marking on the fly" functionality on the specified axis of a GL4 or an Nmark CLS, GCL, or SSaM.

GALVO LASEROVERRIDE AUTO

Specifies that the laser on a galvo axis is automatically controlled.

GALVO LASEROVERRIDE OFF

Commands the laser of the GL4 or the Nmark CLS, GCL, or SSaM to power off.

GALVO LASEROVERRIDE ON

Commands the laser of the GL4 or the Nmark CLS, GCL, or SSaM to power on.
