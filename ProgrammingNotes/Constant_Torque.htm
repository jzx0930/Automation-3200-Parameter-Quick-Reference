Constant Torque

Supplies a sample program that contains a subroutine to produce a constant torque.

Key Concepts

Program Automation

Download-only programs

FARCALL Command

Objective

Configure the controller to use predefined commands from an auto-include file, which call subroutines in another program (FARCALL) to apply a predefined quantity of torque.

Table:     Program Supplied in the \A3200\Samples\AeroBasic Folder

Program Description

ConstantTorque.Pgm

Main subroutine called that applies the defined torque

Applying Constant Torque

In the example that follows, the Z axis is applying the constant torque.

  1. Edit the following #defines in the \A3200\Samples\AeroBasic\ConstantTorque.Pgm as necessary.

    2. Example

    #define FiftyLbs 1.0
' Amperes of current equal to 50 lbs. The properties of the motor have an effect on the value of the constant that follows. 
' To calculate this value for a linear motor, you must know the Force Constant. 
' To calculate this value for a rotary motor, you must know the Torque Constant. 
' For an Aerotech motor, you can find this information in the Specifications section for the motor at www.aerotech.com.
#define FiftyLbs 1.0 
' Velocity at which to move when overdriving the axis (in IPM).
#define ZPos 6.2 
' Set down position for Z to overdrive the axis to apply the force (+.1").
  2. Add \A3200\Samples\AeroBasic\ConstantTorque.Pgm to program automation on the Setup page as a download only program.

    3. Examples

    ' <- Z1 = Apply constant torque.
FARCALL "ConstantTorque.Pgm" "TrqCmd" Z1
' <- Z0 = Remove constant torque.
FARCALL "ConstantTorque.Pgm" "TrqCmd" Z0

    Or, you can add an auto-include file to program automation that defines these commands. 

    #define TORQUEON FARCALL "ConstantTorque.Pgm" "TrqCmd" Z1
' Allow the lowercase command.
#define torqueon TORQUEON
#define TORQUEON FARCALL "ConstantTorque.Pgm" "TrqCmd" Z0
' Allow the lowercase command.
#define torqueoff TORQUEOFF

    Then, you can use these commands in your program.

    ' Apply constant torque.
TORQUEON
' Remove constant torque.
TORQUEOFF
