Distance Logging Overview

Distance Logging lets you track the total distance, in counts, that a motor has traveled. Before you can enable this feature, you must create a distance logging file that uses the Distance Logging File Format. After you create the distance logging file, you must add it to A3200.mconfig, which is the machine configuration file. The following table shows the location of the machine configuration file for supported operating systems.

Table:     Machine Configuration File Location

Operating System

File Location

Windows 7

C:\ProgramData\Aerotech\A3200\A3200.mconfig

To Enable Distance Logging

NOTE: You cannot use Distance Logging with virtual axes. You must connect to a controller.
  1. Shut down the A3200. Click Start → All Programs → Aerotech → Automation 3200 → Shutdown A3200. A black Command Prompt window comes into view momentarily.
  2. Open A3200.mconfig.
  3. Immediately before the line </MachineConfiguration>, add the line <DistanceLoggingFile>$Path</DistanceLoggingFile>.
  4. Replace $Path with the full path to the distance logging file that you created. Refer to the following example.

  5. For Example

    <DistanceLoggingFile>C:\Users\Public\Documents\mileagecounter.txt</DistanceLoggingFile>
</MachineConfiguration>
  6. Save your changes to A3200.mconfig.
  7. Reset the controller.

After you enable Distance Logging, make sure the axis is enabled before you issue a command so the controller can track the distance that the motor travels. Each time you shut down or reset the A3200, the controller writes the distance, in counts, to the distance logging file and saves the file.

To see the total distance that the motor traveled, you can look at the distance logging file or you can issue the AXISSTATUS function.

  • Distance Logging File The distance logging file does not update continuously. It shows the total distance only after you reset the controller. The measurement in the distance logging file is the most accurate and is recommended if you are tracking large distances, such at 1016 counts.
  • AXISSTATUS Function Use the AXISSTATUS Function to see the total distance without resetting the controller.

The maximum distance that the controller can track is 18 Í 1018counts. The controller does not define distances that exceed this number.

To Disable Distance Logging

  1. Shut down the A3200. Click Start → All Programs → Aerotech → Automation 3200 → Shutdown A3200. A black Command Prompt window comes into view momentarily.
  2. Open A3200.mconfig.
  3. Delete the line <DistanceLoggingFile>$Path</DistanceLoggingFile>.
  4. Save your changes to A3200.mconfig.
  5. Reset the controller.
