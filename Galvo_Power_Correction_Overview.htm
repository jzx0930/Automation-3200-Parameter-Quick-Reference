Galvo Power Correction Overview

Hardware: Galvo Power Correction is available only if you are using the GL4, the Nmark CLS with Rev. A or later, or the Nmark GCL.

Use Galvo Power Correction to correct the laser power output on the galvo axes of a GL4, an Nmark CLS with Rev. A or later, or an Nmark GCL. Based on the position of the two galvo axes, you can scale the voltage of a specified analog output on the drive using <CorrectionMultiplier> arguments. The drive computes the actual output voltage by multiplying the commanded analog output voltage ($AO Function) by the value of the <CorrectionMultiplier> argument that is applicable to the row and column input axes position.

When an analog output is under Galvo Power Correction control, the value of the output voltage that you read is not the same as the actual analog output voltage. To get the actual value of the analog output voltage, use the DRIVEINFO_GalvoLaserPowerCorrectionOutput data item of the DRIVEINFO Function.

Do the steps that follow to configure your system to use a galvo power correction file.

  1. Open Configuration Manager.
  2. In the Network Explorer pane, expand the A3200 node and select the Calibration folder.
  3. In the Viewer pane, select Galvo Power Correction File.
  4. In the Editor pane, click the Browse button, and select the galvo power correction file. Galvo power correction files from Aerotech have a .cal extension. You can also type the full path and name of the galvo power correction file into the text box.
  5. After the file name is in the text box, click the Set button. When you are prompted, reset the controller.
