Axis Type Overview

Use the AxisType Parameter to configure an axis as either dominant or dependent. Axis type is relevant only for coordinated motion (LINEAR, CW, and CCW motion). When doing coordinated motion, you can command dominant and dependent axes at the same time, but you directly control the ramping and vector speed of only the dominant axes. In this condition, the controller moves any dependent axes that are in the coordinated motion at a speed so that they reach their target position at the same time as the dominant axes.

Warning: The motion of the dominant axes controls the speed and ramping of the dependent axes. This can cause the speed and accelerations of the dependent axes to exceed their programmed values. Set the MaxSpeedClamp parameter for each dependent axis to clamp the maximum speed of each axis. Use the DependentCoordinatedAccelLimit task parameter to limit the maximum acceleration of all dependent axes.

Use the F and E Commands to specify the vector speed for coordinated motion. The F command specifies the vector speed for dominant axes and is used if one or more dominant axes are in the coordinated motion. The E command specifies the vector speed for dependent axes and is used only when dominant axes are not in the coordinated motion.

The parameters that follow apply only to dominant axes:

  • DefaultCoordinatedSpeed
  • DefaultCoordinatedRampRate
  • DefaultCoordinatedRampTime
  • SecondaryUnitsName
  • SecondaryUnitsScaleFactor
  • CoordinatedAccelLimit
  • CoordinatedCircularAccelLimit

The features that follow apply only to dominant axes:

  • CW and CCW commands
  • Cutter Radius Compensation
  • Cutter Offset Compensation

The parameters that follow apply only to dependent axes:

  • DefaultDependentCoordinatedSpeed
  • DefaultDependentCoordinatedRampRate
  • DependentSpeedScaleFactor
  • DependentCoordinatedAccelLimit

The features that follow apply only to dependent axes. You must have the CNC Option to use these features:

  • Spindles
  • Normalcy Motion
Example Program

In the example that follows, X is configured as a dominant axis and is programmed in millimeters. A is configured as a dependent axis and is programmed in degrees.

' Set the time base in seconds.
SECONDS
' Set the dominant speed to 10 mm/s.
F10
' Set the dependent speed to 50 degrees/s. 
E50
' X uses the dominant speed and moves at 10 mm/s.
LINEAR X10
' Because no dominant axes are part of the move, A uses the dependent speed and moves at 50 degrees/s.
LINEAR A200
' X moves at 10 mm/s. But, because A is dependent on X, A does not move at its specified speed of 50 degrees/s.
' Because the dominant speed is 10 mm/s, X completes its motion in 2 seconds (20 mm divided by 10 mm/s). 
' Because A is a dependent axis moving in coordinated motion with a dominant axis, A completes its move in the
' same quantity of time. To do this, A moves at a speed of 100 degrees/s (200 degrees divided by 2 seconds).
LINEAR X20 A200
