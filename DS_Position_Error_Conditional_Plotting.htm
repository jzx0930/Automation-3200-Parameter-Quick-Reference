Position Error Conditional Plotting

You can use Position Error Conditional Plot Formatting to make an analysis of the quantity of error in your data based on a defined threshold. With this information, you can determine if the quantity of error that is caused is acceptable, or if you need to adjust the source of the data in order to cause less error.

To access Position Error Conditional Plot Formatting in Digital Scope

  1. In Configuration Manager, select Tools → Digital Scope. The Digital Scope application comes into view.
  2. Figure: Digital Scope

  3. In Digital Scope, select Configuration → Conditional Plotting. The Conditional Plot Formatting dialog comes into view.
    4. Figure: Digital Scope Conditional Plot Formatting

1D Plots

  • The conditional plot formatting has an effect only on the position-error signals.
  • The colors of position-error signals change to the Conditional Format Color when the values of the position-error signals exceed the In Position Threshold.
  • Figure: Digital Scope Conditional Format Color

  • The section of the position-error signal that is outside the In Position Threshold is in the Conditional Format Color.
  • The section of the plot that is within the In Position Threshold is in the original signal color.

1D Plot Formatting

In the 1D Error Formatting Options section of the Conditional Plot Formatting dialog, you can select either the Step and Settle or Contouring option for 1D plot formatting.

  • Step and Settle formatting applies only when the value of the Velocity Command signal is zero. This formatting has an effect only when the stage stops being in motion. For Digital Scope to apply Step and Settle formatting correctly, you must collect the Velocity Command signal. For information about how to collect signals in a 1D plot, refer to the Collecting Signals and Configuring a Plot section of Data Collection in Digital Scope.
  • Contouring mode formatting applies at all times. In this mode, the value of the Velocity Command signal does not have an effect on conditional plot formatting. If this signal is the original color, then the error is within the In Position threshold. If this signal is the Conditional Format Color, then the error exceeds the In Position Threshold.

2D Plots

The In Position Threshold is the maximum allowable vector difference between the Position Command and Position Feedback signals on the axes that are being plotted.

  • All vector differences between these signals that exceed the In Position Threshold are in the Conditional Format Color.
  • All vector differences between these signals that stay within the In Position Threshold are in the original signal colors.
