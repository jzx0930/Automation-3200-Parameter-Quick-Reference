Working with the TANGO Device Server

Before you can use a TANGO Device Server, you must configure the A3200 to use an Ethernet connection. For more information about TANGO Device Server, refer to the documentation that is supplied with TANGO.

To configure Ethernet

  1. Open Configuration Manager.
  2. In the CommandPort parameter, specify the port that you want to use.
  3. The TANGO Device Server connects to the port that you specify.
  4. Configure the CommandSetup parameter.
    1. Select Enabled.
    2. Select Always Send EOS.
