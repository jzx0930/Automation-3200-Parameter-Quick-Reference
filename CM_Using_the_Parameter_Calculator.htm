Using the Axis Parameter Calculator

The Axis Parameter Calculator is the primary calculator to use to configure an axis. You can use this calculator to configure axis types that include Ballscrew Stage, Linear Stage, Rotary Stage, Voice Coil Stage, and No Stage (Motor Only). For each axis, you can specify information about the Mechanics, Drive, Units, Feedback, Servo Loop, and Current Loop. The sections that follow tell you how to make and apply changes to parameters in the Axis Parameter Calculator.

The Axis Parameter Calculator Interface

Figure: Configuration Manager — Axis Parameter Calculator

Label Description

Shows the sections of the calculator for which you can specify information. These sections include Mechanics, Drive, Units, Feedback, Servo Loop, and Current Loop.

Select the parameters that you want to calculate. Do one of these procedures:

  • To calculate all of the parameters in a category, select the Calculate box that is adjacent to the parameter category that contains the parameters that you want to calculate.
  • To calculate some of the parameters in a category, select the Calculate box that is adjacent to the parameter category that contains the parameters that you want to calculate. Then, clear the boxes that are adjacent to the parameters that you do not want to calculate.

Click OK to accept the changes and close the calculator.

Click Cancel to discard the changes and close the calculator.

Click Apply to save the changes and keep the calculator open.

The Details View shows you all of the axis data that the Calculator uses. You can copy the information from the Details View to your clipboard.

Units section

Units

 Select the units. The list contains the most commonly used units. If the units that you want are not in the list, select Other.

Inverted

If positive encoder direction is the inverse of positive motor direction, select the Inverted box.

Type

 Select Linear or Rotary to specify the units. This is necessary only if you use custom units.

Custom Units Name

 Enter the name of the units (for example, milli rad). This is necessary only if you specify custom units.

Scale Factor

Enter the scale factor to convert to the correct units. For linear units, the units are in mm/unit. For rotary units, the units are in rev/unit. This is necessary only if you specify custom units.

Position Feedback

Feedback section
  • If you use Aerotech Position Feedback, or if the custom stage or custom motor you selected includes feedback information, select the Position Feedback option that is applicable for your mechanics. If this Position Feedback supports multiplication, then enter a value between 1 and 262144 in the Multiplication Factor box to set the multiplication factor. Some mechanics, such as galvo scanners, use a fixed multiplication factor that you cannot change.
  • If you do not use Aerotech feedback, or if the custom stage or custom motor you selected does not include feedback information, select Other Encoder in the Position Feedback box.
    • If you use a linear analog sine wave (or 1 Vpp) encoder, select the MX box and enter the resolution of the encoder in mm/cycle in the Resolution (mm/cycle) box. You can find this value in the encoder specifications as the signal period or pitch of the encoder. To set the multiplication factor, enter a value between 1 and 262144 in the Multiplication Factor box.
    • If you use a linear square wave (or TTL/RS422) encoder, enter the resolution of the encoder in mm/count in the Resolution (mm/count) box. You can find this value in the encoder specifications. In the value that you enter, include the quadrature done on the encoder signal by the controller.
    • If you use a rotary analog sine wave (or 1 Vpp) encoder, select the MX box and enter the resolution of the encoder in counts/rev in the Resolution (counts/rev) box. You can find this value in the encoder specifications as the number of counts, pulses, or lines that the encoder supplies during one revolution of the motor. To set the multiplication factor, enter a value between 1 and 262144 in the Multiplication Factor box.
    • If you use a rotary square wave (or TTL/RS422) encoder, enter the resolution of the encoder in counts/rev in the Resolution (counts/rev) box. You can find this value in the encoder specifications as the number of counts, pulses, or lines that the encoder supplies during one revolution of the motor. In the value that you enter, include the quadrature done on the encoder signal by the controller.

Servo Update Rate (Hz)

The calculator applies a servo rate based on the drive type that you select. You cannot change this value and this value does not show in the calculator.
To Use Other Mechanics from a Calculator Version Before 5.04

In versions of the calculator before 5.04.000, if you used stages and motors that were not manufactured by Aerotech, you could select Other Stage or Other Motor in the calculator. In 5.04.000 and later versions, if you have a parameter file that is configured to use Other Stage or Other Motor, you can edit the properties of your stage and motor in the Mechanics section of the calculator. Or, use the Catalog Manager to import your stage and motor into the custom mechanics catalog. After you import your stage and motor, you can select them in the calculator.

To Apply Custom Changes from the Mechanics Section of the Catalog Manager

If you use the Catalog Manager to make changes to a custom item, use the Axis Parameter Calculator to apply the changes to parameter files that are configured to use that item. When you open the calculator, it identifies axes in the selected parameter file that are configured to use items from your custom mechanics catalog. The calculator shows a warning if the properties of a catalog item changed since the last time you applied changes to axes that use the item. The calculator uses the properties of the stage or motor in your catalog to calculate parameters, and ignores the properties of the stage or motor that are stored in the parameter file.

To Apply Changes from the Catalog

  1. Select the axis that you configured to use the stage or motor that is in the catalog. A warning shows that the results include changes to the properties of the stage or motor.
  2. In the Results pane, look at the changes to the parameters. You will not see changes if only properties such as the name of the item changed.
  3. Click OK or Apply to update the parameter file.

For more information about the Catalog Manager, refer to Catalog Manager Overview.
