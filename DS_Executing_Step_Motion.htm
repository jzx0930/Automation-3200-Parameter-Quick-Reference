Executing Step Motion

Digital Scope supplies functionality for executing a motion command and collecting data simultaneously. This functionality is accomplished using the Step Forward, Step Reverse, and Auto Step features supplied in the Action menu in the Scope tab of Digital Scope.

Figure: Action Menu

The Step Forward and Step Reverse functions execute a single motion command and trigger data collection, while the Auto Step function executes the Step Forward command followed by the Step Reverse command repeatedly until stopped. To enter a forward or reverse step motion command, click on the arrow shown on the right of the Step Forward or Step Reverse function to display the Forward (or Reverse) Motion Command box.

Figure: Forward Motion Command Box

If the command that is entered is invalid, the Forward Motion Command box will display an error icon.

Figure: Invalid Forward Motion Command Box

After you configure the forward or reverse command, when you click the Step Forward or Step Reverse button on the menu, will execute the motion command and start collecting data. Prior to executing the forward or reverse command, you may want to configure data collection, as described in Data Collection.
