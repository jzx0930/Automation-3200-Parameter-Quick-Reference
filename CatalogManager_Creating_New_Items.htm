Creating New Items in the Catalog Manager

You can create a new item that has all new configurations or you can create a new item that is based on a previously configured item. Refer to the procedures that follow.

To Create a New Item that has all new configurations

  1. Open Configuration Manager.
  2. On the Tools menu, select Catalog Manager. Or click the Catalog Manager button on the toolbar. The Catalog Manager dialog comes into view.
  3. On the Edit menu of the Catalog Manager dialog, select Add Item. Or click the Add Item button on the toolbar. The New Item section comes into view.
  4. In the New Item section, enter the information that follows. If it is necessary to cancel your changes, click the Discard button .
    5. Figure: Adding New Item That Has New Configurations

    1. Click the Item Type arrow and select Stage or Motor.
    2. Click the Stage Type or Motor Type arrow and select the type of stage or motor that you want to use.
    3. Type a unique name for your stage or motor.
    4. Enter information for all the required properties of your stage or motor. If you do not have this information, contact the manufacturer of the stage or motor to get these specifications.
    5. Optional. To specify a feedback device for your stage or motor, select the Include Feedback box. Then enter information for the required properties. If you use the new item in the Axis Parameter Calculator, the specified feedback device becomes available in Feedback section of the calculator.

    6. NOTE: If you select the MX box, it tells the Catalog Manager that the specified feedback device requires you to supply a multiplication factor when you select the device in the Axis Parameter Calculator. Thus when you enter a value for the Resolution in the Catalog Manager, do not include multiplication.
  5. To add the new item to your catalog, click the Add button . The Catalog section of the dialog shows an asterisk adjacent to this item because you must save it to the catalog file.
  6. Click Apply to save all of your changes to the catalog file.

To Create a New Item Based on previous configurations

  1. Open Configuration Manager.
  2. On the Tools menu, select Catalog Manager. Or click the Catalog Manager button on the toolbar. The Catalog Manager dialog comes into view.
  3. On the Edit menu of the Catalog Manager dialog, select Add Item. Or click the Add Item button on the toolbar. The New Item section comes into view.
  4. In the New Item section, enter the information that follows. If it is necessary to cancel your changes, click the Discard button .
    5. Figure: Adding New Item That Has Previous Configurations

    1. Click the Item Type arrow and select Stage or Motor.
    2. Click the Stage Type or Motor Type arrow and select the type of stage or motor that you want to use.
    3. Click the Apply Template (Optional) arrow to select a previously configured item to use as the source of properties for your new item. The list of templates includes custom items from your catalog and standard Aerotech items that have the stage or motor type that you selected.
    4. Click the Apply Template (Optional) button to copy the properties of the template to the new item.
    5. Type a unique name for your stage or motor.
    6. Make the necessary changes to the properties of the stage or motor.
  5. To add the new item to your catalog, click the Add button . The Catalog section of the dialog shows an asterisk adjacent to this item because you must save it to the catalog file.
  6. Click Apply to save all of your changes to the catalog file.
