Configuration Manager Overview

Use Configuration Manager to change parameter files and set the active parameter file, as well as connection settings and program automation. You can open Configuration Manager from the Tools menu of one of the A3200 applications. Or, click Start → All Programs → Aerotech → Automation 3200 and select A3200 Configuration Manager.

Note: After you change the value of a parameter in the active parameter file, you must reset the controller for the change to have an effect.

Interface

Configuration Manager has four window panes or sections. The Main Menu and Toolbars give you access to almost all of the features that are available in the application.

You can use the Alt key to access menus in the menu toolbar of the application. Press Alt. Then press the letter that is underlined in the name of the menu that you want. In the menu that comes into view, press the letter that is underlined in the name of the option that you want.

Figure: Configuration Manager: Interface

Label Description

Menus, Menu Icons, and Parameter Search.

Network Explorer: Configure most of the A3200 environment. For example, you can set up the controller, configure Program Automation, and organize files.

Viewer: Shows contextual information about the current selection in the Network Explorer. For example, when you select a parameter category, all the parameters in that category show in the Viewer.

Editor: Modify the values of different items such as the name of a controller or the value of a parameter.

Help Browser: Shows the help topic for the subject that you select in the viewer.
