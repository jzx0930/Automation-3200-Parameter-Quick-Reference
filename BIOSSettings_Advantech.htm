Advantech PC Specifications and BIOS Settings

This topic supplies specifications for Advantech PCs that Aerotech approves for A3200 installation. If you are using one of the Aerotech-approved Advantech PCs, use the BIOS settings in this topic. Do not use the BIOS settings supplied for PCs that are not approved by Aerotech.

Form Factors

Motherboard Manufacturer Product Number

AIMB-785 with Intel i7 or i5 processor

SYS-4U4000-9A52

The motherboard is available in an Advantech ACP-4000 4U rackmount chassis and other Advantech form factors.

Advantech PC Configuration Tested at Aerotech

  • Advantech AIMB-785 (ATX size)
  • BIOS V1.12 10/5/2016, 10/7/2016
  • Intel Core i5-6500 3.2 GHz and Intel Core i7-6700 3.4 GHz.
  • 8 GB RAM
  • Windows 10, 64-bit
