PC Modbus Configuration

After you have correctly configured the network interface card under INtime control, you must configure the INtime TCP/IP stack to use the correct driver, IP address, and subnet mask. Configuration Manager simplifies the process of this configuration.

To Configure the Ethernet Stack

  1. Find which Network Interface Card to use.
    1. Open INtime Device Manager. (Start → All Programs → INtime → INtime Configuration. Then double-click INtime Device Manager).
    2. In the INtime devices tree, find your NIC. Right-click your NIC and click Properties.
    3. On the Details tab, in the Property list, click Hardware Ids.
    4. The Value box shows the Hardware IDs for your NIC. Record the Vendor ID, which starts with VEN_, and the Device ID, which starts with DEV_.
    5. Figure: Locating the Vendor and Device IDs
    6. Open the Compatible Network Interface Cards topic. Find the Vendor ID and Device ID of your device. The table heading is the device driver to use with your card.
      Note: If you cannot find your card in the list, it might not be supported. You can select the device that is most like your device, but it might not work.
      Figure: Finding your NIC Device Driver
  2. Open Configuration Manager (Start → All Programs → Aerotech → Automation 3200 → A3200 Configuration Manager).
  3. On the Controller menu, click Connection Settings.
  4. In the Connection Settings dialog box, click the Fieldbus Tab.
  5. In the PC Modbus box, select the NIC Driver that you found in Step 1.
  6. Figure: Selecting the Network Driver

  7. In the IP Address box, enter the IP address to statically assign to the network interface card.
  8. In the Subnet Mask box, enter the appropriate subnet mask. For more information about selecting a subnet mask, refer to PC Modbus Overview.
  9. Click OK to close Connection Settings. You get a message that you must restart your PC.
  10. Close Configuration Manager.
  11. Restart your PC.
