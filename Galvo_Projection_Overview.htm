Galvo Projection Overview

The controller executes the galvo projection function as a non-generic 3x3 matrix transformation. This matrix transformation is non-generic because of how the controller scales the calculated X/Y output by the calculated Z value.

When the controller executes this function, it applies the output of the transformation directly to the scanner positions. Make sure that you enable the scanner at a commanded position that minimizes mirror motion.

where

XCommand and YCommand input positions are in encoder counts.

and

AVG X and AVG Y output positions are in encoder counts.

Based on how you calculate the coefficients, you can use this matrix to apply scaling, rotation, or offsets. You can also apply scaling, rotation, and offsets in the same operation or do projective transformations.
