Compatible Network Interface Cards

To use PC Modbus on the A3200, you must have a supported PCI-Express Network Interface Card.

The following NIC devices are supported by INtime to use with PC Modbus. Use this information to select the correct driver for your NIC.

Note: You must use a PCI-Express Network Interface Card. Standard PCI cards do not work on every platform.
