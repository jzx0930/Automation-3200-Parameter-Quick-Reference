System Requirements

PCs Approved for A3200 Installation

Aerotech supplies free support for A3200 installation if you purchase an Aerotech industrial PC or an approved PC from the supplier list that follows. These PCs have the recommended system requirements. If you have one of the approved PCs, refer to Optimizing A3200 Performance to correctly configure your system.

PCs Approved by Aerotech

  • Advantech (AIMB-785 Motherboard)
  • BSI (RMS575-AERO)
  • Polywell (Aero1650, Aero1763, Aero1960, or Aero2030)

Configuration of PCs Not Approved for A3200 Installation

Aerotech does not guarantee the real-time capabilities of a PC that is not approved for A3200 installation because BIOS settings and chip sets are different for each motherboard manufacturer. If you have a PC that is not approved by Aerotech, use the recommended configurations that follow. The minimum system requirements of the A3200 SMC are included. The minimum requirements are not adequate for all user applications. Use the Recommended PC configuration or discuss your application with an Aerotech Global Technical Support representative.

Table:     PC Configuration

Processor

Recommended: Intel Core 2 Quad, Intel Core i5/i7, or Xeon processor with four or more cores (1) (2) (3)

Minimum: One multi-core processor (Intel Core 2 Duo, Intel Core i3, or newer) (1) (2)

Not supported: AMD, Intel Celeron, Intel Pentium 4, Intel Penitum D, Itanium, or notebook, laptop, and mobile processors (4)

RAM

Recommended:

  • 4 GB for 32-bit Operating System
  • 8 GB for 64-bit Operating System

Minimum: 4 GB for 32-bit and 64-bit Operating System

Operating System

32-bit or 64-bit version of the following:

  • Microsoft Windows 10 (Enterprise LTSB, Enterprise, or Professional)
  • To prevent problems, you must configure Windows 10 updates to work with your A3200 and INtime system. Refer to the User Guide to Windows 10 Updates on www.aerotech.com.
  • Microsoft Windows 8.1 (any Service Pack)
  • Microsoft Windows 7 (Service Pack 1 (SP1))
  • Before you install the A3200 software, run Windows Update a minimum of one time. Make sure that all of the updates complete.

Other operating systems and Windows 10 Home are not supported.

Graphics Display
  • Standard display adapter (5)
  • 1024 × 768 pixels or higher resolution

Hard Disk

500 MB available disk space

IE

Internet Explorer 6 or later (6)

Peripherals

Keyboard, mouse, and USB port

(1) A PC that has two physical processors is not supported. Use a PC with one processor and more than one physical core.

(2) Intel Xeon processors based on the Intel Core 2 or Intel Core i3/i5/i7 architecture are supported. Intel Xeon processors based on older architectures are not supported.

(3) A PC that has INtime configured is always limited to 32 total cores, even when the PC has more than 32 physical cores.

(4) Laptop computers are not supported. They can be used for offline debugging and simulation only. PCMCIA FireWire cards are not supported.

(5) Aerotech recommends that you do not use high-end display adapters because they can cause interference with the A3200. Use a medium-performance onboard integrated graphics adapter or a medium-performance PCI express dedicated graphics card. If your graphics card causes problems with A3200 performance, try a different graphics adapter. For more information, refer to the Change the Display Configuration section of Optimizing A3200 Performance.

(6) Internet Explorer is not included as part of the A3200 installation. If it is not installed on your computer, you must manually install it before you continue with the A3200 installation.

General Recommendations

To configure your PC to run A3200 and INtime, obey the instructions in Optimizing A3200 Performance. Third-party hardware or software devices on your PC can also cause problems with the real-time performance of A3200.

Before you purchase or select a PC to use with A3200, read Optimizing A3200 Performance for information you must know about different hardware and software.
