Loading Drive Firmware

NOTE: The Drive Configuration dialog applies to HyperWire drives only. If you are using FireWire drives and want to load firmware onto the drives, refer to Firmware Loader Overview .

Aerotech strongly discourages you from using a version of the A3200 software that does not match the firmware. Each time you install a new version of the software, use the Drive Configuration dialog to load the same version of firmware onto each HyperWire drive that is connected to your PC.

After Configuration Manager loads firmware onto the drives, it might be necessary for you to restart your PC. If you must restart your PC after the firmware loads, Configuration Manager asks you if you want to continue. If you select Yes, the application completes loading the firmware. Then you must restart your PC to make sure your motion system operates correctly.

TO LOAD FIRMWARE ONTO THE HyperWire DRIVES

  1. Open Configuration Manager.
  2. On the Controller menu, select Drive Configuration. The Drive Configuration dialog connects to the HyperWire network and identifies all of the connected drives.

    		3.

    WARNING: When you open the Drive Configuration dialog, Configuration Manager disables all of the enabled axes and stops all of the AeroBasic programs that are running. It also stops programs that monitor safety-type conditions.
  3. In the Drive Configuration dialog, select the Firmware tab.
  4. Optional. Clear the drive selection boxes for drives to which you do not want to load firmware. By default, all drives in the list are selected. But Configuration Manager loads firmware only onto the drives that stay selected.
  5. Select the firmware file that you want to load onto the drives. Do one of the steps that follow:

    		6.

    WARNING: Aerotech recommends that you use the default firmware file. If you do not use the default file and you select an incorrect file, the firmware and software become mismatched and serious problems can occur.
    • The default firmware file is already selected. If you want to use this file, continue to the next step.
    • If you want to specify a different firmware file, click the Browse button in the LOD File section of the Drive Configuration dialog. Then select the file that you want to use.
  6. Click the Program button to start loading firmware onto the drives and the installed HyperWire card.

    		7.

    Danger: Do not disconnect power from the drives while the firmware is loading. If you disconnect power, severe damage occurs and you will have to send the unit back to Aerotech for repair.

  7. While Configuration Manager loads the firmware, the Message Log shows messages that tell you about each operation that occurs. The progress bar shows the progress as the operations occur. After Configuration Manager loads the drive firmware, it automatically loads the card firmware onto the installed HyperWire card. The Message Log tells you when Configuration Manager successfully completes loading the firmware.
