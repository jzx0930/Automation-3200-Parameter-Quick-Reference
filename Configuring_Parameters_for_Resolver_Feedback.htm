Configuring Parameters for Resolver Feedback

Hardware: The following information applies only if you are using the Ndrive HLe or HPe, the Npaq, or the Nservo with the resolver option. This information does not apply to the Npaq MR.

To configure parameters to use resolver feedback, you must know the following information about your system.

  • If the motor has or does not have Hall-effect switches.
  • The maximum speed of the motor in terms of resolver revolutions per second.
Note: If the Commutation Initialization setting of the ResolverSetup parameter is set to Initialize Using Resolver, then you must determine the commutation offset of the resolver. Refer to CommutationOffset parameter for more information.

To configure parameters to use resolver feedback

  1. You must correctly set the various fields of the ResolverSetup parameter for your system.
    • Set the Resolution Select setting based on the maximum speed of the motor. Refer to the documentation of this setting for the maximum allowable speed for a given resolution.
    • If you plan to use the resolver feedback to initialize the motor commutation, Aerotech recommends that you set the Commutation Initialization setting to Initialize Using Resolver. However, if the motor has Hall-effect switches, or if you plan to use the Automatic Motor Set (Auto-MSET) or Commutation Search methods to initialize the commutation, you can use the Initialize Using Hall Signals setting.
  2. Next, use the Axis Parameter Calculator in Configuration Manager to configure the remaining axis parameters. Refer to Using Calculators to Set Parameters for more information.
  3. For the Position Feedback selection, select Other Encoder. For the resolution, enter the value from the Encoder Resolution Settings and Values table that corresponds to the Resolution Select setting that you selected in Step 1.

  6. Set the PositionFeedbackType parameter (and VelocityFeedbackType, if applicable) to Resolver.
  7. Save these parameter settings and reset the controller.
  8. Use the Feedback Tuning utility in Digital Scope to set the ResolverReferenceGain parameter.
