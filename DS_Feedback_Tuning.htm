Feedback Tuning

The Feedback Tuning utility is used to tune the feedback device signals so that optimum performance can be achieved. It is accessed from the Feedback Tuning tab of Digital Scope. This utility may be used with analog feedback devices only. Referring to the plot below, the goal of the feedback tuning utility is to tune the feedback device such that the blue Lissajous pattern closely matches the ideal red Lissajous circle pattern. The blue Lissajous should be shaped as a circle and should be no larger in diameter than the red circle over the entire length of travel of the axis being tuned.

Figure: Feedback Tuning Window

To Tune an Analog Feedback Device

Tuning an analog feedback device using the Feedback Tuning utility requires the following steps.

  1. Select the desired axis from the axis list in the main menu or the Axis box on the Axis toolbar.
    2. Figure: Digital Scope: Axis Toolbar

  2. Open the motion configuration box by clicking the Tune arrow on the Tuning toolbar.
  3. Select Enable Motion in the Motion Configuration box. The tuning utility will calculate an ideal distance and velocity for the tuning motion profile, which will be displayed in the Motion Configuration box.
    4. Figure: Digital Scope: Tuning Toolbar — Enable Motion

  4. Click the Tune button to enable the tuning motion and start the tuning algorithm. The tuning algorithm will continuously collect data snapshot and adjust the feedback gains so that the feedback device Lissajous matches the ideal Lissajous.
  5. Once the feedback device Lissajous matches the ideal Lissajous, click Stop to stop the tuning cycle.
  6. Once the feedback device gains have been calculated, we recommend that you check the Lissajous pattern over the entire length of travel of the axis. Check the pattern by moving the axis over the entire length of travel while displaying the Lissajous pattern. To display the Lissajous pattern (without tuning enabled), select the Collect Data button from the Action toolbar.
    7. Figure: Digital Scope: Action Toolbar — Collect Data

  7. If at some position the Lissajous becomes much larger than the ideal Lissajous, repeat the tuning procedure with the axis located at this position.
  8. The feedback gains that are calculated by the tuning algorithm are set as active on the controller but are not committed to the file system. To commit the parameters to the file system for future use, select the Commit Parameters button from the Tuning toolbar.
To Tune a Resolver
  1. Select the desired axis from the axis list in the main menu or the Axis box on the Axis toolbar.
    2. Figure: Digital Scope: Axis Toolbar

  2. Open the Motion Configuration box by clicking the Collect Data arrow on the Action toolbar.

    Select Enable Motion in the Motion Configuration box. The tuning utility will calculate an ideal distance and velocity for the tuning motion profile, which will be displayed in the Motion Configuration box.
    3. Figure: Digital Scope: Action Toolbar — Collect Data

  3. Click the Collect Data button to enable tuning motion. The application will continuously collect data snapshots and plot the relative to the ideal Lissajous circle.
  4. While collecting data snapshots, manually adjust the Resolver Gain, Resolver Offset, and Resolver Phase.
  5. After the resolver Lissajous matches the ideal Lissajous, click Stop to stop the tuning cycle.
  6. Once the resolver gains have been determined, check the Lissajous pattern over the entire length of travel of the axis. Check the pattern by moving the axis over the entire length of travel while displaying the Lissajous pattern. To display the Lissajous pattern, click the Collect Data button on the Action toolbar.
  7. If at some position the Lissajous becomes much larger than the ideal Lissajous, repeat the tuning procedure with the axis located at this position.
  8. The feedback gains that are calculated are set as active on the controller but are not committed to the file system. To commit the parameters to the file system for future use, select the Commit Parameters button from the Tuning toolbar.
To Tune the Velocity and Position Feedback Devices

To tune the velocity and position feedback devices that you configure for dual loop mode with MXR, refer to the To Tune the Velocity and Position Feedback Devices procedure in Configuring Parameters for Dual Loop Mode with MXR.
