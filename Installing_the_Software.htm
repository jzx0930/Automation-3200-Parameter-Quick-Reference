Installing the Software

Not all PC platforms can be used successfully with the Automation 3200. Refer to System Requirements for processor and chipset recommendations.

To install and run the A3200 software, you must have Administrative privileges for installation and your PC must have the Minimum PC Requirements.

After you install the software, you can optimize the performance by using the guidelines in Optimizing A3200 Performance.

To install the A3200 software

Warning: If you are using Microsoft Windows 7, run Windows Update a minimum of one time before you install the software. Make sure that all of the updates complete.
  1. Put the A3200 installation device into the applicable drive.
  2. Do one of the following.
    • If the AutoPlay dialog comes into view, click Run AutoRun.hta. Then continue to the next step.
      • Figure: Run AutoRun.hta

    • If the AutoPlay dialog does not come into view, open the A3200 installation device on your computer and run the AutoRun.exe application.
  3. On the Installation Resources dialog, click Install A3200 Software.
    4. Figure: Installation Resources

  4. If the installation does not prompt you to install required software, continue to the next step. If the installation prompts you to install required software, install the required software before you continue to install the A3200 software.
    1. In the InstallShield Wizard, click Install. The installation installs the required software.
      2. Figure: Required Software

    2. After the software installs, you might need to restart your computer. Click Yes to restart now. Click No to restart later. If you restart now, the installation continues automatically.
      3. Figure: Restart Required

  5. The installer opens automatically to the Welcome dialog. Click Next.
    6. Figure: InstallShield Wizard Welcome

  6. Accept the terms of the License Agreement. You can print this agreement for your reference. Click Next.
  7. Figure: License Agreement

  8. Enter the Product Key that is supplied with your installation device. Click Next. If the installation tells you that the key is not valid, enter the correct product key.
    9. Figure: Product Key

  9. If your license key does not include the Remote Option, continue to the next step.
  10. If your license key has the Remote Option, select the Setup Type and then click Next.
    • Complete — Installs the software in server mode and installs the real-time operating system.
    • Client Only — Installs only the client software and does not install the real-time operating system.
    Figure: Setup Type (for Remote option only)

  11. The software saves the file to the default location of C:\Program Files (x86)\Aerotech\A3200\. To save the file to a different location, click Change and select the new location. To continue, click Next.
    12. Figure: Select Destination Location

  12. Review the installation details. Click Install to start copying files.
    13. Figure: Start Copying Files

  13. The installation is completed. If you want to open Configuration Manager at this time, select Launch Automation 3200 Configuration Manager. Click Finish to close the Installation Wizard.
    14. Figure: InstallShield Wizard Complete
