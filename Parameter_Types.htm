  • Double Variables
  • String Variables

(1) In AeroBasic, doubles include Program Variables, $global, and $task. Strings include $strglobal and $strtask.

(2) AeroBasic handles all numerical variables as doubles. However, commands, operators, and functions that use 32-bit or 64-bit signed integers require whole numbers as inputs. In these cases, if you use a decimal value, the compiler will either round the value, truncate the value, generate a compile time error, or generate a runtime error. Refer to Double Type Overview for more information.

(3) These variable types are not used in the A3200 libraries, but they are included in this table for completeness.

4) In the MATLAB library you can pass in any numerical type as a function parameter and the MATLAB function converts the value to the correct data type. You do not need to know the exact numerical type of the input parameter of the function.

(5) You can use these variable types when you create Fieldbus mappings.
