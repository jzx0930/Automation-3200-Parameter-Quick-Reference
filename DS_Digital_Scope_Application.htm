Digital Scope Application Overview

Digital Scope is a system analysis and tuning application that lets you optimize system performance using a collection of advanced tuning tools and features. With system analysis tools, you can plot and graphically analyze system data. With tuning tools, you can automatically calculate servo-loop and feedback device parameters.

The Digital Scope application includes a variety of utilities that let you do analysis and tuning operations that include the following.

  • Collect and plot various signals including position, velocity, and position error on multiple axes.
  • View multiple overlapped 1D (one-dimensional) and 2D plots.
  • Use cursor and zooming tools to graphically examine collected data.
  • Issue motion commands while collecting data.
  • Compute the Discrete Fourier transform of any of the collected signals to identify loop instabilities.
  • Automatically tune the servo loop to achieve optimal system performance using the EasyTune and Autotune features.
  • Calculate and enable multiple digital filters to improve system performance.
  • Do loop transmissions on each axis to allow for advanced analysis and optimization of servo-loop tuning and performance.
  • Automatically tune the encoder feedback device and view the results.

Using Digital Scope

Note: Aerotech does not recommend running Digital Scope, Motion Designer, and Motion Simulator applications at the same time because they use the same data collection features on the controller.

You can use the Alt key to access menus in the menu toolbar of the application. Press Alt. Then press the letter that is underlined in the name of the menu that you want. In the menu that comes into view, press the letter that is underlined in the name of the option that you want.

Digital Scope comes standard with these tabs and utilities. Different tabs might have different components, custom fields, or options. Make sure that you are in the correct tab before you try to use a utility.

  • Scope Tab
    • Data Collection in Digital Scope
    • Conditional Plotting
    • Program Display
    • Executing Step Motion
    • Autotuning an Axis
  • Loop Transmission Tab
  • Feedback Tuning Tab
  • EasyTune Tab

Depending on the license option you have purchased, additional tabs might be included. Refer to License Options for more information.
