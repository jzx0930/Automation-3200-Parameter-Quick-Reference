PSO Hardware Status

Hardware: PSO does not apply when used with the Nmark SSaM or the Nstep.

You can view the PSO Hardware Status in Block 4 of the Drive Info tab of the Status Utility, or it can be retrieved in a program with the DRIVEINFO function. You can also use the PSO Status item to plot this information in Digital Scope.

The table that follows tells you about each bit and shows all of the hardware to which the bit applies.

Hardware: The Npaq MR contains multiple ML or MP drives. Refer to the ML or MP section of this table to get the values for your Npaq MR.
