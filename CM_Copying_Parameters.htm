Copying Parameters

In Configuration Manager you can copy parameters from a parameter file to other parameter files. You can also copy the parameters from an axis or task to parameters in other axes or tasks in the same parameter file.

  1. In Configuration Manager, open the Copy Parameters dialog by doing one of the following.
    • On the Tools menu, select Copy Parameters or click the Copy Parameters button on the toolbar. When you open the Copy Parameters dialog, if a parameter file is selected in the Network Explorer that file becomes the source.
      1. To select or change the source, click the Source arrow and select an item from the tree or click Browse to select a parameter file from the file system.
      2. Select the Destination.
        • To copy the parameters from an axis or task to parameters in other axes or tasks in the same parameter file, click the Destination arrow and select Same as Source.
        • To copy between parameter files, select a Destination that is different from the Source. Click the Destination arrow. Select an item from the tree or click Browse to select a parameter file from the file system.
    • In the Open Files folder of Network Explorer, drag the parameter file that you want to use as the Source over the parameter file that you want to use as the Destination. The Source and Destination that you specify are automatically selected in the Copy Parameters dialog and cannot be changed.
  2. In the Parameter Categories section, select the parameters that you want to copy.
    • If you select one or more axis parameters, use the Axis tab under Parameter Mapping to specify how the axes are mapped from the Source to the Destination.
    • If you select one or more task parameters, use the Task tab under Parameter Mapping to specify how the tasks are mapped from the Source to the Destination.

  3. NOTE: You can copy individual parameter categories instead of the whole parameter file. In the Parameter Categories section of the Copy Parameters dialog, select the categories and subcategories that you want to copy. Then when you click OK, Configuration Manager copies only the parameters in the categories and subcategories that you selected.

  4. Verify your selections and click OK. If the Destination parameter file is not already open in the Network Explorer, the file opens before the parameters are copied. Before you save this file, make sure that the parameters are correct.
  5. Save the Destination parameter file. To save the parameter file, on the File menu select Save or Save As, or click the Save button on the toolbar.
