Drive Array Overview

A3200 drives have a drive array that is used at the drive level for fast data transfer. Encoder positions and analog inputs that are collected by the DATAACQ commands are stored in this array. On some drives, values for some PSO commands are also stored in this array.

Table:     Ways to Use the Drive Array

Operation Command Applicable Drive Types

Storing acquired data

DATAACQ

GL4, Ndrive CP, HLe, HPe, ML, MP, Nmark CLS and GCL, Npaq, Npaq MR, Nservo, XC2, XC4, XC4e, XL4s, and XR3

Distances for firing events

PSODISTANCE ARRAY

GL4, Ndrive FCL, FLS, HLe, HPe, ML, and QLe, Nmark CLS and GCL, Npaq, Nservo, XC2, XC4, XC4e, XL4s, and XR3

Window ranges

PSOWINDOW RANGE ARRAY

GL4, Ndrive FCL, FLS, HLe, HPe, ML, and QLe, Nmark CLS and GCL, Npaq, Nservo, XC2, XC4, XC4e, XL4s, and XR3

PSO output state

PSOOUTPUT BIT MAP

GL4, Ndrive FCL, FLS, HLe, HPe, ML, and QLe, Nmark CLS and GCL, Npaq, Nservo, XC2, XC4, XC4e, XL4s, and XR3

Masking PSO firing events

PSOOUTPUT PULSE BIT MASK

GL4, Ndrive FCL, FLS, HLe, HPe, ML, and QLe, Nmark CLS and GCL, Npaq, Nservo, XC2, XC4, XC4e, XL4s, and XR3

To move data between program variables and the drive array, use the ARRAY READ and ARRAY WRITE commands:

  • To move values from the A3200 SMC controller to the drive array, refer to ARRAY WRITE command. Array values are written at 10,000 elements per second.
  • To read data from the PSO array into a specified A3200 SMC array variable, refer to ARRAY READ command. Array values are read at 500 elements per second. You can increase the number of values that the controller can receive by using the FAST qualifier.

When you reset the drive, all the elements in the array are set to zero. The PSOCONTROL RESET command does not clear the array data.

Each element in the array is 32 bits. The number of elements in the drive array is different for different drive types. The table that follows shows the array size for each drive type. The PSO FIFO Load Rate column shows the rate at which values are transferred from the drive array to the PSO hardware (FIFO).
