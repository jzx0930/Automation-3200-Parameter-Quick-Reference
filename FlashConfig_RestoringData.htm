Restoring FlashConfig Data

When you connect a stage that has FlashConfig to a drive that supports FlashConfig, you can restore the controller with the data used at the factory to initially configure the axis. If you purchased integration from Aerotech, you can restore axis parameters, calculator information, and calibration data (if applicable). If you did not purchase integration, you can restore calculator information.

The data stored in FlashConfig is also used to sense if you connect a new stage, connect a stage to a different drive, or disconnect a stage. If the controller senses one of these conditions, you must resolve the mismatch condition before you can restore individual axes with the data that is stored in FlashConfig on the stage. Refer to Resolving FlashConfig Mismatches.

For information on the types of data that are kept in FlashConfig, refer to FlashConfig Data.

Note: You might not be able to restore 2D calibration from a stage that supports FlashConfig if the number of corrections used to initially calibrate the stage is greater than the maximum calibration table size that can be stored in FlashConfig.

Figure: Restore FlashConfig Dialog Box

To restore data from FlashConfig

  1. Open Configuration Manager and connect to a controller.
  2. In the Network Explorer, select the controller.
  3. On the toolbar, click the FlashConfig button .
  4. In the Restore FlashConfig dialog box, review the types of data that are kept in FlashConfig for each connected axis.
  5. Select the check box for each axis that you want to restore with FlashConfig data. When you click Restore, the data from the stage overwrites the data on the controller for each axis that you select.

  6. If FlashConfig contains valid calibration data, review the changes that you want to make to Existing Calibration.

  7. Click Restore. For the selected axes, the data from the stage is restored to the controller.
  8. If you are prompted to commit parameters or reset the controller, click Yes. After the controller resets, the restore is complete. You can enable and use the axis.

You can change the parameters that were transferred from the stage to the controller by editing the parameters in Configuration Manager or by tuning the axis in Digital Scope.

FlashConfig creates a backup of any data on the controller that is overwritten during the restore operation. The backup file is saved in the User Files directory.
