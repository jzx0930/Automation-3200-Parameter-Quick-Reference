EasyTune

EasyTune is a tool in Digital Scope that automatically tunes and optimizes an axis. You can use EasyTune to stabilize an unstable axis or to improve the performance of an unoptimized axis. When EasyTune is finished, the optimized servo gains and servo filters are committed to the controller. Every time you run EasyTune, a log file is automatically saved to the EasyTune directory in your User Files directory.

NOTE: EasyTune does not support voice coil stages or galvo scanners.

To run EasyTune

  1. Make sure that it is safe for motion to occur on the selected axis.
    • Make sure that the selected axis is away from any end-of-travel limits or software limits.
    • Make sure that there is nothing blocking the selected axis.
  2. To run EasyTune on a position mode gantry, make sure that the parameters that follow are configured correctly.
    • The GantrySeparationThreshold Parameter must be set to a non-zero value. The value must be the same on the master gantry axis and the slave gantry axis.
    • On the FaultMask Parameter, the Gantry Misalignment Fault must be enabled on the master gantry axis and on the slave gantry axis.
    • On the FaultMaskDisable Parameter, the Gantry Misalignment Fault must be enabled on the master gantry axis and on the slave gantry axis.
  3. In Digital Scope, on the EasyTune tab, click Start.
  4. On some axes, EasyTune must command additional motion to identify the selected axis.
    5. Figure: EasyTune — Constant Velocity
    • Review the default values for Travel Distance and Speed. If they are safe, click Continue.
    • If EasyTune does not supply default values, enter values for Travel Distance and Speed that are safe for the selected axis. Then click Continue.
    • For Travel Distance, start with a value of 1 mm for an axis that produces linear motion or 20° for an axis that produces rotational motion. For Speed, start with a value that is 1/10 of the Travel Distance.
  5. When EasyTune is complete, the optimized servo gains and servo filters are committed to the controller.
  6. Test the performance of the selected axis. If it is necessary, you can select a performance target that is sufficient for your system and then start EasyTune again. See the Using a Custom Performance Target section of this topic.

To stop EasyTune

If it is necessary to stop EasyTune, click Stop or click the Abort button .

Using a Custom Performance Target

You can select a custom performance target to help you get to specific performance goals for the selected axis. The first time that you run EasyTune on an axis, use the default value of Normal. After the selected axis is optimized, you can test the performance of the axis and change the target that EasyTune uses.

To change the performance target

  1. Select Use a Custom Performance Target.
    2. Figure: EasyTune — Custom Performance Target
  2. Move the slider to a performance target that is sufficient for your system.
    • If the axis is undershooting when it moves into position, try a more aggressive performance target.
    • If the axis is overshooting when it moves into position, try a more conservative performance target.
  3. Click Start. EasyTune tries to get to the selected performance target.

Troubleshooting EasyTune

EasyTune might not always complete successfully. Here are some common issues that occur and how you might fix them.

Error Message Solution

EasyTune could not identify the axis.

Before you start EasyTune, make sure you are getting valid position feedback from the selected axis.

EasyTune could not complete because an axis fault occurred.

If the axis fault is an end-of-travel limit fault or software limit fault, make sure the axis is away from any limits and start EasyTune again.

EasyTune cannot run on task (number) because the task state is...

Make sure the selected task in Digital Scope is not InActive, not PLCReserved, and not Queue.

If you cannot find the cause of this error, or if you tried the recommended solution and receive the error again, record the text of the error message and contact Aerotech Global Technical Support.
