Recognition of the Target Word

Replacement text is substituted only at locations where the target word is recognized.

The target word is recognized in the text only if the two characters on either side of the word are not alphanumeric characters.

For Example

If the statement ‘#define dog spot’ is made, the following text conversions occur.

    my dog is friendly

    becomes

    my spot is friendly

    the dogs are friendly

    becomes

    the dogs are friendly

    7*dog[6]

    becomes

    7*spot[6]

The target word is not recognized in comments or string literals.

The target word is recognized only on lines after the #define or #undef statement that references the target word. Lines in the file before that statement, or lines in files that are included before that statement are not searched for the target word.

For Example

' No substitution of CONSTANT done here.
$global[0] = CONSTANT
' This defines CONSTANT.
#define CONSTANT 2
' Substitution of CONSTANT with 2 is done here.
$global[0] = CONSTANT
' This "undefines" CONSTANT.
#undef CONSTANT
' No substitution of CONSTANT done here.
$global[0] = CONSTANT

becomes

$global[0] = CONSTANT
$global[0] = 2
$global[0] = CONSTANT
