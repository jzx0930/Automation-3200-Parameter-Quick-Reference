Firmware Loader Overview

NOTE: If you want to load firmware on the HyperWire drives, Aerotech recommends that you use the Drive Configuration dialog in Configuration Manager. Refer to Loading Drive Firmware for more information.

Aerotech strongly discourages you from using a version of the A3200 software that does not match the firmware. Each time you install a new version of the software, use the Firmware Loader application to load the same version of firmware onto a drive.

After Firmware Loader loads firmware onto the drives, it might be necessary for you to restart your PC. If you must restart your PC after the firmware loads, Firmware Loader asks you if you want to continue. If you select Yes, the application completes loading the firmware. Then you must restart your PC to make sure your motion system operates correctly.

Refreshing the List of Drives

When Firmware Loader opens, the list of drives usually refreshes correctly. However, there might be instances when you must manually refresh the list. For example, if the drive was not powered on when Firmware Loader first opened, the list of drives is not complete. To refresh the list, click Refresh at the bottom of the Controller Info section.

Loading Firmware onto the Drives

  1. Select the drives on which you want to have firmware loaded.
  2. By default, all drives in the list are selected and the new firmware is loaded onto all selected drives . To deselect a drive so that the new firmware is not loaded onto that drive, clear the box by that specific drive. The new firmware is loaded only onto the drives that stay selected.
  3. Select the firmware file that you want to load onto the drives.

    		4.

    WARNING: Aerotech recommends that you use the default firmware file. If you do not use the default file and you select an incorrect file, the firmware and software become mismatched and serious problems can occur.
  4. If you need to specify a firmware file that is different from the default firmware file, in the Files section of the Firmware Loader, click Browse. Then select the file that you want to use.
  5. Load the firmware.
  6. To start loading the firmware, click the Program button. While the process executes, the messages in the Operation Log describe the action that is currently being done. The progress bar shows the progress of the current stage of the operation.
  7. Danger: Do not disconnect power from the drives while the firmware is loading. If you disconnect power, severe damage occurs and you will have to send the unit back to Aerotech for repair.
