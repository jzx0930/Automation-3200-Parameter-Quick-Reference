Joystick Configuration

The following sections describe which parameters to configure for correct joystick operation.

After you have successfully configured the joystick, you can activate it by using the JOYSTICK ON command.

Analog and Digital Input Settings

Note: To configure the joystick to the analog inputs that you want, use the JoystickInput0, JoystickInput1, and JoystickInput2 axis parameters.

To configure Joystick Button A, Joystick Button B, and Joystick Button C to the digital inputs that you want, configure the JoystickAxesSelect and JoystickSpeedSelect axis parameters as shown in the following table.

Table:     Joystick Buttons

Use this button to change between axis speeds that are specified by the JoystickLowSpeed and JoystickHighSpeed axis parameters.

Use this button to cancel or deactivate the joystick.

JoystickAxesSelect and JoystickSpeedSelect

Configure the JoystickInterlock parameter to specify a joystick interlock signal. This safety feature makes sure that joystick motion does not occur from an unconnected analog channel.

Some joystick parameter settings are different for each drive type.

Voltage Settings (Optional)

You can use the following parameters to specify a voltage range for the joystick.

  • JoystickInput0MinVoltage
  • JoystickInput0MaxVoltage
  • JoystickInput0Deadband
  • JoystickInput1MinVoltage
  • JoystickInput1MaxVoltage
  • JoystickInput1Deadband
  • JoystickInput2MinVoltage
  • JoystickInput2MaxVoltage
  • JoystickInput2Deadband

Calculate the center voltage for a joystick input by using the MinVoltage and MaxVoltage parameters. The MinVoltage and MaxVoltage defaults are set to 0V and 5V, which gives a central voltage of 2.5V. Use the Deadband parameters to specify a voltage range in which the joystick is in the center position and no motion occurs. The default Deadband parameters are set to 0.1V, which means that motion will not occur when the joystick voltage is between 2.4V and 2.6V.

When you first issue the JOYSTICK command, the joystick (including Joystick Buttons A, B, and C) is not active and will not take control of the axes until all joystick analog inputs are within the deadband range. The deadband range is ± one voltage deadband (set deadband range with the Deadband parameters) of the central voltage of the joystick range (set joystick range with MinVoltage and MaxVoltage parameters). For example, with default parameter values, if the joystick is being held so the voltage on the vertical input is 5 volts, the joystick will not control the axes until the voltage returns to within the range of 2.4V to 2.6V.

After you issue the JOYSTICK command and the joystick is controlling the axes, Joystick Buttons B and C will function at any time. However, Button A will only function if the joystick is in the deadband when you press the button.

Feedrate Settings

The A3200 supplies two joystick speed ranges that you can configure with parameters that follow. When you activate the joystick, the joystick initially operates in the low speed range. To change between the low and high-speed ranges, use the joystick button defined by the JoystickSpeedSelect parameter (typically Joystick Button B).

  • JoystickLowSpeed
  • JoystickHighSpeed

Axis Group Settings

Use the AXISGROUP keyword of the JOYSTICK command to configure axis groups. One set of axes must be defined as an axis group. If you do not designate an axis group with the JOYSTICK command, you will receive a task error when you activate the joystick.

Example: Configure an Axis Group

' Use the X and Y axes to configure a 2D joystick axis group.
JOYSTICK 2D AXISGROUP 0 X Y
' Use the X, Y, and Z axes to configure a 3D joystick axis group.
JOYSTICK 3D AXISGROUP 0 X Y Z
