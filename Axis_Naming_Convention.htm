Axis Naming Conventions

An axis name can be any string of letters, numeric digits, and underscores. In some situations, you can specify an axis by using an axis index or axis variable.

Default Axis Names

By default, axis names are assigned by a communications channel and are named as follows (corresponding to communications channels 1 to 32): X, Y, Z, U, A, B, C, D, x, y, z, u, a, b, c, d, XX, YY, ZZ, UU, AA, BB, CC, DD, xx, yy, zz, uu, aa, bb, cc, and dd.

Note: If you change an axis name, the change does not have an effect in an application until you exit and reenter the application and do a reset from that application.

Changing Axis Names in a Parameter File

Use the following procedure to change the names of one or more axes in a parameter file.

  1. On the main menu, select File → New Parameter File.
  2. Figure: Main Menu — New Parameter File

  3. In the Network Explorer of Configuration Manager, a new parameter file comes into view under the Open Files folder. This file becomes the active parameter file.
  4. Figure: Network Explorer — Active Parameter File

  5. In the Network Explorer of Configuration Manager, right-click the active parameter file and select Configure Axes.
  6. Figure: Network Explorer — Configure Axes

  7. In the dialog that comes into view, select the axis that you want to rename. Then click Rename.
  8. Figure: Renaming Axis

  9. Type a new axis name. Then click OK.
  10. Figure: Renamed Axis

Rules for Custom Axis Names

  • Axis names are case sensitive. (XAXIS is a different axis than xaxis.)
  • You cannot use key words that are used for commands (for example, FOR and IF) as axis names.
  • You cannot use E, e, F, f, G, g, H, h, I, i, J, j, K, k, L, l, M, m, N, n, O, o, P, p, Q, q, R, r, S, s, T, or t as axis names.
  • Axis names cannot be more than 31 characters.
  • The first two characters in an axis name cannot be numeric digits.
  • You must separate the axis name that you create (with a space, equal sign, or parentheses) from a constant value given in a command (for example, LINEAR MYAXIS=5).

Table:     Valid Axis Names

Syntax Explanation 
X

Valid single character.

 
Xaxis_1

Valid multi-character.

 
LINEAR MYAXIS $global[0] F100

Valid multi-character.

 
LINEAR MYAXIS 7.8 F100

Valid multi-character: constant follows a space.

 
LINEAR MYAXIS=7.8 F100

Valid multi-character: constant follow an equal sign.

 
LINEAR MYAXIS(7.8) F100

Valid multi-character: constant is in parentheses.

Table:     Invalid Axis Names

Syntax Explanation 
If

Invalid: If is a command key word.

 
ThisAxisNameIsMuchMuchMuchTooLong

Invalid: more than 31 characters.

 
X1X

Invalid: second letter is a numeric digit.

 
LINEAR MYAXIS7.8 F100

Invalid: constant immediately follows the axis name.
