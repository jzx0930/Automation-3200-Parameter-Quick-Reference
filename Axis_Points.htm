Axis Points

Syntax:

 
<axis> <value> <axis> <value> ...

Example:

 
X10 Y11
X10Y=11
MYAXIS 10 Y11
$aptglob[7]    ; axis point variable

Axis points can be constants or variables. You can use them to do the following.

  • Specify target points to motion commands, including RAPID, LINEAR, CW, or POSOFFSET SET.
  • Pass an argument to a subroutine.

Axis point constants are a series of one or more axis words, each optionally separated by white space. Each axis word consists of two parts: an axis name, followed by an expression.

If you are using a custom axis name, you must separate the axis name from the expression with a space, equal sign, or parenthesis.

Note: If an element of an axis point has not been loaded with a value, it does not command motion on that axis. For example, if four axes are present and three axes are loaded with values, the axis point is passed to a LINEAR motion command, and the axis that has not been assigned a value does not command any motion.
