Motion Composer Suite Overview

Aerotech offers standard Motion Composer Suite applications and optional Motion Composer Suite applications. If you have questions about how to use an application, contact Aerotech Global Technical Support. To purchase an optional Motion Composer application, contact your Aerotech sales representative.

Standard Motion Composer Suite Applications

Configuration Manager

Edit parameters and change system settings.

Console

Debug your programs with this low-level command line debug utility.

Digital Scope

Collect and analyze data and create one and two-dimensional plots.

Firmware Loader

Load new firmware onto connected hardware.

Motion Composer

Develop AeroBasic programs, jog axes, and view status.

Status Utility

See the status of all 32 axes and all four controller tasks. If you purchased the Professional Option, Status Utility shows the status of all 32 axes and all 32 tasks.

System Snapshot

Capture the state of your system and output this information into a standard HTM file that you can use to help diagnose problems or to identify a setup that is not valid.

Optional Motion Composer Applications

CNC Operator Interface

Run programs, jog axes, view status, and operate the machine.

Motion Designer

Graphically generate motion profiles that are executed on the controller.

Motion Simulator

Do simple and advanced motion stimuli and analyze the results.
