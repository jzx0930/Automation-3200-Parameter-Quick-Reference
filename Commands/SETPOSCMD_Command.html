SETPOSCMD Command

Sets the position command of the axis at the servo loop level and adjusts the position feedback for position error.

Syntax:

 
SETPOSCMD <Axis>, <Position>

Example:

 
SETPOSCMD X, 5.2

Table:     SETPOSCMD Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<Position>

64-bit IEEE floating point

The position to set the axis to.

Use the SETPOSCMD command to set the value of the position command of the axis at the servo loop level, without physically moving the position of the axis. The command also adjusts the position feedback so that the position error is preserved.

You can read the Position Command and Position Feedback with the AXISSTATUS function.

Warning: Do not use this command to arbitrarily change the axis position if axis calibration is active. Doing so will cause the calibration table to be misaligned and incorrect. Use the POSOFFSET SET command instead to set axis programmed positions without having on effect on calibration table alignment.

For Example

Say that the position command of Axis X is 10.0 and the position feedback of Axis X is 10.1. Note that feedback might be different from the position command because of servo dither.

To reset the position command to 0.0 and reset the position feedback to 0.1, use this syntax. 

SETPOSCMD X 0
