WAIT Command

Pauses program execution until a specified condition is true or the command times out.

Syntax:

 
WAIT (<ConditionalExpression>) <Time>
 
WAIT (<ConditionalExpression>) <Time> <Flag>

Example:

 
WAIT( $global[1] == 1) 10000
WAIT( $global[1] == 1) -1

Table:     WAIT Command Arguments

Argument

Type

Description

<ConditionalExpression>

Other

The condition to examine.

<Time>

64-bit IEEE floating point

The quantity of time to pause program execution, in milliseconds.

<Flag>

32-bit signed integer

(Optional) Sets the info variables to prevent causing a task error when timing out.

Use the WAIT command to pause program execution until the specified condition is true or the command times out. The conditional expression is evaluated at the rate of once per millisecond, including the first millisecond. The expression is always evaluated a minimum of one time. To wait for motion to complete, use the WAIT MOVEDONE Command or WAIT INPOS Command.

If the WAIT command times out, the controller generates a task error. But if this command times out and you specified a nonzero value for the <Flag> argument, the controller sets the $info[0] variable to a nonzero value that lets the program continue. For more information about the $info[0] variable, refer to Info Variables.

<Time>

If the value specified for <Time> is greater than 231 milliseconds (24 days), the value is truncated to 231 msec. <Time> must be greater than 0.0, with the following exceptions.

  • If the time-out is specified as -1, the command never times out. It waits indefinitely.
  • If the time-out is specified as -2, a PROGRAM RESTART Command is automatically executed if the wait condition is false.

Examples

' A task error occurs after a timeout of 10000 ms.
WAIT( $global[1] == 1) 10000

' The program continues without a task error.
' The $info[0] variable is set to zero if no timeout occurs.
' The $info[0] variable is set to a nonzero value if the WAIT command times out after 1000 ms.
WAIT( $global[1] == 1) 1000 1

' Waits forever (never times out).
WAIT( $global[1] == 1) -1
