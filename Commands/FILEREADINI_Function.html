FILEREADINI Function

Reads data from a formatted .ini file.

Syntax:

 
<return> = FILEREADINI <FileName>, <SectionName>, <ValueName>, <DefaultValue>

Example:

 
$global[1] = FILEREADINI "C:\MyFolder\MyFile.ini", "MySection", "MyValue", -1

Table:     FILEREADINI Function Arguments

Argument

Type

Description

<FileName>

String

The name of the file that you want to read.

<SectionName>

String

The name of the section (the text in the brackets of the .ini file).

<ValueName>

String

Specifies the entry in the section of the .ini file (The text that is to the left of the equal sign (=)).

<DefaultValue>

64-bit IEEE floating point

The value that is returned if the entry is not found. The value must be floating point.

RETURNS

other

The variable that stores the return value. The value is either a string or a 64-bit IEEE floating point value.

Note: This command is executed as a callback and does not execute in real time. The time that it takes for this command to execute is different for different PC configurations. Refer to Callback Commands for more information.

Use the FILEREADINI function to read a floating point or string value from a text file formatted as a Windows .INI file. This function requires an .INI file name. You cannot use this function to access the active parameter file because the format of the parameter file is XML (not INI).

If you do not specify an absolute file path in the file name, the file path is relative to the program that is currently executing. If the specified section or entry name does not exist in the file, the <defaultValue> argument value is returned. If the specified file does not exist, a task error is generated.

Example

file: "FILE.INI" contents:
[Section1]
Variable1 = 11
Variable2 = 22
[Section2]
Variable3 = 33

' Example program commands. Comments occur at the end of each line.
' Set $global[0] to 22.
$global[0] = FILEREADINI "File.Ini", "Section1", "Variable2", -1   
' Set $global[1] to -1.
$global[1] = FILEREADINI "File.Ini", "Section1", "garbage", -1
