MSGFILESELECT Function

Opens a Windows file dialog box.

Syntax:

 
<return> = MSGFILESELECT <Flag>
 
<return> = MSGFILESELECT <Flag>, <Messages>

Example:

 
$strglob[0] = MSGFILESELECT DF_FILESELECT_OPEN, "Browse for a program 
 file;C:\MyFolder;FILEWRT1.PGM;Parameter
 File(*.pgm)|*.pgm|All Files(*.*)|*.*"

Table:     MSGFILESELECT Function Arguments

Argument

Type

Description

<Flag>

64-bit IEEE floating point

Specifies the type of file dialog box to open. To see the possible values, refer to Table: <Flag> Argument Values.

<Messages>

String

(Optional) A semi-colon delimited text string that configures the file dialog box. To see the possible values, refer to Table: <Messages> Argument Values.

RETURNS

other

Returns the file name and the full path of the selected file.

Use the MSGFILESELECT function to open and configure a Windows file dialog box.

Use the <Flag> argument to specify the dialog box that you want to open. The following table shows the possible values for the <Flag> argument.

Table:     <Flag> Argument Values

Value Definition

DF_FILESELECT_OPEN

Show the File Open dialog box.

DF_FILESELECT_SAVE

Show the File Save dialog box.

You can use the <Messages> argument to configure the dialog box. You can specify a maximum of four messages in a semi-colon delimited string. If you do not want to use one of the messages, skip the message by putting two semicolons together. You must start and end the string with quotation marks (" "). And, you must put semicolons between the messages. For example, "title;directory;file;filter".

Table:     <Messages> Argument Value Examples

Definition Example Value

The title of the dialog box.

Browse for a program file.

The directory to show in the dialog box. The default install directory is C:\Program Files\Aerotech\A3200

[Install directory]\Samples

The file to select.

FILEWRT1.PGM

The file filter that you want to use.

Parameter File(*.pgm)|*.pgm|All Files(*.*)|*.*

This function returns the full path and file name of the selected file to a string variable. The function does not do actual file operations. If the user clicks the Cancel button in the dialog box, the function returns an empty string.

Note: Motion Composer, CNC Operator Interface, Motion Simulator, or custom application must be open for this command to work. By default, Motion Composer, CNC Operator Interface, and Motion Simulator all try to register for these callbacks. The first application that registers will get the callbacks. The next application that tries to register will get an error. You can change the default behavior in Preferences of each application.
