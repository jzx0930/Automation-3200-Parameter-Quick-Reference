INCREMENTAL Command

Sets motion commands to be in incremental mode.

Syntax:

 
INCREMENTAL

Example:

 
INCREMENTAL

Use the INCREMENTAL command to set incremental mode. The controller interprets move target positions as absolute or incremental. Incremental mode means that move target positions are relative to the current axis location. You can set incremental mode at any time during program execution. INCREMENTAL is modal and remains in effect until absolute mode is specified.

You can set the default mode used by the task by using the DefaultProgrammingMode parameter

The incremental programming mode state is indicated in Bit 1 of the TaskMode data item of the TASKSTATUS Status Function.

The RolloverCounts axis parameter does not have an effect on the direction of travel in incremental axes.

Example

' This example program is illustrated in Figure: Incremental Mode Programming Example.
' This example program uses a starting position of (0,0).

' Enable incremental programming mode.
INCREMENTAL
' Set the feedrate for subsequent moves.
F100
' Move the X and Y axes an increment of 10.
LINEAR X10 Y10
' Move the X axis 15 and the Y axis 25 from the axis position.
LINEAR X15 Y25
' Move the X axis 15 and the Y axis 10 from the axis position.
LINEAR X15 Y10
