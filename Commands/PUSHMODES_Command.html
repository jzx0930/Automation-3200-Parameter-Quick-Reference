PUSHMODES Command

Stores the value of the modal settings of the program.

Syntax:

 
PUSHMODES

Example:

 
PUSHMODES

The PUSHMODES command stores the value of the TaskMode status item. You can use this command before you call a subroutine to store the modal task states. Then, use the POPMODES command to restore the modal task states.

The modal task states are automatically restored when one or more of these actions occur.

  • You issue a program stop.
  • You issue a program reset.
  • You issue a program restart.
  • You issue a FARGOTO command.
  • A task error occurs.
  • The program completes.

Example

' Enable absolute programming mode.
ABSOLUTE
' Save the task modes.
PUSHMODES
' Restore the task modes.
CALL mysub
POPMODES
' In absolute programming mode.
X10
' ...
' Complete the primary program.
END PROGRAM
' This is a subroutine entry.
DFS mysub
' The modes change in the subroutine.
INCREMENTAL
' Subroutine return.
ENDDFS
