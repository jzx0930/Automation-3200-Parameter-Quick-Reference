POSOFFSET Command

Sets or clears an arbitrary program offset position.

Syntax:

 
POSOFFSET SET <Axis> <Value> [ <Axis> <Value> ... ]
 
POSOFFSET CLEAR <Axis> [ <Axis> ... ]

Example:

 
POSOFFSET SET X-1 Y-2
POSOFFSET CLEAR X Y

Table:     POSOFFSET Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<Value>

64-bit IEEE floating point

(Optional) The value at which to set the parameter.

Use the POSOFFSET command to set the current program positions to the specified values. Setting the current program position defines a new coordinate system offset and does not cause axis movement. To clear the offset positions and align the software positions with the home positions, use POSOFFSET CLEAR or the HOME command. A POSOFFSET CLEAR clears the program position offset only on the specified axes.

You can use the POSOFFSET command to set a reference point for a sequence of moves specified in absolute mode. For example, if you execute a POSOFFSET SET X0 Y0 while the axes are at (2,3), the program positions become (0,0) although no movement takes place. A subsequent ABSOLUTE RAPID X1 Y1 moves to (3,4), although the program positions will display as (1,1).

You can also declare a software home at a position other than the existing program position by giving nonzero coordinates to the POSOFFSET command. For example, if the axes are at (2,3) (with no software home active) and you execute a POSOFFSET SET X-1 Y-1, the new software home is at (3,4). A subsequent ABSOLUTE LINEAR X0 Y0 moves to (3,4), although the program positions will display as (0,0).

Example

' This example program shows X,Y positions as (x, y).
' Enable absolute programming mode.
ABSOLUTE
' Move the X and Y axes to PositionCommand=(3, 3).
' ProgramPositionCommand=(3, 3).
LINEAR X3 Y3
' Do not move the X and Y axes to PositionCommand=(3, 3).
' ProgramPositionCommand=(-1, -3).
' Refer to Figure: Example Program Illustration.
POSOFFSET SET X-1 Y-3
' Move the X and Y axes to PositionCommand=(4, 6).
' ProgramPositionCommand=(0, 0), which is the software home position.
' Refer to Figure: Example Program Illustration.				
LINEAR X0 Y0 
' Do not move the X and Y axes to PositionCommand=(4, 6).
' ProgramPositionCommand=(4, 6).
POSOFFSET CLEAR X Y
