THERMOCOMP CONFIG UPDATEINTERVAL Command

Configures the regular updating interval of thermal compensation (THERMOCOMP) on the specified calibration table.

Syntax:

 
THERMOCOMP CONFIG UPDATEINTERVAL <TableNumber>, <UpdateInterval>

Example:

 
THERMOCOMP CONFIG UPDATEINTERVAL 0 0.5

Table:     THERMOCOMP CONFIG UPDATEINTERVAL Command Arguments

Argument

Type

Description

<TableNumber>

32-bit signed integer

The zero-based calibration table index.

<UpdateInterval>

64-bit IEEE floating point

The automatic UpdateInterval specified in seconds. Resolution is limited to 0.1 seconds. Specifiy 0 to enable manual updating only.

Optional Purchase Required: The information in this topic applies only if you purchased the ThermoComp™ feature on your Aerotech positioning product.

The THERMOCOMP CONFIG UPDATEINTERVAL command specifies how frequently the thermal compensation is updated in the calibration table that is specified by the <TableNumber> argument. If the value of the <UpdateInterval> argument is set to 0, this table must be updated manually. Tables configured to update manually update only when the controller executes a THERMOCOMP UPDATE ONCE command. Refer to THERMOCOMP UPDATE Command.

Advanced Configurations

NOTE: This information is supplied for advanced users.

Aerotech fully configured these commands at the factory with coefficients that are specific to your system. You can issue the THERMOCOMP ON/OFF Command without making more changes to these commands. But if you want to customize your system more, you can edit the arguments of these commands before you issue the THERMOCOMP ON command.

  • THERMOCOMP CONFIG MODE Command
  • THERMOCOMP CONFIG UPDATEINTERVAL Command
  • THERMOCOMP CONFIG VALUES Command
