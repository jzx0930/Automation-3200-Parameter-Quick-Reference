HOME Command

Homes the axes.

Syntax:

 
HOME <Axis> [ <Axis> ... ]
 
HOME <Axis> [ <Axis> ... ] CONDITIONAL

Example:

 
HOME X Y

Table:     HOME Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

Use the HOME command to send the specified axes to their hardware absolute reference positions. At the end of the home cycle, the position command value is set to zero unless the HomePositionSet parameter is a nonzero value. Program execution halts until all axes have completed their home cycle. The manner in which an axis or axes return to their home positions is determined by the parameters in the Home Parameter Category. There is no coordination between axes during a home cycle.

If the controller encounters a fault (with the exception of Probe faults and the End of Travel limit fault in the direction of the home) during the home cycle, the controller will stop the home cycle and continue with program execution.

You can use the optional CONDITIONAL keyword so that the controller homes the specified axes only if they are not currently homed. Using this keyword, the controller ignores specified axes that were previously homed.

The controller sets the Homed bit of the Axis Status item for axes that were previously homed. The controller sets the Homing bit of the Axis Status item while it is homing the axis and clears this bit when the homing completes or if an error occurs. Use the AXISSTATUS function to retrieve the Axis Status item.

Example

' Home the X and Y axes only if the axes have not been previously homed.
HOME X Y CONDITIONAL
