GALVO LASEROVERRIDE OFF Command

Commands the laser of the GL4 or the Nmark CLS, GCL, or SSaM to power off.

Syntax:

 
GALVO LASEROVERRIDE <GalvoAxis> OFF

Example:

 
GALVO LASEROVERRIDE X OFF

Table:     GALVO LASEROVERRIDE OFF Command Arguments

Argument

Type

Description

<GalvoAxis>

Axis

The first axis of the GL4 or the Nmark CLS, GCL, or SSaM.

Hardware: This command applies only when used with the GL4 or the Nmark CLS, GCL, or SSaM.

NOTE: Before you use the laser, you must configure all of the Galvo Laser Configuration commands.

Use the GALVO LASEROVERRIDE OFF command to power off the laser. This command lets you control the laser in the manual laser mode. If you issue this command during a move sequence, the controller synchronizes the laser with the motion.

To power on the laser, issue the GALVO LASEROVERRIDE ON command. If you want the controller to control the laser in the automatic laser mode, issue the GALVO LASEROVERRIDE AUTO command. For the laser output to activate, you must enable all the axes of your galvo drive. Make sure to include axes that do not engage in marking motion.

Note: For this command to synchronize to the accurate start or end of a move, Aerotech recommends that you set the MotionUpdateRate parameter to 48 kHz.
