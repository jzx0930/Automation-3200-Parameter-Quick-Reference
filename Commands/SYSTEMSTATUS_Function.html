SYSTEMSTATUS Function

Gets the specified system status item.

Syntax:

 
<return> = SYSTEMSTATUS(<DataItem>)
 
<return> = SYSTEMSTATUS(<DataItem>, <AdditionalData>)

Example:

 
$global[1] = SYSTEMSTATUS(DATAITEM_EstimatedProcessorUsage)

Table:     SYSTEMSTATUS Function Arguments

Argument

Type

Description

<DataItem>

Other

Specifies the system status to retrieve.

<AdditionalData>

32-bit signed integer

(Optional) Additional data for the specified data item. This argument is required by some data items.

RETURNS

64-bit IEEE floating point

The value of the specified system status item.

Use the SYSTEMSTATUS function to read values from the controller. The following table shows the values that you can read.

For some <DataItem> values, you can supply an integer as an optional argument. If you can use an optional argument with the <DataItem> value, the Additional Data column gives information about the argument.
