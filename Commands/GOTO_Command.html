GOTO Command

Transfers program execution to a specified label.

Syntax:

 
GOTO <Label>

Example:

 
GOTO setIO

Table:     GOTO Command Arguments

Argument

Type

Description

<Label>

Label

The label to transfer program execution to.

Use the GOTO command to transfer execution to a specified label in the current program. The label location can come into view anywhere in the program, before or after the line containing the GOTO. There is no limit to the number of GOTO commands that reference the same label, but a label can be defined (using the colon) only once in a program.

Do not execute the GOTO command to a label contained within a subroutine IF command, REPEAT command, or WHILE command block.  

Use the FARGOTO command to transfer to locations in other programs.

Example

GOTO setIO
' Because of the GOTO command that comes before this command, the controller does not execute the line that follows.
END PROGRAM

:setIO
$BO[1] = 1
END PROGRAM
