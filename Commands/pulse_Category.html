Pulse Command Category

Use the commands in this category to generate an output signal that is in proportion to the speed of one or more axes. The output signal can be quadrature, clock and direction, or an internal signal that is configured as an input to PSO.

When you configure the pulse output to use one input axis, the controller will calculate the speed and direction of the input axis. When you configure the pulse output to use multiple input axes, the controller will calculate a vector sum of the commanded speed of the input axes, without a direction signal. The controller then sends the pulse output value in encoder counts to the specified output axis at an 8kHz rate.

Pulse commands can be used to do the tasks that follow.

  • Drive a clock and direction stepper motor.
  • Pulse a laser input signal that is in proportion to the velocity command of an axis or axes.
  • Perform PSO operations based on the position or velocity of one or more axes, including virtual axes.

Refer to the PULSE SET OUTPUT Command for more information about supported drive configurations and how to use the pulse output mode.

Commands

  • PULSE ON/OFF
  • PULSE SET OUTPUT
