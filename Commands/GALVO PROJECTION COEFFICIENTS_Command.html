GALVO PROJECTION COEFFICIENTS Command

Specifies the projective transformation coefficients that are applied to the two galvo axes of the GL4 or the Nmark GCL.

Syntax:

 
GALVO PROJECTION <GalvoAxis> COEFFICIENTS <A>, <B>, <C>, <D>, <E>, <F>, <G>, <H>, <I>

Example:

 
GALVO PROJECTION X COEFFICIENTS 0.5, 0.1, 0.05, 0.89, 1.4, 6.7, 2.2, 7.3, 1.0

Table:     GALVO PROJECTION COEFFICIENTS Command Arguments

Hardware: This command applies only when used with the GL4 or the Nmark GCL.

Use the GALVO PROJECTION COEFFICIENTS command to set the projective transformation coefficients that are applied on the GL4 or the Nmark GCL. After you specify the coefficients, use the GALVO PROJECTION ON/OFF Command to enable and disable the transformation.
