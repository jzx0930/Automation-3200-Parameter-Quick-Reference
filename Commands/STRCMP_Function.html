STRCMP Function

Determines if two strings are the same.

Syntax:

 
<return> = STRCMP(<String1>, <String2>, <Case>)

Example:

 
$global[0] = STRCMP("string", "String", 1)

Table:     STRCMP Function Arguments

Argument

Type

Description

<String1>

String

First string to compare.

<String2>

String

Second string to compare.

<Case>

32-bit signed integer

Determines if the comparison is case sensitive. Specify 1 for case sensitive, specify 0 for case insensitive.

RETURNS

32-bit signed integer

Returns 0 if the two strings are the same. Returns 1 if the two strings are different.

Use the STRCMP function to compare two strings. If the two strings are the same, the STRCMP function will return 0. Otherwise, it will return 1.
