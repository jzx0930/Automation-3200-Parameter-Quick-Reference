GALVO LASERMODE Command

Specifies the mode in which the laser output signals operate.

Syntax:

 
GALVO LASERMODE <GalvoAxis> <Mode>

Example:

 
GALVO LASERMODE X 0

Table:     GALVO LASERMODE Command Arguments

Argument

Type

Description

<GalvoAxis>

Axis

The first axis of the GL4 or the Nmark CLS, GCL, or SSaM.

<Mode>

64-bit IEEE floating point

The integer value of the laser output mode. To see the possible values, refer to Table: <Mode> Argument Values.

Hardware: This command applies only when used with the GL4 or the Nmark CLS, GCL, or SSaM.

Warning: Do not issue the GALVO LASERMODE command when the laser is being commanded to fire.

Read Galvo Laser Configuration before you configure this command. To set or change the laser configuration, you must complete the procedure in Galvo Laser Configuration for this command to have an effect.

Use the GALVO LASERMODE command to select the mode in which the laser output signals operate. When the laser is being commanded to fire, O3 is active. When the laser is not being commanded to fire, O3 is not active. To set the active polarity (for RS-422 outputs) and transistor state (for opto-isolated outputs) of the laser output signals, use the switch on the GL4 or the Nmark CLS, GCL, or SSaM. For more information, refer to the hardware manuals for your drives.

The table that follows shows the possible values for the <Mode> argument. Before you issue the GALVO LASERMODE command, you must configure the applicable commands in the Related Commands column of the table. Refer to the procedure in Galvo Laser Configuration.

Table:     <Mode> Argument Values

Value Typical Application Description Related Commands

0

CO2 laser

When the laser is firing, O1 and O2 are modulation signals with a configurable frequency and a fixed 180° phase shift. You can independently specify the O1 and O2 pulse widths. When the laser is not firing and you use the GALVO STANDBYPERIOD and GALVO STANDBYPULSEWIDTH commands to specify a standby period and pulse width, O1 and O2 output standby pulses. When the laser is not firing and you do not specify a standby period and pulse width, O1 and O2 are not active.

YAG laser

O2 is a suppression pulse that is output when the laser starts firing. When the laser is firing, O1 is a periodic signal with a configurable frequency and pulse width. When the laser is not firing, O1 and O2 are not active.

YAG laser

O2 is a suppression pulse that is output when the laser starts firing. O1 does not become active until the suppression pulse stops. While the laser is firing, O1 is a periodic signal with a configurable frequency and pulse width. When the laser is not firing, O1 and O2 are not active.

YAG laser

O2 is a suppression pulse that is output when the laser starts firing. 10us after the start of the suppression pulse, O1 becomes active. When the laser is firing, O1 is a periodic signal with a configurable frequency and pulse width. When the laser is not firing, O1 and O2 are not active.

4

General purpose

O1 and O2 are standby signals with a configurable frequency and pulse width. They have a fixed 180° phase shift. When the laser is firing, O1 and O2 are active. When the laser is not firing, O1 and O2 are still active.
