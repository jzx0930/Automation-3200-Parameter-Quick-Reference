PROGRAM START Command

Runs an AeroBasic program that is already associated with a task.

Syntax:

 
PROGRAM <Task> START

Example:

 
PROGRAM 4 START

Table:     PROGRAM START Command Arguments

Argument

Type

Description

<Task>

TaskID

The task on which to run the program.

Use the PROGRAM START command to start running a program that is already added to the controller and associated with the specified task. This command puts the task into the Program Running Task State. You can use this command to continue running a program that is in the Program Paused or Program Feedheld Task State. You can also use this command to restart a program that is in the Program Complete Task State.

To add a program to the controller and associate it with a task, issue the PROGRAM LOAD command. As an alternative, you can issue the ADDPROGRAM command followed by the PROGRAM ASSOCIATE command.
