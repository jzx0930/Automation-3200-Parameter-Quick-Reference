DataAcquisition Command Category

Hardware: The DATAACQ commands do not apply when used with the Ndrive FCL, FLS, QL, or QLe, the Nmark SSaM, or the Nstep.

Use the DATAACQ commands to collect encoder positions or analog inputs and save them to the drive array. You can use an external signal or a PSO firing event signal to collect the data. These commands define the source of the data acquisition event and the data that you capture. The captured data can be a maximum of 32-bits of information.

You can read the number of collected data elements by using the DRIVEINFO function. For an example program, refer to the End-Point Modification Example.

Note: You cannot use the DATAACQ commands during homing.

Note: Because the DATAACQ ARRAY command enables data acquisition, you must issue the DATAACQ TRIGGER and DATAACQ INPUT commands before using the DATAACQ ARRAY command.
