MOVETOLIM Command

Moves an axis into a limit condition.

Syntax:

 
MOVETOLIM <Axis> CW
 
MOVETOLIM <Axis> CCW

Example:

 
MOVETOLIM X CW

Table:     MOVETOLIM Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

Use the MOVETOLIM command to move an axis into a limit condition. The CW or CCW keyword that you use controls the direction of the move.

This is an asynchronous command which means that program execution continues to the next program line without waiting for the move to complete. Use the WAIT MOVEDONE Command or WAIT INPOS Command to wait for motion to complete.

For Piezo Drives

The MOVETOLIM command moves an axis until one of the following conditions occurs.

  • The drive detects a hardware end-of-travel limit.
  • The Piezo Voltage Command exceeds the value that you specified for the PiezoVoltageClampLow Parameter or the PiezoVoltageClampHigh Parameter. The direction of the move determines which parameter the Piezo Voltage Command exceeds.

Configure the HomeSpeed Parameter to specify the speed of the move. The acceleration and deceleration ramp types are always linear. When the drive detects a limit condition, it stops immediately. The HomeRampRate parameter does not have an effect.

For All Other Drives

The MOVETOLIM command moves an axis until the drive detects a hardware end-of-travel limit.

Specify the speed of the move and the acceleration and deceleration rates by configuring the parameters that follow:

HomeSpeed Parameter specifies the speed of the move.

HomeRampRate Parameter specifies the acceleration and deceleration rates of the move. The acceleration and deceleration ramp types are always linear.
