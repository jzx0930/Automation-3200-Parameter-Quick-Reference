THERMOCOMP UPDATE Command

Pauses, resumes, or triggers an automatic update of thermal compensation (THERMOCOMP).

Syntax:

 
THERMOCOMP UPDATE ONCE
 
THERMOCOMP UPDATE RESUME
 
THERMOCOMP UPDATE PAUSE

Example:

 
THERMOCOMP UPDATE RESUME

Optional Purchase Required: The information in this topic applies only if you purchased the ThermoComp™ feature on your Aerotech positioning product.

The THERMOCOMP UPDATE command manually updates thermal compensation for all of the calibration tables configured for ThermoComp™. The sections that follow supply information about each keyword.

ONCE

The ONCE keyword updates the thermal compensation one time on all of the calibration tables.

PAUSE

The PAUSE keyword pauses the automatic updates to the calibration tables that are specified by the <TableNumber> argument of the THERMOCOMP CONFIG UPDATEINTERVAL Command. While ThermoComp is paused, the compensation value stays the same and does not change relative to the sensor readings.

RESUME

The RESUME keyword starts the automatic updates again for only the calibration tables that are configured to update automatically. After the controller executes the THERMOCOMP UPDATE RESUME command, ThermoComp reads the sensor values and updates the thermal compensation value relative to the sensor readings. To update tables that are configured to update manually, the controller must execute the THERMOCOMP UPDATE ONCE command.

WARNING: If the thermal state of the stage changed since the controller executed the THERMOCOMP UPDATE PAUSE command, the THERMOCOMP UPDATE RESUME command can cause position step changes.
