THERMOCOMP CONFIG MODE Command

Configures the mode of operation for thermal compensation (THERMOCOMP) on a specified axis calibration table number and specifies the source of the analog input.

Syntax:

 
THERMOCOMP CONFIG MODE <TableNumber>, <Mode>, <AnalogAxisIndex>, <AnalogChannel>

Example:

 
THERMOCOMP CONFIG MODE 1, 0, X, 2

Table:     THERMOCOMP CONFIG MODE Command Arguments

Argument

Type

Description

<TableNumber>

32-bit signed integer

The zero-based number of the axis calibration table. The axes specified in this table are the axes to which thermal compensation is applied.

<Mode>

32-bit signed integer

The operation mode dictated by the type of stage. Specify 0 for Linear Stages and 1 for Ball Screw Stages.

<AnalogAxisIndex>

Axis

The axis from which to read the thermal state of the stage. This axis specifies the source of the analog input used to monitor temperature.

<AnalogChannel>

32-bit signed integer

Specifies the analog input number to which the sensor is connected.

Optional Purchase Required: The information in this topic applies only if you purchased the ThermoComp™ feature on your Aerotech positioning product.

The THERMOCOMP CONFIG MODE command specifies the type of stage to which ThermoComp™ is applied.

The table that follows shows the possible values for the <Mode> argument.

Table:     <Mode> Argument Values

Value Description

0

Specifies a linear stage.

1

Specifies a ball-screw stage.

The <AnalogChannel> argument specifies the analog input number of the axis to which the sensor that reads the thermal state of the stage environment is connected on the drive.

Advanced Configurations

NOTE: This information is supplied for advanced users.

Aerotech fully configured these commands at the factory with coefficients that are specific to your system. You can issue the THERMOCOMP ON/OFF Command without making more changes to these commands. But if you want to customize your system more, you can edit the arguments of these commands before you issue the THERMOCOMP ON command.

  • THERMOCOMP CONFIG MODE Command
  • THERMOCOMP CONFIG UPDATEINTERVAL Command
  • THERMOCOMP CONFIG VALUES Command
