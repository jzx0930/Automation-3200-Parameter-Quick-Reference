DRIVEINFO Function

Gets the specified drive info item from the specified axis.

Syntax:

 
<return> = DRIVEINFO(<Axis>, <DriveInfoItem>)
 
<return> = DRIVEINFO(<Axis>, <DriveInfoItem>, <AdditionalData>)

Example:

 
$global[5] = DRIVEINFO(X, DRIVEINFO_PSOWindow2)

Table:     DRIVEINFO Function Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<DriveInfoItem>

32-bit signed integer

The DRIVEINFO_ constant that represents the requested drive info item. Possible values are listed in the DRIVEINFO Data Items table.

<AdditionalData>

32-bit signed integer

(Optional) Additional data for the specified data item. This argument is required by some data items.

RETURNS

64-bit IEEE floating point

The value of the specified drive info item.

Use the DRIVEINFO function to read values from the drive. The following table shows the values that you can read.

For some <DriveInfoItem> values, you can supply an integer as an optional argument. If you can use an optional argument with the <DriveInfoItem> value, the Additional Data column gives information about the argument.
