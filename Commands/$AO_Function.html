$AO Function

Accesses the value of a specified analog output.

Syntax:

 
<return> = $AO[<Channel>].<Axis>
 
$AO[<Channel>].<Axis> = <Value>

Example:

 
$AO[1].X = 5.2
$global[0] = $AO[0].X

Table:     $AO Function Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<Channel>

32-bit signed integer

The analog channel to access the value of.

<Value>

64-bit IEEE floating point

(Optional) The value to set the specified analog output to.

RETURNS

64-bit IEEE floating point

(Optional) The value of the specified analog output.

Use the $AO function to set a drive analog output. On most drives you can set analog outputs in the range of -10 to +10 volts. On some drives the range is smaller. For more information about analog output ranges, refer to the hardware manual for your drive.

The <Channel> argument specifies which output to set, with the first output being number 0. The <Axis> argument specifies the axis on which the output will be set.

For information about how I/O is mapped to different drive types, refer to Specifying I/O Numbers.

Note: When you set a digital or analog output on a virtual axis, the corresponding input is set to the same value. For example, if you set $WO[0].X = 45, then $WI[0].X becomes 45.
