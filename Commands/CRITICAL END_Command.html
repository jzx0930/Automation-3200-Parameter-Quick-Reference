CRITICAL END Command

Ends a critical section on the executing task.

Syntax:

 
CRITICAL END

Example:

 
CRITICAL END

Use the CRITICAL END command to stop a critical section that started with a CRITICAL START command. When the controller executes a CRITICAL END command, the executing task exits the critical section and goes back to normal task scheduling. Refer to Task Scheduling Overview for more information.

If the executing task is not in a critical section while the controller executes a CRITICAL END command, the command has no effect on the task.
