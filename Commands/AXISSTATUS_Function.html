AXISSTATUS Function

Gets the specified status item from the specified axis.

Syntax:

 
<return> = AXISSTATUS(<Axis>, <DataItem>)
 
<return> = AXISSTATUS(<Axis>, <DataItem>, <AdditionalData>)

Example:

 
$global[1] = AXISSTATUS(X, DATAITEM_PositionError)
$global[1] = AXISSTATUS(%$strglob[0]%, DATAITEM_VelocityError)

Table:     AXISSTATUS Function Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<DataItem>

Other

The DATAITEM_ constant that represents the axis status item. Possible values are listed in the AXISSTATUS Data Items Table.

<AdditionalData>

32-bit signed integer

(Optional) Additional data for the specified data item. This argument is required by some data items.

RETURNS

64-bit IEEE floating point

The value of the specified axis status item.

Use the AXISSTATUS function to read a value from the controller. The following table shows the values that you can read.

For some <DataItem> values, you can supply an integer as an optional argument. If you can use an optional argument with the <DataItem> value, the Additional Data column gives information about the argument.

Unless specified in the table, position values do not include calibration or backlash, and position is not rolled over. POSOFFSET and HomePositionSet offsets are included in the values unless specified differently.

Coordinated Distance Remaining

DATAITEM_CoordinatedDistanceRemaining

User Units

This item returns the distance remaining in a coordinated motion.

The distance remaining is defined as the coordinated target position minus the program position. The program position reflects the position specified in the program and does not include the output of any transformations.

This item is valid only during coordinated motion. Its value is zero for all other types of motion.
