Data Collection Command Category

Use the DATACOLLECT commands to configure and trigger data collection from an AeroBasic program without the use of Digital Scope. A Windows thread monitors the data collection, uploading blocks of data from the SMC as they are collected. When the collection is complete, the Windows thread writes the data to an ASCII text file. The resulting file can be read at a later time, and displayed using Digital Scope.

Note: Digital Scope will not correctly open files created by DATACOLLECT unless the file has a .DAT file extension.

Table:     DATACOLLECT Commands

Erase prior DATACOLLECT configuration

 
DATACOLLECT ITEM RESET

Specify a data item to collect

 

DATACOLLECT ITEM <ItemNumber>, <Index>, <DataItem>

Start data collection

 

DATACOLLECT START <FileHandle>, <NumSamples>, <SampleTime>

Stop data collection

 
DATACOLLECT STOP

Use the DATACOLLECT ITEM command to specify the data items to collect, and DATACOLLECT START to begin collecting.

You must use the DATACOLLECT STOP command to terminate the data collection. Use the FILEOPEN function and FILECLOSE command in conjunction with DATACOLLECT START and DATACOLLECT STOP to open and close the file to which the data samples are written.

Data collection is real time and the intervals between samples in one DATACOLLECT START execution are deterministic. The time between when DATACOLLECT START is issued and the first data point occurs, and the time between the last data point collected and when the DATACOLLECT STOP is issued, is not completely deterministic. This is because the start and stop operations are performed from a Windows thread and have a variability of 10 milliseconds or more. Note that the DATAACQ command can be used for hardware-based triggering of encoder positions for more precise position capturing.

To append data to the end of an existing output file, either start another acquisition with DATACOLLECT START before closing the file or reopen an existing data file with FILEOPEN mode 2. The latency between successive collections is not deterministic because file operations are performed by Windows.

For more information about Digital Scope, refer to Digital Scope Application Overview.
