SERVO ON/OFF Command

Changes between open-loop and closed-loop mode for piezo stages.

Syntax:

 
SERVO <Axis> ON
 
SERVO <Axis> OFF

Example:

 
SERVO X ON 
SERVO X OFF

Table:     SERVO ON/OFF Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

Hardware: This command applies only when used with the Ndrive QL or QLe.

Use the SERVO ON/OFF command to change the mode of the servo loop. You can use the PiezoDefaultServoState parameter to configure which mode is active when you reset the controller.

The SERVO ON command sets the controller to run in closed-loop mode. In closed-loop mode, the controller uses position feedback to correct the output of the servo loop. For more information about the parameters that have an effect on closed-loop mode, refer to Servo Loop Block Diagram and Servo Loop Parameter Category.

The SERVO OFF command sets the controller to run in open-loop mode. In open-loop mode, the servo loop does not use position feedback and the controller generates the output of the servo loop only through the PiezoVoltsPerUnit parameter. If position feedback is available, the controller reads the feedback as if the controller is in SERVO ON mode. But, error between the position feedback and position command does not have an effect on the output of the servo loop.

When you change between open-loop and closed-loop mode, the controller updates the position command so that the output of the servo loop does not change.
