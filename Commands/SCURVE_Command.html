SCURVE Command

Specifies the S-curve value to use.

Syntax:

 
SCURVE <Value>

Example:

 
SCURVE 0
SCURVE 50

Table:     SCURVE Command Arguments

Argument

Type

Description

<Value>

64-bit IEEE floating point

The value by which to S-curve.

NOTE: This command applies only to coordinated motion (for example, LINEAR, CW, CCW, and BEZIER commands).

Use the SCURVE command to set the percentage that becomes curved, of the ramping part of the coordinated velocity command. This occurs only if you set the RAMP TYPE to SCURVE. The SCURVE command has an effect on the acceleration and deceleration of coordinated motion. You can also use the DefaultSCurve parameter to set this percentage value.

The <Value> argument has a range of 0 to 100. The following velocity profiling figures show only the acceleration ramp. But, the percentage values that you set for <Value> have an effect on the acceleration ramp and the deceleration ramp.

  • If you set <Value> to 0%, all of the ramping part of the velocity profile is linear.
  • Figure: Velocity Profiling — SCURVE 0

  • If you set <Value> to 50%, the middle half of the ramping part of the velocity profile is linear. On both sides of the ramp, the other half of the ramping part of the velocity profile is curved.
  • Figure: Velocity Profiling — SCURVE 50

  • If you set <Value> to 100%, all of the ramping part of the velocity profile is curved. A curved velocity profile generates smoother axis motion by limiting the rate of change of the acceleration command.
  • Figure: Velocity Profiling — SCURVE 100

After you specify a value for the SCURVE command, use either the RAMP RATE command or the DefaultCoordinatedRampRate task parameter to specify the average acceleration and deceleration rate. During S-curve ramping, the peak acceleration (maximum acceleration) might exceed the average acceleration. If you increase the S-curve value, the increased value causes higher peak acceleration. For example if you set the S-curve to 50%, the peak acceleration is 1.33 times the specified average acceleration. Refer to the Acceleration and Deceleration Ramping Overview for more information.
