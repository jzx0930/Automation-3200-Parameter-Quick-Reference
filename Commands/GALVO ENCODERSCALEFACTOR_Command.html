GALVO ENCODERSCALEFACTOR Command

Enables "marking on the fly" functionality on the specified axis of a GL4 or an Nmark CLS, GCL, or SSaM.

Syntax:

 
GALVO ENCODERSCALEFACTOR <GalvoAxis> <Value>

Example:

 
GALVO ENCODERSCALEFACTOR X 5.7

Table:     GALVO ENCODERSCALEFACTOR Command Arguments

Argument

Type

Description

<GalvoAxis>

Axis

The axis of the GL4 or the Nmark CLS, GLC, or SSaM on which to execute the command.

<Value>

64-bit IEEE floating point

The ratio of scanner counts to encoder counts. This value must be greater than -32,768 and less than 32,767.

Hardware: This command applies only when used with the GL4 or the Nmark CLS, GCL, or SSaM.

Use the GALVO ENCODERSCALEFACTOR command to enable "marking on the fly" functionality on the specified axis of the GL4 or the Nmark CLS, GCL, or SSaM. This feature lets the drive monitor the position of an external encoder. The drive adjusts the position of the scanner to compensate for changes in the external position.

You can use "marking on the fly" functionality in many applications. For example, you might want to draw a pattern on a part that is continuously moving through the field of view of the scanner during the marking process. (The part might be on a conveyor belt.)

For the GL4 or the Nmark CLS, GCL, or SSaM to correctly adjust the position of the scanner, you must specify the ratio of scanner counts to encoder counts. Specify this ratio in the <Value> field of this command. If encoder adjustment is being applied to two GL4 or Nmark CLS, GCL, or SSaM axes, then you must issue this command twice (one time for each <Axis>). If you need to change the <Value> command argument of the GALVO ENCODERSCALEFACTOR on the GL4 or the Nmark CLS, GCL, or SSaM, issue GALVO ENCODERSCALEFACTOR <Axis> 0 first. If you do not issue this command first, the controller step changes the position command, which causes a large current increase in the motor. You can use a different scale factor in each command. Specify a value greater than -32,768 and less than 32,767 for <Value>.

When the GALVO ENCODERSCALEFACTOR command executes, the external position counter for the specified axis clears and encoder adjustment starts at the current position of the scanner.

To deactivate "marking on the fly" on an axis, issue this command with a value of 0. If encoder adjustment is being applied to two GL4 or Nmark CLS, GCL, or SSaM axes, then you must issue this command two times, which is one time for each <Axis>.
