LOADCALFILE Command

Downloads and activates a specified axis calibration file or galvo power correction file.

Syntax:

 
LOADCALFILE <FileName>, <FileType>

Example:

 
LOADCALFILE "C:\MyFolder\MyCalFile.cal", 1D_CAL

Table:     LOADCALFILE Command Arguments

Argument

Type

Description

<FileName>

String

The name of the axis calibration file or galvo power correction file that you want to use.

<FileType>

Other

The type of axis calibration file or galvo power correction file that you are using. To see the possible values, refer to Table: <FileType> Argument Values.

Note: This command is executed as a callback and does not execute in real time. The time that it takes for this command to execute is different for different PC configurations. Refer to Callback Commands for more information.

Use the LOADCALFILE command to immediately download and activate the axis calibration file or galvo power correction file that is specified by the <FileName> argument. If you do not specify an absolute file path in the file name, the file path is relative to the program that is currently executing.

For axis calibration, galvo calibration, and galvo power correction files, use this command only if you must change calibration or galvo power correction data during program execution. If it is not necessary to change calibration or galvo power correction data during program execution, specify the calibration or galvo power correction files in the Calibration folder of Motion Composer or Configuration Manager. Refer to Configuring Axis Calibration or Galvo Power Correction Overview for more information.

Table:     <FileType> Argument Values

Value Description

1D_CAL

1D position calibration data.

2D_CAL

2D position calibration data.

GALVO_CAL_2D

2D Galvo calibration data. This uses the 2D Galvo Calibration Table Format and is used to load data to the GL4 or the Nmark CLS or GCL.

GALVO_CAL_GCAL(1)

Galvo GCAL calibration data. This uses the GCAL calibration file format and is used to load data to the Nmark SSaM. An <Axis> argument is required for this file type.

LOADCALFILE <FileName>, GALVO_CAL_GCAL, <Axis>
GALVO_POWERCORRECTION(2)

Galvo laser power correction data. This uses the Galvo Power Correction File Format and is used to load data to the GL4 or the Nmark CLS or GCL.

(1) You can use the GALVO_CAL_GCAL type keyword to load a galvo GCAL calibration file in the GCAL format only when you are using an Nmark SSaM.

(2) Galvo Power Correction is available only if you are using the GL4, the Nmark CLS with Rev. A or later, or the Nmark GCL.
