RAMP RATE Command

Specifies rate-based acceleration and deceleration.

Syntax:

 
RAMP RATE <Value>
 
RAMP RATE ACCEL <Value>
 
RAMP RATE DECEL <Value>
 
RAMP RATE <Axis> <Value> [ <Axis> <Value> ... ]
 
RAMP RATE ACCEL <Axis> <Value> [ <Axis> <Value> ... ]
 
RAMP RATE DECEL <Axis> <Value> [ <Axis> <Value> ... ]

Example:

 
RAMP RATE X 10
RAMP RATE ACCEL 5

Table:     RAMP RATE Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

<Value>

64-bit IEEE floating point

The acceleration and deceleration rate.

Use the RAMP RATE command to set acceleration and deceleration rates for axis-based motion and coordinated motion. You must specify the <Value> argument in user units/s2. If <Value> is zero, the controller uses instantaneous acceleration or deceleration ramping.

Note: The rate set with this command is only used when in rate-based ramping mode (RAMP MODE RATE).

Axis-Based Motion

Examples

' Set the X axis acceleration and deceleration rate to 10 user units/s2.
RAMP RATE X 10
' Set the X axis acceleration rate to 5 user units/s2.
RAMP RATE ACCEL X 5
' Set the X axis deceleration rate to 5 user units/s2 and the Y axis deceleration rate
' to 7 user units/s2.
RAMP RATE DECEL X 5 Y 7

If you specify an axis in the RAMP RATE command, the specified acceleration and deceleration rates will be applied to axis-based (non-coordinated) motion on that axis. You can set different rates on different axes with the same command.

To set only the acceleration rate on a given axis, use the RAMP RATE ACCEL <Axis> command. To set only the deceleration rate, use the RAMP RATE DECEL <Axis>command.

Coordinated Motion

Examples

' Set the X axis acceleration and deceleration rate for coordinated motion to 10 user units/s2.
RAMP RATE 10

' Set the acceleration rate for coordinated motion to 5 user units/s2.
RAMP RATE ACCEL 5
' Set the deceleration rate for coordinated motion to 7 user units/s2.
RAMP RATE DECEL 7

If you do not specify an axis in the RAMP RATE command, the specified acceleration and deceleration rates will be applied to coordinated motion. In a coordinated move, individual axis accelerations will be equal to or less than the specified rates.

To set only the acceleration rate for coordinated motion, use the RAMP RATE ACCEL command. To set only the deceleration rate, use the RAMP RATE DECEL command.

The RAMP RATE command sets the acceleration and deceleration rates for both dominant and dependent axes. To set the acceleration and deceleration rate for just dependent axes, use the RAMP RATE DEPENDENT command.

Default Ramp Rates

The DefaultCoordinatedRampRate task parameter sets the default coordinated acceleration and deceleration rates. The DefaultRampRate axis parameter sets the default axis-based acceleration and deceleration rates.
