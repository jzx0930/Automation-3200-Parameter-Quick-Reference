RAND Function

Generates a pseudorandom value.

Syntax:

 
<return> = RAND(<Seed>)
 
RAND(<Seed>)

Example:

 
RAND(5)
$global[0] = RAND(0)

Table:     RAND Function Arguments

Argument

Type

Description

<Seed>

32-bit signed integer

The seed for the random-number generation. If you specify a nonzero value, the pseudorandom number generator is reseeded with that value.

RETURNS

64-bit IEEE floating point

(Optional) A random value in the range of 0.0 to 1.0.

Use the RAND function to control the pseudorandom number generator. It generates one of 32,768 unique values in the range of 0.0 to 1.0. The table that follows shows the possible values for the <Seed> argument.

Table:     <Seed> Argument Values

<Seed> Description

0

Returns the next pseudorandom number in the sequence of random values.

Nonzero Value

Seeds the pseudorandom number generator.

For each <Seed> argument to which you specify the same value, the pseudorandom number generator generates the same sequence of values. When you reset the controller, the controller reseeds the pseudorandom number generator with a seed of 1.

Each task uses a different pseudorandom number generator.

Example

' Seed the pseudorandom number generator.
RAND(TIMER(0))
				
' Read the next value from the pseudorandom number generator.
$global[0] = RAND(0)
