AXCALPARM Function

Modifies the axis calibration parameters.

Syntax:

 
AXCALPARM <AxisCalTableNumber>, <Mode>, <ParameterValue>
 
<return> = AXCALPARM <AxisCalTableNumber>, <Mode>

Example:

 
AXCALPARM 1, AXISCAL_CURRENT_MATERIAL_TEMP, 22
$Var = AXCALPARM 1, AXISCAL_EXPAND_COEFF

Table:     AXCALPARM Function Arguments

Argument

Type

Description

<AxisCalTableNumber>

Other

The name of the 1D calibration table to use.

<Mode>

String

The axis calibration value.

<ParameterValue>

32-bit signed integer

(Optional) Specifies the value of the parameter.

RETURNS

64-bit IEEE floating point

(Optional) The correction offset that is read.

Note: This command applies only to 1D calibration.

Use this command to change the axis calibration parameters. Changes to calibration parameter values are not applied until you home the axis that serves as the master reference for the calibration table.

<AxisCalTableNumber> must refer to a 1D cal table.

Temperature Compensation Keywords

Use the following AXCALPARM keywords to change the axis calibration parameters for temperature compensation. Refer to Temperature Compensation for more information.

AXISCAL_EXPAND_COEFF

AXISCAL_CURRENT_EXPAND_COEFF

AXISCAL_CURRENT_MATERIAL_TEMP

Position Offset Keyword

Use the AXISCAL_CORRECTION_OFFSET keyword to apply a constant position offset as an axis calibration value. This offset can be set at run time from an AeroBasic program.

The units of AXISCAL_CORRECTION_OFFSET are in counts of the corrected (or calibration slave) axis. Refer to UNITSTOCOUNTS Function for more information.

After a successful home, all correction offset values for all tables will be reset to zero.

Warning: On a gantry, you must set the AXISCAL_CORRECTION_OFFSET to the same value on both the gantry slave and the gantry master, or you could damage the gantry.

Example: Using AXISCAL_CORRECTION_OFFSET Keyword

DVAR $tableNumber
DVAR $offset

$tableNumber=0

' Set the correction offset to three counts.
$offset = 3
AXCALPARM $tableNumber, AXISCAL_CORRECTION_OFFSET, $offset

' Read the correction offset into a global variable.
$global[4] = AXCALPARM $tableNumber, AXISCAL_CORRECTION_OFFSET
