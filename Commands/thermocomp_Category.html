ThermoComp Command Category

Thermal Compensation

Optional Purchase Required: The information in this topic applies only if you purchased the ThermoComp™ feature on your Aerotech positioning product.

The ThermoComp™ feature in the A3200 software applies physical calibration to stages when the thermal state of the stage environment changes. The software uses axis calibration tables to apply ThermoComp. The axes that are specified in the axis calibration tables are the axes to which ThermoComp is applied. For information about how to use axis calibration, refer to 1D Axis Calibration Overview.

The .pgm file that Aerotech supplied with your ThermoComp system correctly configures the ThermoComp feature. You can configure this file to run automatically so that ThermoComp is always active on your system.

To configure Program Automation for ThermoComp

  1. Open Configuration Manager.
  2. In the Network Explorer, expand the A3200 node. Expand Program Automation.
  3. Right-click Programs. Then select Add. The Program Automation dialog box comes into view.
  4. Click Browse. Find the ThermoComp file in [Install Directory]\A3200\User Files\.
  5. Select the ThermoComp file and click Open.
  6. In the Program Automation dialog box, in the Mode list, select Run Silent.
  7. Under Task, associate the file with an unused task.
  8. Click OK. The ThermoComp program file shows under the Programs node.

Your ThermoComp configuration .pgm file will now run after each controller reset.

In the configuration .pgm file, the calibration tables are configured with coefficients that are specific to your system.

Advanced Configurations

NOTE: This information is supplied for advanced users.

Aerotech fully configured these commands at the factory with coefficients that are specific to your system. You can issue the THERMOCOMP ON/OFF Command without making more changes to these commands. But if you want to customize your system more, you can edit the arguments of these commands before you issue the THERMOCOMP ON command.

  • THERMOCOMP CONFIG MODE Command
  • THERMOCOMP CONFIG UPDATEINTERVAL Command
  • THERMOCOMP CONFIG VALUES Command

Table:     ThermoComp Commands and Descriptions

Command Description

THERMOCOMP ON/OFF Command

Turns on or turns off ThermoComp.

THERMOCOMP CONFIG MODE Command

Specifies the mode of operation for a specified calibration table, and the analog input number of the sensor that reads the thermal state of the stage environment.

THERMOCOMP CONFIG RESET Command

Clears all the values that configure ThermoComp. The controller can execute this command only when ThermoComp is turned off.

THERMOCOMP CONFIG VALUES Command

Specifies all the values necessary for ThermoComp to apply thermal compensation accurately. A range can be set for the thermal state of the stage, specific to your system and environment, in which ThermoComp is applied.

THERMOCOMP CONFIG UPDATEINTERVAL Command

Specifies how frequently the thermal compensation is updated in the specified calibration table.

THERMOCOMP UPDATE Command

Triggers an immediate, one-time compensation update. Or, pauses or resumes the ThermoComp feature. This command can apply compensation at a specified point in a program.

During run time, ThermoComp can be turned on, turned off, paused, or resumed.
