MSGDISPLAY Command

Writes a message to the Messages tab.

Syntax:

 
MSGDISPLAY <Priority>, <ArgumentList>
 
MSGDISPLAY <Priority>, <FormatSpecifier>, <ArgumentList>

Example:

 
MSGDISPLAY 1, "Beginning motion"

Table:     MSGDISPLAY Command Arguments

Argument

Type

Description

<Priority>

Other

The priority that you want to assign to the message. You can clear the messages with a specified priority by using the MSGCLEAR command.

<FormatSpecifier>

String

(Optional) The characters that determine the format for each numerical value that is shown. For more information, refer to Argument Lists and Format Specifiers.

<ArgumentList>

Other

A mixture of comma-separated integer, double, and string arguments. The controller concatenates these arguments into one text string.

Use the MSGDISPLAY command to write a line of text into the Messages tab. You must issue an MSGDISPLAY command for each line of text that you want to show. If the display window already contains messages, the new message shows on the next line. The text does not wrap.

Use the <Priority> argument to specify a priority for each message. Then, you can clear all messages of a specified priority from the display by using the MSGCLEAR command. Do not specify the <Priority> argument as a value of -1 because you use -1 in the MSGCLEAR command to clear all of the messages from the tab.

Example

' Clear all messages.
MSGCLEAR -1
' Show data.
$global[0] = 1.23456789
$global[1] = 5.678 
$global[2] = 0 
$global[2] = $global[2] +1
' Show text message. 
MSGDISPLAY 1, "Beginning motion"
' Show two 2 values in the default format.
MSGDISPLAY 1, "Value 1 = ", $global[0], " Value 2 = ", $global[1]
' Use "C" type format specifiers.
MSGDISPLAY 1, "{%6.2f %5.3f}" "Value 1 = ", $global[0], "  Value 2 = ", $global[1]
' Use # format specifiers.
MSGDISPLAY 1, "{#F #D}" "Value 1 = ", $global[0], " Value 2 = ", $global[1]
' Show as integers.
MSGDISPLAY 1, "{#D #D}" "Value 1 = ", $global[0], " Value 2 = ", $global[1]
MSGDISPLAY 1, "{#D}" "Loop Count = ", $global[2] 
MSGDISPLAY 1, "{#F3 #F}" "Time is #TM, Float 3 decimals and default " $global[0] " "
$global[0]
MSGDISPLAY 1, "{#D #H}" "Integer and Hex " $global[0] " 0x" $global[0]
MSGDISPLAY 1, "{#D #U #H}" "Integer, Unsigned, and Hex " -1 " " -1 " 0x" -1

Note: Motion Composer, CNC Operator Interface, Motion Simulator, or custom application must be open for this command to work. By default, Motion Composer, CNC Operator Interface, and Motion Simulator all try to register for these callbacks. The first application that registers will get the callbacks. The next application that tries to register will get an error. You can change the default behavior in Preferences of each application.
