TASKSTATUS Function

Gets the specified status item from the specified task.

Syntax:

 
<return> = TASKSTATUS(<DataItem>)
 
<return> = TASKSTATUS(<Task>, <DataItem>)
 
<return> = TASKSTATUS(<Task>, <DataItem>, <AdditionalData>)

Example:

 
$global[0] = TASKSTATUS(DATAITEM_MFO)
$global[0] = TASKSTATUS($taskindex, DATAITEM_TaskMode, TASKMODE_Absolute)

Table:     TASKSTATUS Function Arguments

Argument

Type

Description

<Task>

TaskID

(Optional) The task from which to retrieve the status item. To retrieve the status item from the current task, use $taskindex as the value.

<DataItem>

Other

Specifies the task status to retrieve.

<AdditionalData>

32-bit signed integer

(Optional) Additional data for the specified data item. This argument is required by some data items.

RETURNS

64-bit IEEE floating point

The value of the specified task status item.

Use the TASKSTATUS function to read values from the controller. The following table shows the values that you can read.

To retrieve the status item from the current task, use $taskindex as the value for <Task>.

For some <DataItem> values, you can supply an integer as an optional argument. If you can use an optional argument with the <DataItem> value, the Additional Data column gives information about the argument.
