PSOOUTPUT PULSE LASER MASK EXTSYNC Command

Sets the PSO output mode.

Syntax:

 
PSOOUTPUT <Axis> PULSE LASER MASK EXTSYNC

Example:

 
PSOOUTPUT X PULSE LASER MASK EXTSYNC

Table:     PSOOUTPUT PULSE LASER MASK EXTSYNC Command Arguments

Argument

Type

Description

<Axis>

Axis

The axis on which to execute the command.

Hardware: This command applies only when used with the GL4 or the Nmark CLS or GCL.

Use the PSOOUTPUT PULSE LASER MASK EXTSYNC command to configure the PSO output to be generated from the PSO pulse generator and masked by the status of the laser output. You must first configure the pulse generator by issuing the PSOPULSE Command. Before the pulse generator can generate pulses, you must configure the PSODISTANCE Command so that the controller generates firing events.

If the galvo laser output is off after you issue this command, the PSO tracking counters are reset and held in reset so that no firing events can occur. When the laser output turns on, a firing event is generated and the PSO counters are taken out of reset and begin tracking. While the laser output remains on, firing events are generated based on the firing distance you specified for the PSODISTANCE command.

When a firing event triggers the pulse generator, the pulse generator waits to generate the pulse output until after the external synchronization signal transitions high. This applies to distance-based firing events that you configure with the PSODISTANCE Command. It also applies to the firing event that is generated when the laser output turns on. The table that follows shows the correct inputs for the external synchronization signal.

Table:     Inputs for the External Synchronization Signal

Drive Type Input

GL4

Pin 7 on the Galvo Motor Connector

Ndrive FCL

Pin 7 on Connector J103-7

Ndrive HLe, HPe, ML, and QLe

Auxiliary Sine

Ndrive FLS and Nmark CLS

Pins 6 and 19 on Connector J104

Nmark GCL

Pin 7 on Connector J103-7

Npaq MR with ML drives

Auxiliary Sine

XC4 and XC4e

Pins 3 and 4 on the AUX I/O connector

XL4s

Pin 7 on the Stage connector

XR3

Pins 2 and 3, 4 and 5, or 6 and 7 on the PSO connector

When a pulse sequence begins it generates to completion, even if the laser output turns off during the pulse sequence. The PSOOUTPUT PULSE LASER MASK EXTSYNC command does not configure the laser output. To configure the laser output, you must first use the commands in Galvo Command Category.
