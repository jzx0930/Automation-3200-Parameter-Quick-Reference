Unlocking Optional Features

The A3200 has many options that you can purchase separately. You can use Feature Manager to unlock some of these options for a trial period. During this trial period, you can access all of the features in each option that is available in Feature Manager, which can help you make a decision about the options that you need. If you have questions about how to use an option, contact Aerotech Global Technical Support. If you want to purchase an option, contact your Aerotech sales representative.

To Enable Feature Demonstration

  1. Open Feature Manager.
    1. Go to the BIN directory that is below the A3200 installation directory. (The following is the default path. C:\Program Files\Aerotech\A3200\Bin)
    2. Double-click FeatureManager.exe. The A3200 Feature Manager dialog comes into view.
  2. In the Demonstrate features section, type the Verification Code into the Enter Verification Code box.
  3. Click the Start Demonstration button. Feature Manager unlocks all of the Controller Options that are listed in the Options Available in Feature Manager table.

NOTE: When you click the Start Demonstration button for the first time, the Controller Options become available for you to use for a two week trial period. If you reset the controller, the controller disables the demonstration. During the trial period, you can restart the demonstration an unlimited number of times by clicking the Start Demonstration button.

If you want to review the Help files for the options that are available with Feature Demonstration, you must install them separately. After you install these files, they are integrated and available through the main A3200 Help file.

To Install the Optional Help Files

  1. Open Feature Manager. You must run Feature Manager as an Administrator to install the optional Help files.
  2. In the Install full versions of help files section, click the Install button.
