Configuring Parameters for an EnDat Absolute Encoder

Hardware: You can use an EnDat encoder with incremental signals only if you are using the Ndrive CP, HLe, or HPe, the XC2, the XC4, the XC4e, or the XR3 with one of the encoder multiplier options.

EnDat Encoder Interface with Aerotech Position Feedback

To configure the parameters to use the EnDat encoder interface with Aerotech Position Feedback, use the Axis Parameter Calculator. For more information, refer to Using Calculators to Set Parameters.

If the stage does not have hardware limits, you must configure the applicable Software Limits. For more information, refer to Limits Overview.

EnDat Encoder Interface on a Custom Stage

To configure the parameters to use the EnDat encoder interface on a custom stage, you must know this information about the encoder.

  • Is the encoder linear or rotary?
  • Does the encoder supply incremental signals?

Note: In the EnDatEncoderSetup Parameter, if you set the Commutation Initialization setting to Use Encoder Absolute Position, then you must determine the commutation offset of the EnDat encoder. Use Encoder Absolute Position is the default setting. Refer to CommutationOffset Parameter for more information.

Use the applicable procedure for the encoder.

Linear Encoder with Incremental Signals

For this configuration, you must know the information that follows.

  • What is the absolute position resolution (the width of the serial communication word) in bits?
  • What is the resolution of the incremental feedback signals (the number of lines per revolution)?

To configure the parameters

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

NOTE: In the EnDatEncoderSetup Parameter, if you set the Encoder Configuration setting to Use Manufacturer Settings, then ignore steps 1, 2, 3, and 4. Start with step 5 to set the EncoderMultiplicationFactor parameter.
  1. Set the Encoder Type setting of the EnDatEncoderSetup Parameter to Linear.
  2. Set the Feedback Mode setting of the EnDatEncoderSetup Parameter to Use Incremental Signals. This is the default setting.
  3. Set the value of the EnDatEncoderResolution Parameter to the absolute position resolution in bits. For example, type 32 for a 32-bit absolute encoder.
  4. Set the value of the EnDatEncoderIncrementalResolution Parameter to the ratio of the incremental resolution to the absolute position resolution.

    5. For Example

    If you are using an encoder with a 20 µm resolution for the incremental signals and a 10 nm resolution for the absolute position, set this parameter to 2,000, which is .

  5. To specify the effective resolution of the axis, set the EncoderMultiplicationFactor Parameter to an integer multiple of the value of the EnDatEncoderIncrementalResolution Parameter.
  6. Open the Axis Parameter Calculator by selecting Tools → Axis Parameter Calculator. Then do the steps that follow. Refer to Using Calculators to Set Parameters for more information.
    1. For the Position Feedback, select Other Encoder.
    2. Select the MX box.
    3. For the Resolution (mm/cycle), enter the resolution of the incremental feedback signals that are supplied by the encoder.
    4. For the Multiplication Factor, enter the same value that you used to set the EnDatEncoderIncrementalResolution Parameter in Step 4.
    5. Configure the remaining axis parameters that are applicable to your project.
    6. Click OK to accept the changes and close the calculator.
  7. Change the PositionFeedbackType Parameter to EnDat Absolute Encoder.
  8. To use the absolute position information to initialize the motor commutation, you must determine the correct setting for the CommutationOffset Parameter.
  9. Click the Save Parameter File button to save all of your changes to the parameter file.
Linear Encoder without Incremental Signals

For this configuration, you must know the information that follows.

  • What is the absolute position resolution (the width of the serial communication word) in bits?

To configure the parameters

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

NOTE: In the EnDatEncoderSetup Parameter, if you set the Encoder Configuration setting to Use Manufacturer Settings, ignore steps 1, 2, and 3. Start with step 4 to open the Axis Parameter Calculator.
  1. Set the Encoder Type setting of the EnDatEncoderSetup Parameter to Linear.
  2. Set the Feedback Mode setting of the EnDatEncoderSetup Parameter to Use Absolute Position Only.
  3. Set the value of the EnDatEncoderResolution Parameter to the absolute position resolution in bits. For example, type 32 for a 32-bit absolute encoder.
  4. Open the Axis Parameter Calculator by selecting Tools → Axis Parameter Calculator. Then do the steps that follow. Refer to Using Calculators to Set Parameters for more information.
    1. For the Position Feedback, select Other Encoder.
    2. For the Resolution (mm/count), enter a value based on the absolute position resolution of the encoder.
    3. Configure the remaining axis parameters that are applicable to your project.
    4. Click OK to accept the changes and close the calculator.
  5. Change the PositionFeedbackType Parameter to EnDat Absolute Encoder.
  6. To use the absolute position information to initialize the motor commutation, you must determine the correct setting for the CommutationOffset Parameter.
  7. Click the Save Parameter File button to save all of your changes to the parameter file.
Rotary Encoder with Incremental Signals

For this configuration, you must know the information that follows.

  • What is the absolute position resolution (the width of the serial communication word) in bits?
  • What is the number of absolute encoder turns per motor revolution?
  • What is the resolution of the incremental feedback signals (the number of lines per revolution)?

To configure the parameters

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

NOTE: In the EnDatEncoderSetup Parameter, if you set the Encoder Configuration setting to Use Manufacturer Settings, ignore steps 1, 2, 3, 4, and 5. Start with step 6 to set the EncoderMultiplicationFactor parameter.
  1. Set the Encoder Type setting of the EnDatEncoderSetup Parameter to Rotary. This is the default setting.
  2. Set the Feedback Mode setting of the EnDatEncoderSetup Parameter to Use Incremental Signals. This is the default setting.
  3. Set the value of the EnDatEncoderResolution Parameter to the absolute position resolution in bits for a single turn of the encoder.

    4. For Example

    For a single-turn or multi-turn absolute encoder that has 8192 positions per revolution, type 13.

  4. For multi-turn rotary encoders only, set the EnDatEncoderTurns Parameter to the number of absolute encoder turns per motor revolution.

    5. For Example

    For a multi-turn encoder that has 4096 distinguishable revolutions, type 4096.

  5. Set the EnDatEncoderIncrementalResolution Parameter to the incremental resolution of the encoder in lines per revolution.
  6. Use the equation that follows to calculate the correct minimum value for the EncoderMultiplicationFactor Parameter.

  7. Specify the effective resolution of the axis by setting the EncoderMultiplicationFactor Parameter to an integer multiple of the value you calculated in Step 6.
  8. Open the Axis Parameter Calculator by selecting Tools → Axis Parameter Calculator. Then do the steps that follow. Refer to Using Calculators to Set Parameters for more information.
    1. For the Position Feedback, select Other Encoder.
    2. Select the MX box.
    3. For the Resolution (counts/rev), enter the resolution of the incremental feedback signals that are supplied by the encoder.
    4. For the Multiplication Factor, enter the same value that you used to set the EncoderMultiplicationFactor Parameter in Step 7.
    5. Configure the remaining axis parameters that are applicable to your project.
    6. Click OK to accept the changes and close the calculator.
  9. Change the PositionFeedbackType Parameter to EnDat Absolute Encoder.
  10. To use the absolute position information to initialize the motor commutation, you must determine the correct setting for the CommutationOffset Parameter.
  11. Click the Save Parameter File button to save all of your changes to the parameter file.
