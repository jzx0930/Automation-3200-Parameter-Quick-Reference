About the Automation 3200

The A3200™ software-based motion controller supplies 1 to 32 axes of coordinated motion and uses FireWire (IEEE-1394) or HyperWire™ to network with intelligent drives. These drives use dedicated processors to digitally close current and servo loops.

User programs can control motion, I/O, or general background tasks. The standard A3200 controller runs a maximum of four user programs at the same time. With the Professional Option, you can run 1 to 32 user programs at the same time.

The A3200 controller runs under Windows operating systems in a real-time kernel. This kernel runs independently of other programs that are running on the PC. You can write custom applications that run on the PC in the MATLAB, C, .NET, and LabVIEW programming languages.

Software Architecture

The A3200 is a layered software product. Each layer gets instructions from the next higher layer and sends commands down to the next lower layer. The A3200 handles real-time activities at the lower levels and handles less time-critical activities at the higher levels. Refer to System Architecture for more information.
