Using the AbsoluteFeedbackOffset Calculator

Use the AbsoluteFeedbackOffset calculator to calculate the correct value for the AbsoluteFeedbackOffset Parameter on piezo stages. The calculator is available only when you are using a piezo stage with a capacitance sensor and you are working with the active parameter file.

The AbsoluteFeedbackOffset Calculator calculates an AbsoluteFeedbackOffset parameter value that centers the closed-loop travel range in the full travel range that is available on the piezo stage. The calculator makes sure that the available voltage-output range and the capacitance sensor range have over-travel margins that are equal on each side of the closed-loop travel.

For information about when to use the AbsoluteFeedbackOffset Calculator and an example of how to calculate deflection, refer to Closed-Loop Models (stages with feedback only).

Figure: Configuration Manager — Absolute Feedback Offset Calculator

To use the AbsoluteFeedbackOffset Calculator

  1. Make sure that it is safe for motion to occur on the selected axis.
    • Make sure that the selected axis is away from any end-of-travel limits or software limits.
    • Make sure that there is nothing blocking the selected axis.
  2. In Configuration Manager, select the AbsoluteFeedbackOffset parameter.
  3. In the Editor section of Configuration Manager, click Calculator. The calculator comes into view.
  4. In Select the Piezo Stage for Axis, make sure that the correct stage is selected.
  5. Click Start. Your axis moves in the order that follows.
    1. Moves to the lower voltage limit.
    2. Moves to the upper voltage limit.
    3. Moves back to the lower voltage limit.
    4. Moves to 0 Volts.
    5. The axis disables.
  6. When the calculation is finished, click Accept. The result shows in the calculator as the New Value of the AbsoluteFeedbackOffset parameter.
  7. The new value of the parameter is not automatically saved to the controller. To save the parameter file, on the File menu select Save or Save As, or click the Save button on the toolbar.

To stop the calculation

If it is necessary to stop the calculation, click Stop or click the Abort button .
