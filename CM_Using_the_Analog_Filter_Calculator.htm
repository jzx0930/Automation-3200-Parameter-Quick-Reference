Using the Analog Filter Calculator

Use the Analog Filter Calculator to calculate coefficients for the two filters on analog inputs 0 and 1 for each axis. Each analog input is shown as a different color signal on the plot.

Figure: Configuration Manager — Analog Filter Calculator

Label Description

The Filter Type menu shows the available filter types.

This section shows the cursor display for the graph.

This graph shows the frequency response of active filters.

Click Remove to remove all active filters.

Click Accept to save the changes and close the Calculator.

Click Cancel to discard the changes and close the calculator.

Click Apply to save the changes and leave the Calculator open.

In the calculator, the frequency response of all active filters is shown below the filter coefficients. The following table gives descriptions of the four Filter Type options.

Table:     Filter Type Options

Filter Type Options Description

LeadLag

The lead lag filter implements a first order digital filter by adding (or subtracting) the specified phase angle in degrees to the open loop response at the specified frequency. A positive angle corresponds to an additional phase lead and a negative angle corresponds to a phase lag at the specified frequency.

LowPass

The low pass filter implements a second order digital filter, rolling off (attenuating) all frequencies above the specified frequency.

Notch

The notch filter implements a second order digital filter, rolling off (attenuating) the specified frequency range. This range is determined by the notch width, which is centered at the specified center frequency. The notch depth determines the attenuation in db, where every 3 db attenuation increment corresponds to a subsequent 50 percent reduction in amplitude, at the center frequency.

Resonant

The resonant filter implements a second order digital filter, amplifying the specified frequency range. This range is determined by the width, which is centered at the specified center frequency. The resonant gain determines the gain in db, where every 3 db gain increment corresponds to a subsequent 50 percent increase in amplitude, at the center frequency.
