Creating Fieldbus Mappings for Drive Modbus

When you configure the A3200 to connect to a Modbus slave device, you must define a connection and create mappings that synchronize I/O between the Modbus device and AeroBasic variables. You link the connection with the axis that is assigned to the drive that you want to use as a Modbus master controller.

To create and associate a Drive Modbus Master Configuration

  1. In the Network Explorer of the Configuration Manager, open the parameter file that you want to use.
  2. Open the Fieldbus node and click the Drive Modbus Master node.
  3. Click the Edit Drive Modbus Master button that is shown in the Editor. The Drive Modbus Master Configuration dialog comes into view.
  4. Figure: Drive Modbus Master Configuration Dialog for Fieldbus Mapping

  5. In the Connections area, click the Add Connection button to add a new Drive Modbus Master connection.
  6. Each connection that you create in the Drive Modbus Master Configuration dialog has its own set of mappings. The connection is associated with one axis that is assigned to a drive that supports drive Modbus.
  7. In the Connection Details area, select the axis of the drive that you want to use as a Modbus controller.

To create mappings

  1. In the Variables area, select the tab for the type of mapping that you want to create. Refer to the documentation from the manufacturer of the Modbus slave module for the correct register types and addresses to use.
  2. Click the Add Mapping button that is above the mapping grid.
  3. Type a name for the mapping. You will reference the mapping by this name in AeroBasic. The last column of the grid shows a preview of the mappings defined in AeroBasic.
  4. Click the arrow in the Type cell and select the type for the variable.
  5. In the Byte Address and Bit Address cells, type the address of the I/O that you want to synchronize. To edit these values, double-click on the cell.
  6. If you want to declare the mapping as a contiguous block of I/O of the same type, then edit the Count. The AeroBasic Definition cell updates to show that you can use the mapping in AeroBasic as an array.
  7. To save your mappings, click OK or Apply.
