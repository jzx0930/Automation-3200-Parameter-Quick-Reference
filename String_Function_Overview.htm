String Function Overview

Strings can include carriage returns, line feeds, spaces, system text, and printable characters. You can add the date and time, floating-point values, and more complex formatting to a string by issuing the MAKESTRING function. For more information, refer to String Variables.

Example

' Make a string from the program name.
$strglob[0] = MAKESTRING "#PG1"
' Make a string from the time.        
$strglob[1] = MAKESTRING "The time is now: #TM"
' Add runtime data to the string.
$strglob[2] = MAKESTRING "Time= #DT, #DT pgm = #PG1"
				
' Find the length of a string.
$global[0] = STRLEN( $strglob[0] )
' Compare two strings.
$global[1] = STRCMP( $strglob[0], "end", 0 )
' Find a string within another string.
$global[2] = STRFIND( $strglob[0], "=", 0 )
' Find a set of characters within another string.
$global[3] = STRCHAR( $strglob[0], "1234567890" )
' Change a number to a string.
$strglob[6] = DBLTOSTR( $global[0], 4 )
' Change a string to a number.
$global[4] = STRTODBL( $strglob[6] )
' Find the ASCII value of a character.
$global[5] = STRTOASCII( $strglob[0], 4 )
' Remove a string from a larger string
$strglob[3] = STRMID( $strglob[0], 1, 5 )
' Change a string to uppercase.
$strglob[4] = STRUPR( $strglob[0] )
' Change a string to lowercase.
$strglob[5] = STRLWR( $strglob[0] )

The previous example (when called STRNGFUN.PGM), produces the following assignments:

$strglob[0] = STRNGFUN.PGM

$strglob[1] = The time is now: 16:05

$strglob[2] = Time= 12/29/2004, 12/29/2004 pgm =STRNGFUN.PGM

$strglob[3] = TRNGF

$strglob[4] = STRNGFUN.PGM

$strglob[5] = strngfun.pgm

$strglob[6] = 12.0000

 

$global[0] = 12.000000

$global[1] = 1.000000

$global[2] = -1.000000

$global[3] = -1.000000

$global[4] = 12.000000

$global[5] = 71.000000
