FlashConfig Overview

The FlashConfig feature makes it easy for you to configure a controller because the axis parameters, Calculator information, and calibration data that you need are kept on the stage. When you connect a stage that has FlashConfig to a drive that supports FlashConfig, the controller automatically identifies the stage each time the controller is powered on or is reset. FlashConfig works for all combinations of drives and stages that support the feature.

You can use the FlashConfig data that is kept on the stages to do the following tasks.

  • Resolve mismatches. The controller senses if you connect a new stage, connect a stage to a different drive, or disconnect a stage. Before you can enable the axis, you must resolve the mismatch condition. Refer to Resolving FlashConfig Mismatches.
  • Restore your controller to the original factory configuration. You can restore the controller with the data used at the factory to initially configure the axis. If you purchased integration from Aerotech, you can restore axis parameters, calculator information, and calibration data (if applicable). If you did not purchase integration, you can restore calculator information. For more information, refer to Restoring FlashConfig Data.
  • Use calculators to set your parameters. Each time a calculator opens, Configuration Manager automatically enters all of the information that is available in FlashConfig. For information on calculators, refer to Using Calculators to Set Parameters.

For information on the types of data that are kept in FlashConfig, refer to FlashConfig Data.

FlashConfig Drives

These drives support FlashConfig:

  • GL4
  • Ndrive QL
  • Ndrive QLe
  • Nmark GCL
