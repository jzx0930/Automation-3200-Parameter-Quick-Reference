1D Axis Calibration Overview

Axis calibration tables apply position corrections to an axis based on the position of a master axis. The SMC software continuously reads the master axis position and uses this to find a corresponding correction value from the calibration table. The correction value is then added to the commanded axis position. Each calibration point uses 8 bytes of controller memory.

You can use calibration tables for:

 

Accuracy Correction

if the master and corrected axis are the same
 

Orthogonality Correction

if the master and corrected axis are different
 

Straightness Correction

to compensate for the variation in straightness of table travel
 

Temperature Compensation

to correct for thermal expansion
  Thermal Compensation to correct for changes to the thermal state of the stage

Format

A calibration table contains calibration data (correction values) for one axis. A calibration file (.cal) can contain a maximum of 32 axes of calibration data. You can apply a maximum of eight tables on one corrected axis. The effect of each table is additive. Each table produces its own correction value independently from the other tables and the final correction value is the sum of the values from all of the tables. There is a maximum of 100 tables available for all axes. Each table is limited by the quantity of available memory or 2.1 billion points, whichever is less.

If the position of the master axis is equal to the master position of a table entry, the correction value for that table entry is used as the active correction value. However, if the position of the master axis lies between two master positions in the table, linear interpolation is used to find a correction value between the two correction values listed in the table. Finally, if the position of the master axis lies outside of the table; that is, all table entry master positions are either greater or less than the position of the master axis, the nearest end point of the table is used to calculate the correction value.

For example, if the lowest master position in the table is 3, with a correction value of 5, then the correction value is 5 for all master axis positions from 3 through minus infinity. To ensure zero axis calibration outside the position range covered by the calibration table, place a correction value of 0 at the end of the table.

After the correction value for the corrected axis has been determined by lookup and interpolation into the table, this value is added to the raw position as seen from the encoder, and used as the actual position of the axis.

Refer to 1D Axis Calibration File Format for more information.

Homing

1D axis calibration requires homing to synchronize the axis with the calibration table.

Note: 1D calibration always follows the sign of the encoder. Changing the ReverseMotionDirection parameter does not have an effect on 1D calibration operation.

Note: To see the calibration file data, use the DUMPAXISCAL Console command in Console.
