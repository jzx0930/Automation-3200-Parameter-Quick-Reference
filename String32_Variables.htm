String Variables

The controller supports global string variables and task string variables. For both types of variables, you can use a maximum of 187 characters. String constants also have a maximum of 187 characters. For more information about variable scope and initialization, refer to Variants.

Global String Variables

Global variables are shared by, and common to, all four tasks (32 with the Professional Option) running on the controller. The number of global string variables ($strglob) is defined by the GlobalStrings global parameter. Use string global variables to specify axis names for commands.

Examples

' Global string variable.
$strglob[5] = "long "
$strtask[6] = "long " + $strglob[5]

Task String Variables

Task variables are local (private) and each task has its own set. The number of task string variables ($strtask) is defined by the TaskStrings task parameter.

Example

' Task (local) string variable.
$strtask[2] = "long "
