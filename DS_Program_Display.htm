Program Display

You can use the Program Display to show the command in your program that corresponds to the cursor location on the current plot. The Program Display is located in Digital Scope in the bottom left corner on the Scope tab. You can pin it so that it stays open.

To use the program display with an existing plot, use the following steps:

  1. Open the plot you wish to view.
  2. Click the Open button in the program display and locate the program used to create the plot.
  3. In the task drop down box, select the task that the program was run on.

  4. Click on the plot to set the cursor. The program display will jump to and highlight the line of the program corresponding to that location on the plot.

    Note: You must collect the Program Line Number signal for the task on which the program was run for the program display to work correctly. If you did not collect the signal in the current plot for the task you would like to view, an error will be shown underneath the program display toolbar. To fix this, use the Configure Data Collection window to add the Program Line Number signal for the desired task and then run the program again.

To use the program display by creating a new plot, use the following steps:

  1. Use the Configure Data Collection window (Configuration > Configure Data Collection) to add the Program Line Number signal for the task on which the program will be run. This signal Converting Plot Files to Matlab Files can be found in the Task tab.
  2. Run the program while performing data collection, either manually or with a SCOPETRIG command.
  3. Click the Open button in the program display and locate the program that was run.
  4. In the task drop down box, select the task that the program was run on.
  5. Click on the plot to set the cursor. The program display will jump to and highlight the line of the program corresponding to that location on the plot.
