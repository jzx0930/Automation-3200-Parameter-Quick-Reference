Configuring Axis Calibration

To make use of axis calibration, your system must be properly configured to use an axis calibration file. Use the following steps to configure your system to use an axis calibration file.

  1. Open the Configuration Manager and expand the Controller node in the Network Explorer.
  2. Under the Controller node, expand the A3200 node and select the Calibration folder.
  3. In the Viewer part of the Configuration Manager you will see entries for 1D Calibration File and 2D Calibration File. Select the appropriate option depending on which type of file you are configuring.
  4. After you have selected one of the options, click the Browse button in the Editor part of the Configuration Manager to select the axis calibration file. Alternatively, you can type the full path and name of the axis calibration file into the text box. Axis calibration files supplied by Aerotech have a .cal extension.
  5. After selecting the file or entering the name manually, click the Set button to finalize your selection. When you are prompted, reset the controller.

Axis calibration tables are loaded on the controller during a reset. If you have specified an axis calibration table for a given axis, a letter C will be displayed in the Status area of the Axis Manager within Motion Composer, and the Calibration Enabled 1D or Calibration Enabled 2D bits of the Axis Status value will also be set.

You can see the calibration value that is currently being applied to an axis in the Position Calibration All field of the Diagnostics tab of the Status Utility. Calibration corrections are not applied until you home the axis that serves as the master reference for the calibration table.
