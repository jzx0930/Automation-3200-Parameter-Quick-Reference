Computing a Fourier Transform of a Plot

When you tune a system, computing a Fourier transform of a signal can help you to identify loop instabilities. Only the visible area of the plot is used to compute the transform. You can zoom in on an area and analyze only that part of the data.

Before you compute the transform, you can configure some options.

To Configure Options for Fourier Transform in Digital Scope

  1. On the Tools menu, point to Fourier Transform.
  2. Click the option that you want to configure and select a setting from the list. You can configure the following options.
    Type
    • Narrowband Computes the narrowband discrete Fourier transform.
    • PSD Computes the discrete Power Spectral Density.
    • Cumulative PSD Computes the discrete Cumulative Power Spectral Density.
    Window

    Windowing improves the results of the Fourier transform computation by reducing the disturbance caused by having a finite sequence of data. You can use the following window types.

    • None (rectangular window)
    • Blackman window
    • Hamming window
    • Hanning window
    • Flattop window
    Mode
    • DFT Computes the discrete Fourier transform. This algorithm does not have any requirements on the length of the input data and does not do any zero-padding. This algorithm is the most computationally intensive.
    • FFT Computes the fast Fourier transform. For the algorithm, the length of the data must be an even power of two, such as 1024 points. If the data is not an even power of two, then it is zero-padded to the next power of two. This algorithm is less computationally intensive than the DFT. If the data sequence meets the length requirement and no zero-padding is necessary, the results of the FFT are identical to the DFT.
    • Auto Computes either the DFT or FFT. If the data length meets the requirements for an FFT, then the FFT algorithm is used. Otherwise, the DFT algorithm is used. This is the default mode.
    Remove the DC Offset

    When Remove the DC Offset is selected, the mean value of the data is subtracted before the Fourier transform is computed.

To compute a Fourier Transform in Digital Scope

  1. Zoom in on the area of the plot that you want to use to compute the transform.
  2. On the Tools menu, point to Fourier Transform and then click Compute Fourier Transform.
  3. On the plot, click the signal that you want to compute. The result shows in a new plot that contains only the newly computed transform.

You can also compute a Fourier transform in Motion Simulator. Refer to Motion Simulator Help for more information.
