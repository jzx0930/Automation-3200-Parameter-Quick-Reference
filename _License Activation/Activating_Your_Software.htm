Activating Your Software

After you complete the installation procedure, you must activate the software. It is only necessary to activate one time on each PC where you install the software. It is only necessary to activate from one of the Aerotech applications. The activation procedure starts the first time you open most of the Aerotech applications. Status Utility, Console, and Feature Manager do not start the activation procedure.

If you are using the Remote Client and Server configuration, you must activate the Server computer, but you do not have to activate the Client computer.

The first time you open the software on a PC, the License Activation dialog box comes into view.

Figure: License Activation Dialog

Internet Activation

If the computer where you installed the software connects to the Internet, click Activate Now. Activation is complete. Click OK and start using your software.

Manual Activation

If the computer where you installed the software does not connect to the Internet, click manual activation. The Manual License Activation dialog box comes into view. Use the procedure that follows to manually activate your software.

Figure: Manual License Activation Dialog Box

Step 1. Create an Activation Request

  1. Click Create Activation Request. The Save As dialog comes into view.
  2. Type or browse to a network location or an external file-storage device where you can have access to the file from a computer that connects to the Internet. The file name format is the name of your PC, underscore, and the name of the software product you are activating. You can change the name of the file, but the Activation Request file must have the .txt extension. Click Save.
    3. If you cannot have access to the file from a computer that connects to the Internet

    If you cannot remove files from the area or otherwise cannot have access to the electronic file from a computer that connects to the internet, you must record the contents of the file and manually enter the information on the Aerotech website. Open the file in a text editor and record the contents exactly as it is in the Activation Request file.

    • Product
    • Version
    • Key
    • Code 1
    • Code 2

Step 2. Visit the Software Activation Website

  1. Go to a computer that connects to the Internet. If you saved the Activation Request file to an external file-storage device, take the file-storage device with you. You must either upload the file, or manually enter the information on the website.
  2. Open any Web browser and go to https://activation.aerotech.com/.
  3. Figure: Aerotech Software Activation Website

  4. Click Choose File.
    5. Or, manually enter the activation information

    If you do not have access to the electronic Activation Request file, click manually enter the activation information.

    Type the information that you recorded from the file and go to step 5.

  5. Type or browse to where you saved the Activation Request file. Click Open. On the Web page, the name of your activation file is adjacent to the Choose File button.
  6. Click Submit. An Activation Response file is created.
  7. Save the Activation Response file to a network location or an external file-storage device where you will have access to the file from the computer where you installed the software. The name of the file is the same as the Activation Request file, but the extension is .ares.

Step 3. Validate the Activation File

  1. Go to the computer where you installed the software. If you saved the Activation Request file to an external file-storage device, take the file-storage device with you.
  2. In the Manual License Activation dialog box, click Validate Activation File.
  3. Type or browse to where you saved the Activation Response file. Select the file and click Open.
  4. Activation is complete. Click OK and start using your software.
