Parameter File Configuration

The default parameter file is located in User Files folder. Multiple parameter files can exist on the PC but the controller only downloads parameters from the active parameter file. The Active file is marked in bold in the Configuration Manager and contains the <Active> tag following the name.

If you purchased system integration from Aerotech, the User Files folder contains a parameter file using this naming convention: ######-A-1-1 (for example, 132123-A-1-1.prm). This file is automatically installed from the installation device. If a file is present with this naming convention, refer to Selecting a Parameter File in the Getting Started section.

If no active parameter file has been established the Configuration Manager will walk you through setting one up starting with the following dialog:

Configuring Axis Names

Right click on the parameter file and select Configure Axes. From the resulting dialog, highlight an axis and select Rename to change the name. Any axis that is listed in the parameter file, but not physically connected, will be a virtual axis.

When the changes are complete, click OK. You will need to save the parameter file by clicking the Save button in the menu or right clicking the parameter file and selecting Save.
