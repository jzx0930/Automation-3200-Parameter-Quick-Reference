Working with Parameters

In Configuration Manager you can edit parameter files and select an active parameter file.

Working with a Parameter File

Making a New Parameter File

In Configuration Manager menu, click File, then click New Parameter File. A new file is made in the Open Files folder.

Opening an Existing Parameter File

To open an existing parameter file, do one of the following.

  • Double-click the file under the User Files node or one of its subfolders.
  • On the Configuration Manager menu, click File, then click Open Parameter File. In the Open File dialog, double-click the file that you want to open, or select the file and click Open.
Editing a Parameter File

When you edit a parameter file, the parameters are listed in the Open Files node of the Network Explorer under the parameter file name.

Figure: Editing the Parameter File

  1. In the Network Explorer, select the category of the parameter that you want to edit. (For example, FaultSetup is an Axis parameter in the Fault subcategory.) All of the parameters in that category show in the Viewer.
  2. In the Viewer, click the row of the parameter that you want to edit.
  3. In the Editor, use one of the following procedures to edit the value of the parameter.
    • Select the appropriate check box or option button.
    • Enter the new value in the text box or IP address field.
    • Use the available calculator. Using the calculator might change more than one parameter at a time.
  4. Repeat the procedure for each parameter that you want to edit.
  1. Changes to parameters are not automatically saved. To save the parameter file, on the File menu select Save or Save As, or click the Save button on the toolbar.

Note: After you change the value of a parameter in the active parameter file, you must reset the controller for the change to have an effect.

Selecting an Active Parameter File

Before you can correctly initialize the controller, you must select an active parameter file. The parameters in this file are then used to initialize the SMC. Refer to Selecting a Parameter File for more information.
