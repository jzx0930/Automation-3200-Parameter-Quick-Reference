Creating Your First Program

Use the following procedure to create an AeroBasic program in Motion Composer.

Create an AeroBasic Program

  1. Open Motion Composer by going to Start → All Programs → Aerotech → Automation 3200 → A3200 Motion Composer.
  2. In Motion Composer, click the Connect button . The Axis Manager comes into view.

    3. Figure: Axis Manager

  4. Go to File → New File to create a new program file. The Create New File dialog comes into view.
  5. In the Create New File dialog, select AeroBasic, then click OK.
  6. Copy and paste the following example program into the Motion Composer editor.

    7. Example

    ' This is a simple A3200 program.

' Enable servo loop control for the axes.
ENABLE X Y

' Move to the "0" reference position.
HOME X Y

' Enable incremental programming mode.
INCREMENTAL

' Set the speed to 100 mm/s. The default units are mm/s.
F100 

' Start a repeat loop.
REPEAT 10

' Move the axes incrementally.
LINEAR X2.54 Y2.54

' Wait for one second.
DWELL 1

' Stop the repeat loop.
ENDRPT

' Enable absolute programming mode.
ABSOLUTE

' Move to the start position at 50 mm/s.
LINEAR X0 Y0 F50

' You must include this line at the end of the program.
END PROGRAM
  7. Click the Run button to save, compile, associate, and start the program. If you want to save, compile, associate and start the program separately, use the individual buttons.
    • If the program is new, you are asked to enter a program name.
    • If you save this program in the default location, it comes into view under the User Files node in the Network Explorer.
    • To reopen this program, double-click the program in the User Files node in the Network Explorer.
    • If the program encounters errors, these errors are shown in the Error List at the bottom of the application.
