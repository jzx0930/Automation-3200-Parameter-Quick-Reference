Capturing Axis Data until an Axis Fault Occurs

When you troubleshoot an unexpected fault condition, it helps if you collect axis state information in Digital Scope. You can configure Digital Scope to collect data continuously and to automatically stop collecting it when a fault occurs. Then the application shows a snapshot of the data immediately before the fault event.

TO CAPTURE AXIS DATA UNTIL A FAULT OCCURS

  1. In Digital Scope, on the Tools menu, select Preferences. The Preferences dialog comes into view.
  2. In the Scope section of the left pane, select DataCollection.
  3. In the Settings section of the right pane, make sure that the OnAxisFaultAction setting is set to Stop Collecting After Current Snapshot or Stop Collecting Immediately.
  4. Click OK to close the Preferences dialog.
  5. Configure Digital Scope to collect important information about the axis on which the fault is occurring.
  6. Start the data collection by clicking the Collect Continuous Data button on the toolbar.

After you complete these steps, Digital Scope continuously collects and shows the selected data. It stops when a fault occurs, which lets you examine the state of the axis before the fault event.
