Info Variables

Syntax:

 
$info[<Index>]

Example:

 
IF ($info[0] != 0)
	TASKERROR $info[0], $info[1]
ENDIF

Info variables are special task variables that certain A3200 commands use to return status information to the programmer. Some commands use these variables to return error related information so that you can customize error handling behavior.

Most A3200 commands automatically cause a task error if they cannot complete correctly, but some commands complete without causing a task error. Instead, these commands store an error code in $info[0] and an error location in $info[1]. In these cases, you can use the TASKERROR command to force a task error condition.

Example

$hFile = FILEOPEN "C:\MyFile.dat", 0
IF ($info[0] != 0)
	TASKERROR $info[0], $info[1]
ENDIF

Note: If a command returns error related information in $info[0] and $info[1], make any of your programs that use the command check $info[0] and $info[1] to catch potential errors.
