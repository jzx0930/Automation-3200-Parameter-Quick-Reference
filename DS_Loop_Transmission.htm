Loop Transmission

Use the Loop Transmission utility to analyze the characteristics of the servo loop and the associated mechanical system. You can find the Loop Transmission utility in the Loop Transmission tab of Digital Scope. The Loop Transmission utility injects a digitally-generated disturbance into the Proportional-Integral-Derivative (PID) servo loop and measures the corresponding response. The disturbance is injected into the output from the PID controller directly before the current loop input.

The signals before and after the summation point are sampled and saved for magnitude and phase analysis. The ratio of the magnitudes of the collected signals represents the open-loop magnitude response of the system, including the PID controller, digital filters, current loop and plant (mechanical system). The difference in phase between the collected signals represents the phase response of the open-loop system, including the PID controller, digital filters, current loop, and plant. The result is an open-loop frequency response of the system with the magnitude and phase plotted by the Loop Transmission utility.

Figure: Loop Transmission Utility

Running a Loop Transmission

  1. Select the axis that you want from the axis list in the main menu or the Axis box on the Axis toolbar.
    2. Figure: Digital Scope — Axis Toolbar

  2. To enable the axis, click the Enable button on the Axis toolbar.
  3. Enter a starting and ending frequency for the loop transmission on the Loop Transmission Action toolbar. The ending frequency must be less than or equal to half of the servo update frequency. The default values are 10 Hz for the starting frequency and 2000 Hz for the ending frequency as shown in the following figure.
    4. Figure: Loop Transmission Action Toolbar

  4. Enter the number of divisions for which the loop response will be evaluated. The loop response is evaluated at the number of frequencies (including the start and end frequency) given by this value.
  5. Enter the maximum current value of the disturbance injected into the loop as a percentage of the maximum rated current of the amplifier. Start with the default value of 5%. If any invalid points are contained in the final loop transmission plot (shown by an “X” on the plot), this value might be increased.
  6. Start the loop transmission by clicking the Start button on the Loop Transmission toolbar or from the Main menu.

Constant Velocity Option

The constant velocity option moves the axis in both the forward and reverse direction a specified distance and velocity while the loop transmission is running. This option is useful for obtaining more accurate low frequency measurements on systems with high friction.

On the Configuration menu, select Constant Velocity to enable and configure the constant velocity option. You can see if the Constant Velocity Option is or is not enabled in the Status toolbar.

Response Configuration Options

To select which response type to show after loop transmission is complete, open the Configuration menu and select Response Type. If you select more than one response type, the types will overlap on the plot.

List of Response Types
  • Open-Loop shows the open-loop response, which includes the PID controller, plant, digital filters, and current loop. This is the default option and is only available when running an open-loop loop transmission.
  • Plant shows the Plant response, which is a plot of the stage frequency response. This option is only available when running an open-loop loop transmission.
  • Sensitivity shows the sensitivity frequency response. The sensitivity response is the derived response from the position command input to the position error signal without feedforward effects. This shows the change in the closed-loop transfer function with respect to a change in the plant (loop sensitivity to variations in the plant). This option is only available when running an open-loop loop transmission.
  • Complementary Sensitivity shows the complementary sensitivity frequency response. This is the derived response from the position command input to the position output without feedforward effects. This option is only available when running an open-loop loop transmission.
  • Process Sensitivity is derived from the Plant input to the position output. You can use Process Sensitivity to analyze the effects that disturbances at the Plant input have on the output response.
  • Closed-Loop shows the closed-loop frequency response. This is the derived response from the position command input to the position output with feedforward effects included.  This option is only available when running an open-loop loop transmission.
  • Closed-Loop Error shows the closed-loop error frequency response. This is the derived response from the position command input to the position error with feedforward effects included. This option is only available when running an open-loop loop transmission.
  • PID Controller shows the frequency response of the PID controller. This response is determined by the parameters GainKi, GainKp, GainKpi, and GainKpos.
  • PID Controller and Servo Filters shows the combined frequency response of the PID controller and the servo filters. This response is determined by the parameters GainKi, GainKp, GainKpi, GainKpos, and the ServoFilter parameters.
  • Feedforward shows the feedforward response. The feedforward response is determined by the values of the GainAff, GainPff, and GainVff parameters.
  • Inverse Feedforward shows the inverse of the feedforward response. The inverse feedforward response is determined by the values of the GainAff, GainPff, and GainVff parameters. For the best system performance, this response should closely match the plant response.

Loop Shaping Toolbar 

You must be in the Loop Transmission tab in Digital Scope to use the Loop Shaping toolbar. Use the Loop Shaping toolbar to graphically change the open-loop frequency response of the system by changing the controller gains and adding digital filters. To use Graphical Loop Shaping, you must have Open Loop Response Type selected in the Configuration > Response Type menu.

Figure: Loop Shaping Toolbar

After you run an open-loop loop transmission, you can shape the open-loop frequency response by changing the controller gains or adding digital filters.

Shifting the Open-Loop Magnitude and Phase

Click the magnitude (blue) or phase (red) up and down arrows to graphically shift the magnitude plot or the phase plot. When you shift the magnitude plot, click the up or down phase (red) arrows to shift the magnitude response by plus or minus 1 dB, and the phase will stay relatively constant. When you shift the phase plot, click the up or down magnitude (blue) arrows to shift the phase response by plus or minus 1 degree at the frequency listed, and the magnitude plot will change accordingly.

You can also shift the phase by changing Kpos or by changing the ratio of Ki over Kp. You can select the method by clicking the down arrow that is adjacent to the phase (red) arrows, and then selecting either the Kpos or Ki/Kp option.

Filter #, Type, Apply - Adding Digital Filters

Use the loop shaping toolbar to graphically add LowPass, Notch, LeadLag, and Resonant filters. To add a filter, select the filter type in the Type box. Add the values that you want to specify into the applicable boxes and click Apply to add the filter to the plot. After you add a filter, you can select the next available filter by selecting the next number in the Filter # box.

Edit Filter - Editing and Removing Digital Filters

The loop shaping tool shows a small icon at the filter frequency above the magnitude plot that shows the type of filter and the filter number. Each filter type has a special icon containing the filter number where each icon is listed in the following table.

To remove a filter, right-click the filter icon and select Remove. To adjust a filter, right-click the filter icon and select Edit. You can also select the Edit Filter button on the Loop Shaping toolbar, which will let you edit the filter that has a number listed in the Filter # box. When you adjust a filter, you can enter the new filter parameters in the loop shaping toolbar or graphically adjust the filter. Hold the mouse pointer over any filter icon to show the filter type and the filter parameters.

When you add or adjust a digital filter, the new filter coefficients are calculated but are not saved or downloaded until you click the Apply button in the Loop Shaping toolbar.

Auto Fit Digital Filters

If you click the Auto Fit Digital Filters button, Digital Scope will automatically put the filters in the best positions.

Optimize

If you click the Optimize button , Digital Scope optimizes the gains and digital filters. To configure the filter and optimization settings, click the Optimize arrow.

Status Toolbar

You must be in the Loop Transmission tab in Digital Scope to use the Status toolbar. Use the following buttons to overlap loop transmission plots, view input and output signals, and view pole-zero maps.

Figure: Status Toolbar

Results — Phase and Gain Margins

Use the Results button on the Status toolbar to view detailed information about all Phase and Gain Margins for a loop transmission.

Overlap Plots — Overlapping Loop Transmission Plots

You can overlap multiple loop transmission plots to compare the results of the plots with the Overlap Plots button on the Status toolbar. To select which plots you want to overlap, click the Overlap Plots button.

Figure: Overlap Plots Dialog

Use the Select Gains and Filters drop-down to force all overlapped plots to share the same gains and filters from a specific plot. If you do not want to override the gains or filters for the plots, select Keep Original. If the gains or filters are different in any of the overlapped plots, loop shaping will be disabled.

When you view overlapped plots, the Status toolbar will show the most unsatisfactory phase and gain margin in all of the plots.

Show Time Data — Loop Transmission Time Data

To view the input and output signals used to calculate the response of the loop transmission, click the Show Time Data button on the Status toolbar. You can use this data to troubleshoot problematic frequencies in your loop transmission.

Figure: Loop Transmission Time Data

To show the input and output signals versus time, click the Show Time Data button to change the active plot.You can only view time data for a loop transmission that is currently active or one that has been collected during the session of the Scope. Loop shaping is disabled while you are viewing time data.

The Frequency Selection drop-down shows all of the frequencies that were collected for the loop transmission. Select a frequency to plot the input and output time data collected at that frequency.

Pole-Zero — Controller Pole-Zero Map

After you have run a loop transmission, you can view a pole-zero map of the controller by clicking the Pole-Zero button in the Status toolbar. The pole-zero map is a plot of the poles and zeros of the controller in the complex plane.

Figure: Pole-Zero Map

The controller always contains a pole at zero (represented by an X) and at least two zeros (represented by circles) in the left-half complex plane. The zero resulting from the velocity loop is shown in black, and the zero resulting from the position loop is shown in red. Aerotech recommends that the position loop zero always be to the right of the velocity loop zero. You can move the zeros along the real axis by changing the controller gains or using the shift phase loop shaping function. The locations of the controller zeros affect both the magnitude and phase of both the open-loop and closed-loop system.
