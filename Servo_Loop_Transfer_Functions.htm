Feedforward

FF(z) represents the discrete-time transfer function of the feedforward controller. The transfer function is derived from Position Command to Current Command.

Digital Filters

F(z) represents the discrete-time transfer function of the servo-loop digital filters. F(z) depends on the number and type of active digital filters. You can use the Filter Calculator in Configuration Manager or Digital Scope to see the frequency response of the digital filter transfer function.

Open-loop

L(z) represents the discrete-time open-loop transfer function. The open-loop transfer function includes the PID controller (C(z)), the digital filters (F(z)) and the plant (P(z)). The open-loop transfer function is derived from the Position Error to the Position Feedback.

Closed-loop

CL(z) represents the discrete-time closed-loop transfer function. The closed-loop transfer function includes the PID controller (C(z)), the digital filters (F(z)), the feedforward controller (FF(z)) and the plant (P(z)). The closed-loop transfer function is derived from Position Command to Position Feedback.
