Configuring Parameters for Dual Loop Mode with MXR

Hardware: This information does not apply to the Npaq MR.

Because the Npaq is a multi-axis drive, you can configure a single axis for dual-loop mode by using two different channels of the primary encoder multiplier.

To Configure Dual-Loop Mode

NOTE: The Npaq has one set of encoder multiplier parameters on each axis. For this configuration, you must use a set of encoder multiplier parameters on one axis to configure the velocity feedback device and a set of encoder multiplier parameters on a different axis to configure the position feedback device. To see a list of these parameters, refer to Multiplier Category.
  1. Open Configuration Manager.
  2. Make sure that you are connected to an Npaq.
  3. Connect the velocity and position feedback devices. Do the following.
    1. Connect the velocity feedback device to the channel that corresponds to the axis feedback connector that you are using on the Npaq. The axis that corresponds to this channel becomes the axis that this procedure refers to as "the axis that you are configuring."
    2. Connect the position feedback device to a channel that is different from the channel that you are using for the velocity feedback device.

      3. NOTE: If you have an ATX series stage, interchange the velocity feedback device and the position feedback device, and configure the FeedbackSetup parameter options. Use the steps that follow.

      1. Connect the position feedback device to the channel that corresponds to the axis feedback connector that you are using on the Npaq. The axis that corresponds to this channel becomes the axis that this procedure refers to as "the axis that you are configuring."
      2. Connect the velocity feedback device to a channel that is different from the channel that you are using for the position feedback device.
      3. Change the Limit Input Source and Hall Input Source of the FeedbackSetup parameter from Velocity Feedback Channel to Position Feedback Channel.

      For Example — Axis Configurations

      Your Npaq uses the default axis names of X, Y, Z, U, A, and B. Of these axes, X, Y, Z, and U have stages connected while axes A and B are unused.

      For the Velocity Feedback Device

      You configure Axis X for dual-loop mode by using two encoder multiplier channels. Because you are configuring the first axis of the Npaq, you must connect the velocity feedback device to Channel 0.

      Axis X becomes the axis that you configure throughout most of this procedure. The procedure refers to this axis as "the axis that you are configuring".

      For the Position Feedback Device

      Usually you can connect the position feedback device to one of the other channels. But axes Y, Z, and U are currently using Channels 1 - 3. Thus you connect the position feedback device to Channel 4, which corresponds to Axis A.

  4. Use the Axis Parameter Calculator to configure the following Feedback parameters for the velocity feedback device. Make sure that you specify the values of these parameters on the axis that you are configuring.

    Label Description

    In the Feedback section of the calculator, you can configure the settings for your position feedback device and velocity feedback device.

    If you manually configure the EncoderMultiplicationFactor parameter in Step 5, the value that you use for that parameter must be the same as the value that you specify for the Multiplication Factor of the position feedback device.

    The Feedback Parameters section of the calculator shows all of the new values for the Feedback parameters that you configured in Step 4.
  5. Manually configure the following parameters for the position feedback device. Make sure that you specify the values of these parameters on the axis that corresponds to the channel that you specified for the PositionFeedbackChannel parameter.

  6. NOTE: If only one of your feedback devices requires multiplication, continue to the next step. The Axis Parameter Calculator does all of the parameter configurations which means that it is not necessary for you to manually configure parameters.
    1. Specify a value for the EncoderMultiplicationFactor parameter. This value must be the same as the value that you specified for the Multiplication Factor of the position feedback device in the Axis Parameter Calculator.
    2. Configure all of the Multiplier Category parameters that are applicable to your project settings.
  7. Reset the controller.

To Tune the Velocity and Position Feedback Devices

You need to tune only the feedback devices that require multiplication. Refer to the Feedback section of Table: Axis Parameter Calculator — Commonly Used Sections in Using the Axis Parameter Calculator for more information.

  1. In Digital Scope, click the Axis arrow and select the axis that you are configuring.
  2. This step is required only if your velocity feedback device supports multiplication. Configure the following Multiplier Category parameters for the velocity feedback device.
    1. Select the Feedback Tuning tab.
    2. Click the Device arrow and select Velocity.
    3. To automatically calculate the Sine and Cosine values for the following parameters, refer to the Tuning an Analog Feedback Device procedure in Feedback Tuning. Then continue to the next step in this procedure.

    1. Select the Feedback Tuning tab.
    2. Click the Device arrow and select Position.
    3. To automatically calculate the Sine and Cosine values for the following parameters, refer to the Tuning an Analog Feedback Device procedure in Feedback Tuning. Then continue to the next step in this procedure.
