Configuring Parameters for Position Feedback Averaging on Single-Axis Drives

By default, the axis uses feedback signals from the primary encoder as the position feedback. If you want to include feedback signals from the auxiliary encoder, use the Feedback Averaging option of the FeedbackSetup Parameter. The controller will average the feedback signals from the primary and auxiliary encoders. Then the axis uses the average of these signals as the position feedback. The controller averages the feedback signals after you configure the applicable parameters, enable Feedback Averaging, and home the axis.

In the active parameter file of Configuration Manager, configure the parameters that follow. For information about how to edit parameter files, refer to Working with Parameters.

TO AVERAGE THE PRIMARY AND AUXILIARY FEEDBACK SIGNALS AS THE POSITION FEEDBACK

  1. Connect each encoder to the drive so that all the encoders count in the same direction.
  2. Open Configuration Manager.
  3. Make sure that the PositionFeedbackType Parameter is set to Local Encoder Counter.
  4. Make sure that the VelocityFeedbackType Parameter is set to None (Use Position Feedback).
  5. For the FeedbackSetup Parameter, do the steps that follow:
    1. Set the Feedback Averaging option to Enabled.
    2. Set the Position Feedback Direction option to Normal.
  6. Save the parameter file. Then reset the controller for these changes to have an effect.
  7. Open Motion Composer.
  8. Enable the axis on which you configured the parameter values.

    9. Status Item Values Before the Controller Averages the Feedback Signals

  9. In the Axis Manager section of Motion Composer, the Position Feedback and Position Feedback Auxiliary status items show the values that follow:
    • The Position Feedback status item shows the position feedback from the primary encoder.
    • The Position Feedback Auxiliary status item shows the position feedback from the auxiliary encoder.

    10. NOTE: If you do not see these status items in the Axis Manager, select Tools → Preferences. In the Preferences dialog, select Axis Manager. In the Display section, select Primary Display Type. Click the arrow and select PositionFeedback. Then select Secondary Display Type. Click the arrow and select PositionFeedbackAuxiliary. Click Apply and then OK to close the dialog.
  10. Home the axis. After the home cycle completes, the controller averages the feedback signals from the primary and auxiliary encoders. The axis uses the average of these signals as the position feedback.

    11. Status Item Values After the Controller Averages the Feedback Signals

  11. In the Axis Manager section of Motion Composer, the Position Feedback and Position Feedback Auxiliary status items show the values that follow:
    • The Position Feedback status item shows the average of the feedback signals from the primary and auxiliary encoders. This value is the position feedback for the axis.
    • The Position Feedback Auxiliary status item continues to show the position feedback from the auxiliary encoder.
