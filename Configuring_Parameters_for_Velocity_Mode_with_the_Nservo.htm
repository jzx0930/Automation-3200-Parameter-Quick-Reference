Configuring Parameters for Velocity Mode with the Nservo

You can configure the Nservo to be used with an external velocity amplifier. The Nservo sends an analog signal that is proportional to position error, which is used as a velocity command to the amplifier. This configuration is typically used with tachometer-based systems.

To configure the Nservo to operate in this mode, you must know the following information about your system.

  • What is the scaling of the velocity amplifier (in counts per units for a 1 volt input)?

To configure the parameters to operate in the velocity mode

  1. Set the MotorType parameter to DC Brush.
  2. Set the PositionFeedbackType and PositionFeedbackChannel parameters to values that are appropriate for your system (typically Local Encoder Counter and Default values).
  3. Set the VelocityFeedbackType parameter to None (use Position Feedback).
  4. Set the Tachometer Mode setting of the ServoSetup parameter to Enabled.
  5. Set the GainKp parameter to 4096.

    • For GainKpos, start with a value of 1 and increase the value to decrease following error and settling time. If you increase the gain too much, the position error will show signs of oscillation.
    • Use a small value for GainKi to decrease the position error at the end of a move. Typical values are in the range of 0.1 to 1.
  11. Keep all other servo loop gains set to the default value of 0.
