System Diagnostics Overview

The Status Utility is a debugging tool that shows the status of all 32 axes and all four controller tasks. This includes 32 tasks if you purchased the Professional Option. You can use the Status Utility for initial machine setup and testing. It shows the status of drive signals which includes axis hardware limits, drive enables, drive faults, probe input, Hall-effect, and encoder feedback signals.

The Status Utility shows physical hardware states as -- if the signal is off or ON if the signal is on.
