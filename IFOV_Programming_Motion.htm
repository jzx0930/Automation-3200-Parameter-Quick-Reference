IFOV: Programming Motion

To Program Motion

  1. Issue the ENABLE and HOME Commands to enable and home all of the servo and galvo axes. You must enable and home all the specified axes before you use IFOV.
  2. Issue the RAPID Command to move the servo axes to the initial position.
  3. Issue the RAPID Command to move the galvo axes to (0,0) which is the center of the galvo scan head.
  4. Issue the WAIT MODE AUTO Command to set your task to the automatic wait mode.
  5. In your IFOV program, set the MotionUpdateRate Parameter on the task to one of the values that follow:
    • To decrease the workload of the controller, set the value of this parameter to 24.
    • To use the higher trajectory rate of the galvo scan head, set the value of this parameter to 48.
  6. Optional. To turn on the velocity profiling mode for the task, issue the VELOCITY ON Command.
  7. Issue the IFOV ON Command to enable Infinite Field of View (IFOV) on the task.
  8. In your IFOV program, program all motion on the galvo axes. The controller automatically allocates the motion to the galvo axes and the servo axes to make the correct profile in X-Y space:
    • If you command a marking move such as BEZIER, CCW, CW, or LINEAR on the galvo axes, the controller activates the laser output.
    • If you command a non-marking move such as RAPID on the galvo axes, the controller deactivates the laser output.
  9. Refer to Scanning and Marking for more information.
  10. When you are done commanding IFOV motion, issue the IFOV OFF Command to disable IFOV on the task.
