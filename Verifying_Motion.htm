Verifying Motion

Use the following procedure to verify motion on your system.

To verify motion

  1. Enable the Drives.
    1. Open Motion Composer.
      • Go to Start → All Programs → Aerotech → Automation 3200 → A3200 → Motion Composer.
      • In Motion Composer, click the Connect button . The Axis Manager comes into view.
      • Figure: Axis Manager

    2. Use one of the following options to enable axis X.
      • On the Axis Manager, click the Enable button .
      • OR
      • In the Execute Command section of the Axis Manager, type Enable X.
      • Figure: Execute Command Section

      • When axis X is enabled, the Status section changes to Enabled.
      • Figure: Axis X Enabled Status

  2. Home the Axis.
    • On the Axis Manager, click the Home button that is adjacent to Axis X.
    • When the home completes, the Status section for axis X changes to Homed.
    Figure: Axis X Homed Status

  3. Jog the Axis.
    • On the Axis Manager, click the Jog buttons to jog the axis.
    • Click the Ellipsis button to change the Jog Type to Speed, Distance, or Distance (Hold) in the Jog Settings dialog.
    Figure: Jog Settings Dialog
