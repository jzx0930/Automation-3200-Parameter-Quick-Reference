Fieldbus Mapping Dialog

Use the Fieldbus Mapping dialog to edit the Fieldbus mappings that are stored in your parameter file. For more information about Fieldbus mappings, refer to PC Modbus Overview, Drive Modbus Overview, or Fieldbus Options Overview.

To open the Fieldbus Mapping dialog

Figure: Configuration Manager Edit Fieldbus (This example shows PC Modbus Master)

  1. In Configuration Manager, in Network Explorer, open the parameter file that you want to use.
  2. Open the Fieldbus node and click one of the Fieldbus type nodes (for example, PC Modbus master).
  3. Click the Edit button that shows in the Editor. The Fieldbus Mapping dialog comes into view. The dialog and the dialog title are specific to the Fieldbus type that you selected.

Using the Fieldbus Mapping Dialog

The Fieldbus Mapping dialog has the following main sections.

  • Connections. Add, delete and select connections.
  • Connection Details. Edit the property details of a selected connection. The details you can edit depend on the Fieldbus type that you selected in Network Explorer.
  • Variables. Edit the mappings that are associated with the selected connection. The types of mappings you can create depend on the Fieldbus type that you selected in Network Explorer.

The following figure shows the dialog that opens when you select to edit PC Modbus Master in Network Explorer.

Figure: Fieldbus Mapping Dialog — PC Modbus Master Configuration

Label Name Description
Add Connection Button Click to add a new connection to a Fieldbus, up to the maximum number of supported connections for a Fieldbus type.
Delete Connection Button Click to delete the connection that is selected and all related mappings.
List of Connections

Area that shows a list of all the connections. If you select a connection, the Connection Details and Mappings section update with the information related to that connection.

Connection Details

Area where you edit the connection parameters that are defined by the Fieldbus type that you selected.

Register Type and Mapping Type Tabs

Tabs for each register or mapping type. Click the tab to view and edit the mappings for that type. The availability of the types is defined by the Fieldbus type that you selected.

Add Mapping Button

Click to open the Name cell of the Add Mapping row in the Mapping Grid so that you can add a new mapping.

Delete Mappings

Click to delete the mapping that is selected in the grid.

Mapping Grid

Area where you edit all of the mappings of the selected register or mapping type.

Add Mapping Row

Double-click on the Name cell and type the name of the new mapping. This row will always show at the end of the list of existing mappings.

Name Field

Type the name of the mapping.
Count Field If you enter a number greater than one, then you create an array. For example, if you enter 8, then you create an array of size 8.
Type Field Select the data type that you want to assign to the mapping.
Byte Address Field Type the absolute address, in bytes, that you want to give to the mapping. This address is added to the address that you specify in the Bit Address field to make the full address for the mapping.
Bit Address Field Type the absolute address, in bits, that you want to give to the mapping. This address is added to the address that you specify in the Byte Address field to make the full address for the mapping.
Comment Field Optional. You can define a comment for the mapping.

AeroBasic Definition Field

A preview of how the mapping looks when it is defined in AeroBasic as a program variable. If the mapping is an array, the size of the array shows in this field of the grid. You cannot edit this field.
