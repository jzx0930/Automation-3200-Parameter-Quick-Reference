Axis Point Variables

Axis point (APT) variables hold either motion targets or argument lists. An APT is an array that contains one value for each axis, allowing each of the axis point variables to maintain the coordinates for a 32-dimensional point in space. The controller contains global-based and task-based APT variables. The number allocated in each case is defined by parameters GlobalAxisPoints and TaskAxisPoints.

String variables may be assigned axis point variables to specify axis target positions in commands.

Example 1

An APT variable is used for a motion target.

' Specify the target positions of the axes.
$aptglob[0] = X5 Y7 Z2
' The controller executes the move.
LINEAR $aptglob[0]

Example 2

You can assign APT variables to string variables. In this example, an equals sign (=) or space is required to separate the axis name from the number in the string.

Again, an APT variable is used for a motion target.

' This is a string variable.
$strglob[5] = "X=5 Y=7 Z=2"
' The string variable changes to an APT variable.
$aptglob[3] = $strglob[5]
' The controller executes the move.
LINEAR $aptglob[3]
